# 1.1 无线光通信系统链路特性可视化平台设计的背景和意义

随着5G/6G通信、物联网、工业物联网等领域的快速发展，对高速率、低延迟 、抗电磁干扰的通信技术需求日益迫切。无线光通信（VisibleLightCommunication , VLC；Free Space Optical Communication, FSO）作为一种基于光波传输的无线通 信技术，具有频谱资源丰富、保密性强、无电磁污染等独特优势，在室内短距通 信、室外应急通信、卫星间链路等场景中展现出广阔应用前景。然而，无线光通 信链路性能易受外部环境（如大气衰减、背景光干扰、遮挡）和系统参数（如发 射功率、调制方式、传输距离）的复杂影响，其链路特性（误码率、通信速率、 接收光功率等）具有抽象性和动态变化特点。传统研究中，链路性能分析多依赖 理论公式推导或单一数据表格呈现，存在直观性差、参数关联不清晰、难以快速 定位性能瓶颈等问题，给工程设计、教学演示和技术推广带来不便。 

因此，设计一款能够直观呈现无线光通信链路特性的可视化平台，将抽象的 理论模型、动态的链路变化与关键性能参数转化为可视化界面，成为解决上述问 题的有效途径，兼具理论支撑和工程实用价值。 

# 1.2 国内外研究现状

无线光通信技术的研究始于20世纪60年代。1965年，美国国家航空航天局（N ASA）在双子座7号航天器上完成了最早的无线光通信实验。1979年，Gfeller和 Bapst推出了第一个室内FSO系统，采用红外光波段的漫射辐射。此后，无线光通 信技术在理论和应用方面取得了长足进展。在大气湍流对光通信影响方面，Rytov 近似方法被广泛用于分析光波在随机介质中的传播特性。Alimi和Monteiro[1]在 Sensors期刊发表的综述论文全面总结了FSO通信的使能技术、挑战和发展趋势， 指出大气湍流、天气条件劣化和对准误差是FSO系统面临的主要挑战。Kaur等人[2] 在Telecom期刊发表了基于DP-QPSK调制的MDM-WDM FSO链路建模与性能分析 研究，针对新德里、华盛顿、伦敦和开普敦等不同地理区域的雾、霾和云天气条 件，建立了统一的经验衰减模型，并采用随机森林、K近邻、多层感知器和梯度提 升等机器学习技术估计接收信噪比。Ullah等人[3]在Sensors期刊提出了一个综合的 开源LiFi通信仿真框架，为可见光通信系统的建模和仿真提供了重要参考。 

在调制技术方面，研究人员提出了多种适用于无线光通信的调制方案，包括 开关键控（OOK）、脉冲位置调制（PPM）、正交频分复用（OFDM）等。Wu等 人[4]在Sensors期刊分析了无人机辅助的混合FSO/RF通信系统在各种天气条件下的 性能。Poulose[5]在Signals期刊使用OptiSystem软件仿真了室内可见光通信系统，展 示了专业仿真工具在VLC系统设计中的应用。 

国内在无线光通信领域的研究起步相对较晚，但发展迅速。西安电子科技大 学、北京邮电大学、电子科技大学、中国科学院长春光学精密机械与物理研究所 等高校和科研机构在该领域开展了大量研究工作。刘琦[6]基于Rytov和Markov近似 

方法，分析了考虑湍流内、外尺度影响时的高斯光束闪烁和波束扩展特性，研究 了不同调制方式（OOK、副载波PSK和副载波DPSK）对光通信系统误码率性能的 影响，结果表明副载波DPSK调制在光通信系统应用中较OOK和副载波PSK调制更 具优势。耿文凤[7]分析了激光在大气中雨、雾、雪等天气下的衰减特性和大气湍 流的闪烁特性，讨论了采用多发射机和部分相干光技术对系统性能的改善。姬瑶 等人[8]在《西安电子科技大学学报》分析了弱湍流下斜程大气激光通信的误码率 性能。周焕然等人[9]在《激光与红外》期刊综述了激光大气传输的仿真研究进展 ，总结了现有的数值仿真方法和工具。 

目前，国内外已有一些用于通信系统仿真的软件平台，如MATLAB/Simulink 、NS-3、OPNET等。Elamrawy等人[10]的研究展示了LED阵列配置对VLC系统性能 影响的可视化分析。Kaur等人[2]的研究提供了不同地理区域FSO链路性能的对比可 视化。然而，现有工具大多缺乏综合展示多参数耦合影响、支持实时交互、提供 直观3D可视化的综合性平台。 

综上所述，无线光通信技术在理论研究方面已较为成熟，国内外学者在大气 湍流模型、调制技术、链路性能分析等方面取得了丰硕成果。在仿真工具方面， 已有多种基于MATLAB和Python的开源框架可供参考。然而，在链路特性可视化 方面仍存在以下不足：缺乏专门针对无线光通信链路特性的可视化平台，现有工 具的可视化程度和交互性不足，多参数耦合影响的直观展示手段欠缺，理论模型 与实际应用之间缺乏有效的可视化桥梁。 

因此，设计一款功能完善、交互性强、可视化效果好的无线光通信系统链路 特性可视化平台具有重要的研究价值和应用前景。

# 2.2.1 相关理论

# 1) 误码率计算模型

误码率（BER）是衡量通信系统性能的核心指标。本平台实现了无湍流和含 湍流两种条件下的BER计算。 

（1）无湍流条件下的BER 

在无湍流条件下，接收光功率恒定，BER仅由信噪比（SNR）决定。对于直 接检测系统： 

OOK调制： 

$$
\mathrm {B E R} _ {\mathrm {0 0 K}} = \frac {1}{2} \operatorname {e r f c} \left(\sqrt {\frac {\mathrm {S N R}}{2}}\right) \tag {2.1}
$$

BPSK调制： 

$$
\mathrm {B E R} _ {\mathrm {B P S K}} = \frac {1}{2} \operatorname {e r f c} (\sqrt {\mathrm {S N R}}) \tag {2.2}
$$

其中erfc(⋅)为互补误差函数，SNR为信噪比。 

# （2）含湍流条件下的BER

当存在大气湍流时，接收光功率发生随机起伏，BER需对光强概率密度函数 进行积分。本平台采用对数正态分布描述弱湍流条件下的光强起伏： 

$$
\mathrm {f} _ {\mathrm {I}} (\mathrm {I}) = \frac {1}{\mathrm {I} \sigma_ {\mathrm {l}} \sqrt {2 \pi}} \exp \left[ - \frac {\left(\ln \mathrm {I} - \mu_ {\mathrm {l}}\right) ^ {2}}{2 \sigma_ {\mathrm {l}} ^ {2}} \right] \tag {2.3}
$$

其中 $\mathrm { ~  ~ { ~ \sigma ~ } ~ } _ { 1 } ^ { 2 } = \ln ( 1 + \mathrm { ~  ~ { ~ \sigma ~ } ~ } _ { \mathrm { { I } } } ^ { 2 } )$ 为对数光强方差， $\mu _ { \textrm { l } } = - \textrm { o } _ { 1 } ^ { 2 } / 2$ 为对数光强均值。 

含湍流的BER通过对光强起伏进行统计平均得到： 

$$
\overline {{\mathrm {B E R}}} = \int_ {0} ^ {\infty} f _ {\mathrm {I}} (\mathrm {I}) \cdot \mathrm {B E R} (\mathrm {S N R} \cdot \mathrm {I} / \langle \mathrm {I} \rangle) \mathrm {d I} \tag {2.4}
$$

本平台采用Gauss-Hermite数值积分方法计算上述积分，在保证精度的同时提 高计算效率。该方法适用于弱湍流区（ $\sigma _ { \mathrm { ~ R ~ } } ^ { 2 } < 1$ ），中强湍流区应采用Gamma-Gamma分布进行更精确的计算。 

# 2）大气湍流效应

图 1 大气湍流及其对光通信的影响


大气湍流是指大气中温度、压强的随机变化导致折射率发生随机起伏的现象 ，会引起光强闪烁、光束漂移和波束扩展等效应。本平台采用扩展Rytov方法，该 方法将光强起伏分解为大尺度湍流起伏对小尺度湍流起伏的调制，适用于从弱湍 流到强湍流的全区域分析。 

# （1）Rytov方差（湍流强度量化）

Rytov方差是量化大气湍流强度的核心参数，对于平面波水平传播，其定义为 ： 

$$
\sigma_ {R} ^ {2} = 1. 2 3 C _ {n} ^ {2} k ^ {7 / 6} L ^ {1 1 / 6} \tag {2.5}
$$

其中 $C _ { \mathfrak { n } } ^ { 2 }$ 为折射率结构常数（单位： $\mathrm { m } ^ { - 2 / 3 }$ ），表征湍流强度； $\mathbf { k } = 2 \pi / \lambda$ 为光波 波数；L为传输距离。根据 $\sigma _ { \mathrm { ~ R ~ } } ^ { 2 }$ 的大小，可将湍流分为三个区域：弱湍流（ $\sigma _ { \textrm { R } } ^ { 2 } < 1$ ）、中等湍流（ $1 < \mathrm { ~ \mathbb ~ o ~ } _ { \mathrm { ~ R ~ } } ^ { 2 } < 2 5$ ）和强湍流（ $\sigma _ { \mathrm { ~ R ~ } } ^ { 2 } > 2 5$ ）。 

（2）闪烁指数（扩展Rytov方法） 

闪烁指数 $\mathrm { ~  ~ { ~ \sigma ~ } ~ } _ { \mathrm { { I } } } ^ { 2 }$ 是描述光强起伏程度的重要参数，定义为光强的归一化方差： 

$$
\sigma_ {I} ^ {2} = \frac {\langle I ^ {2} \rangle - \langle I \rangle^ {2}}{\langle I \rangle^ {2}} \tag {2.6}
$$

对于平面波从弱湍流到强湍流的全区域，Andrews扩展Rytov方法给出的闪烁 指数近似表达式为： 

$$
\sigma_ {\mathrm {I}} ^ {2} = \exp \left[ \frac {0 . 4 9 \sigma_ {\mathrm {R}} ^ {2}}{\left(1 + 1 . 1 1 \sigma_ {\mathrm {R}} ^ {1 2 / 5}\right) ^ {7 / 6}} + \frac {0 . 5 1 \sigma_ {\mathrm {R}} ^ {2}}{\left(1 + 0 . 6 9 \sigma_ {\mathrm {R}} ^ {1 2 / 5}\right) ^ {5 / 6}} \right] - 1 \tag {2.7}
$$

该公式统一了弱湍流和强湍流两个区域，避免了分区域计算的复杂性。对于 高斯光束，需考虑光束参数（束腰半径 $\mathrm { W } _ { 0 }$ 、波前曲率半径F等）的影响，闪烁指 数的完整表达式可分解为径向分量和纵向分量之和。 

（3）长期光斑尺寸与光束漂移 

大气湍流引起的光束扩展效应会导致接收端光斑尺寸增大。对于高斯光束， 长期光斑尺寸 $\mathrm { W } _ { \mathrm { L T } }$ 与自由空间光斑尺寸W的关系为： 

$$
\mathrm {W} _ {\mathrm {L T}} ^ {2} = \mathrm {W} ^ {2} \left(1 + 1. 3 3 \sigma_ {\mathrm {R}} ^ {2} \Lambda^ {5 / 6}\right) \tag {2.8}
$$

其中 $\Lambda = 2 \mathrm { L } / ( \mathrm { k W } _ { 0 } ^ { 2 } )$ 为光束参数。 

光束漂移是湍流导致光斑中心随机偏移的现象，对于准直光束，光束漂移方 差为： 

$$
\langle \mathrm {r _ {c} ^ {2}} \rangle = 2. 4 2 \mathrm {C _ {n} ^ {2} L ^ {3} W _ {0} ^ {- 1 / 3}}
$$

当考虑有限外尺度 $\mathrm { L } _ { 0 }$ 的影响时，需对上式进行修正。光束漂移会导致接收光 功率的额外损耗，在长距离链路中尤为显著。 

（4）Fried参数（大气相干宽度） 

Fried参数 $\mathrm { \bf ~ r } _ { 0 }$ 是表征大气相干性的重要参数，定义为： 

$$
r _ {0} = \left(0. 4 2 3 k ^ {2} \int_ {0} ^ {L} C _ {n} ^ {2} (z) d z\right) ^ {- 3 / 5} \tag {2.9}
$$

对于均匀水平路径， ${ \mathrm { r } } _ { 0 } = ( 0 . 4 2 3 \mathrm { k } ^ { 2 } { \mathrm { C } } _ { \mathrm { n } } ^ { 2 } { \mathrm { L } } ) ^ { - 3 / 5 }$ 。 $\boldsymbol { \mathrm { r } } _ { 0 }$ 越大表示大气相干性越好，光 束质量越高。 

# 3)大气衰减模型

大气衰减主要由大气分子和气溶胶粒子的吸收和散射引起，可用Beer-Lambert 定律描述： 

$$
\tau (\lambda , L) = e x p [ - \gamma (\lambda) \cdot L ] \tag {2.10}
$$

其中τ为大气透过率， $\gamma ( \lambda )$ 为总消光系数（单位：km⁻ ¹），L为传输距离 。总消光系数由吸收和散射两部分组成： 

$$
\gamma (\lambda) = \alpha_ {m} + \alpha_ {a} + \beta_ {m} + \beta_ {a} \tag {2.11}
$$

其中αm和αa分别为分子和气溶胶的吸收系数， $\beta _ { \mathrm { ~ m ~ } }$ 和βa分别为分子和气溶 胶的散射系数。 

# （1）Kim经验散射模型

对于雾和霾引起的衰减，Kim经验模型给出了基于能见度V的Mie散射衰减系 数： 

$$
\beta_ {a} = \frac {3 . 9 1}{V} \left(\frac {\lambda}{5 5 0 n m}\right) ^ {- q} \tag {2.12}
$$

其中q为与能见度相关的参数： 

$$
q = \left\{ \begin{array}{l l} 1. 6 & V > 5 0 \mathrm {k m} \\ 1. 3 & 6 \mathrm {k m} <   V <   5 0 \mathrm {k m} \\ 0. 1 6 \mathrm {V} + 0. 3 4 & 1 \mathrm {k m} <   V <   6 \mathrm {k m} \\ V - 0. 5 & 0. 5 \mathrm {k m} <   V <   1 \mathrm {k m} \\ 0 & V <   0. 5 \mathrm {k m} \end{array} \right. \tag {2.13}
$$

Kim模型对较浓的雾（能见度 ${ < } 0 . 5 \mathrm { k m }$ ）计算精度有限，本平台同时实现了 Naboulsi平流雾和辐射雾经验模型： 

平流雾衰减系数：

$$

$$

# 4)链路预算模型

FSO系统的链路预算用于评估系统的功率裕度，基本公式为： 

$$
\mathrm {P} _ {\mathrm {R}} = \mathrm {P} _ {\mathrm {T}} - 4. 3 4 3 \cdot \mathrm {L} \cdot \beta_ {\alpha} - \mathrm {L} _ {\text {G e o m}} - \mathrm {L} _ {0} - \mathrm {L} _ {\mathrm {p}} - \mathrm {L} _ {\mathrm {M}} \tag {2.16}
$$

其中 $\mathrm { P _ { R } }$ 为接收光功率（dBm）， $\mathrm { P _ { T } }$ 为发射光功率（dBm），各项损耗说明如 下： 

（1）大气衰减损耗 

$$
\mathrm {L} _ {\text {a t m}} = 4. 3 4 3 \cdot \mathrm {L} \cdot \beta_ {\mathrm {a}} \tag {2.17}
$$

其中β 为大气衰减系数（ $\mathrm { k m } { - } 1$ ），L为传输距离（km）。系数4.343用于将 自然对数损耗转换为dB单位（ $1 0 \mathrm { l o g } _ { 1 0 } \mathrm { e } \approx 4 . 3 4 3 $ ）。 

（2）几何损耗 

几何损耗描述了光束扩展导致接收孔径仅能截获部分光功率的情况。对于有 限束散角的光束： 

$$
\mathrm {L} _ {\text {G e o m}} = - 2 0 \log_ {1 0} \left[ \frac {\mathrm {d} _ {\mathrm {R}}}{\mathrm {d} _ {\mathrm {T}} + \theta \mathrm {L}} \right] \tag {2.18}
$$

其中d 为接收孔径直径，d 为发射孔径直径，θ为光束发散角（mrad），L为 传输距离。当接收孔径小于光斑尺寸时，几何损耗成为短距离链路的主要损耗来 源。 

（3）其他损耗 

$\mathrm { L } _ { 0 }$ ：光学元件损耗（透镜、滤光片等的插入损耗） 

$\mathrm { L } _ { \mathrm { p } }$ ：指向误差损耗，由收发机对准不精确引起 

$\mathrm { L } _ { \mathrm { M } }$ ：系统余量（通常取3~6 dB以应对环境变化） 

（4）功率裕度 

系统功率裕度定义为： 

$$
\mathrm {P M} = \mathrm {P} _ {\mathrm {R}} - \mathrm {P} _ {\text {s e n s i t i v i t y}} \tag {2.19}
$$

其中Psensitivity为接收机灵敏度（即满足特定误码率要求所需的最小接收功率） 。为保证可靠通信，通常要求 $\mathrm { P M } > 3 \mathrm { d B }$ 。
