# 摘要

无线光通信面临的一个巨大的挑战是如何减少大气湍流信道引起的信号闪烁。 湍流是由大气的温度和压力的不均匀性引起的，导致大气的折射率发生变化。湍流 引起接收光束的幅度和相位起伏。这种起伏使得接收的信号强度恶化，增加误码率 （BER），使得通信连接中断。所以研究大气随机信道本身的特性和光信号在大气信 道中的传输非常重要，对无线激光通信系统的设计和实现有较大的参考价值。 

论文首先对无线光通信的发展历程及各国研究现状，应用前景进行了概述。然 后总结分析了激光信号的大气湍流效应，大气激光载波通信原理，以及各种光载波 调制方式。鉴于通信系统误码率的重要性，利用激光在湍流中传输的误码率计算模 型，计算了采用不同调制方式的副载波调制光通信系统的误码率特性。结果表明， 大气湍流引起的光强闪烁是影响激光无线通信中信号传输误码率的主要因素，而采 用载波 BPSK 强度调制光通信系统能够很好的克服大气湍流的影响。仿真结果表明， 载波 BPSK 强度调制光通信系统误码率特性优于 OOK 调制和载波 FSK 强度调制光 通信系统。最后探讨了从信道编码方面提高无线光通信系统性能的有效方法，提出 了无线光通信系统的改进模型。 

这些理论上的研究和系统模型的改进对无线光通信系统的开发和应用具有重要 的价值。 

关键词：无线光通信；载波调制；光强闪烁；强度调制 

# A b s t r a c t

How to reduce the signal scintillation caused by atmospheric turbulence channel is a huge challenge that Wireless Optical Communication is facing. Atmospheric turbulence is caused by the non-uniformity of the temperature and pressure, resulting in a change in the refractive index of the atmosphere. Turbulence caused amplitude and phase fluctuations of the receiver beam, made the deterioration of received signal, increased the bit error rate (BER), interrupted communication. Therefore, researched in characteristics of atmosphere and optical signal transmission in the atmosphere is very important, it also has greater reference value for the design and realization of wireless laser communication system. 

First of all, the paper summarized the development process of Wireless Optical Communication, the prospects for Research and Application. And then summed up and analyzed the atmospheric turbulence effects, carrier communication theory, and a variety of optical carrier modulation. Due to the importance of bit error rate for communication systems, using the bit error rate calculation model, calculated bit error rate characteristics for different modulation of the sub-carrier modulation optical communication system. The results show that scintillation caused by the atmospheric turbulence is a major factor for BER, optical communication using sub-carrier PSK intensity modulation is able to overcome the effects of atmospheric turbulence. The simulation results show that the BER of optical communication using sub-carrier PSK intensity modulation is superior to optical communication using OOK intensity modulation and FSK intensity modulation. At last, discussed effective methods to improve the performance of wireless optical communication system, improved the model of wireless optical communication system. 

These theoretical studies and improved of the model have important value for the development and application of Wireless Optical Communication Systems.

# 1 绪论

# 1.1 引言

近年来，随着信息化发展，通信也日益向高速率，大容量，长距离方向发展。 光通信技术由于其独有的优点受到了格外青睐。在光通信领域，无线激光通信由于 既具有微波通信灵活机动的特点，又具有光纤通信码率高的优点，误码率也较低， 在空间通信、水下通信、地面通信中都得到了广泛的应用[1]。无线光通信是指利用激 光束作为载波在空间中(陆地附近或外太空)直接进行语音、数据或图像信息传送的 一种技术，又称为“自由空间光通信” (Free Space Optical Communication) [1]。 

无线光通信，以激光光波为载体，把信息调制在激光载波上来进行传输，它结 合了光纤通信和微波通信的优势，在卫星通信、本地网接入、军事通信等领域有广 阔的应用前景，成为下一代光通信的研究热点[1]。调制后的激光波束在大气中传输时， 地球周围的大气层会对光波产生一系列的影响，从而对接收信号造成影响，在接收 端不能完全恢复出原始信号造成失真。大气是由大气分子、水蒸气及杂质微粒组成 的混合物，当激光通过大气时这些微粒分子会对传输中的光波产生吸收和散射的作 用，统称为衰减作用，表现为光波能量的减弱。除此之外，由于大气折射率的不均 匀性，会对光波产生湍流效应，表现为以下几个方面：闪烁、光束漂移、光束扩展 和到达角的起伏等。 

由于在多数情况下，无线光通信需要全部或部分经过大气信道进行传播，所以 本文旨在对大气环境下的激光载波通信系统的调制方式进行研究，寻找最适合的大 气激光通信调制方案。 

# 1.2 大气湍流信道无线光通信研究现状

对于开始和最后一公里的高容量通信网络，无线激光通信技术是一种好的无线 连接技术。无线光通信面临的一个巨大的挑战是如何减少大气湍流信道引起的信号 闪烁。由于大气湍流具有的随机性和不稳定性，使得无线光通信产品的应用受到限 制。在实际大气光通信产品中，为了降低大气湍流的影响，研究人员和生产厂家采 取了一系列措施。 

1. 大气湍流对光波的影响在 20 世纪 60 年代已经进行了广泛的研究[2]，通过实 验对湍流引起的闪烁进行了研究。实验表明可以通过增大接收平均孔径来减少光信 号闪烁。但是实验表明当探测器的接收孔径达到某一特定值，增加它并不能进一步 减小闪烁水平[3, Fig.4]。在接收光圈直径从 1mm 到 1m 进行测量发现[4]：当直径从 1mm 到 10cm 对数正态变量平滑减小，当接收光圈直径从 $1 0 \mathrm { c m }$ 到 1m 时没有减小。关于 光圈平均孔径理论和实验结果在[5]中有讨论。 

2. 扩大发射光束的发散角来克服光束漂移带来的影响，当发散角大于漂移角时， 探测器总能位于接收光斑内，但是其代价是接收能量的大大减小，对系统接收灵敏 度的要求大大提高。扩大发射光束的发散角只能克服光束漂移，并不能克服湍流引 起的光强闪烁。 

3. 提高发射功率来补偿由于光束发散和光强闪烁所引起的噪声误差，但对于半 导体激光器来说，提高功率，调制带宽就会降低，目前来说，提高功率在技术上是 有限的。提高发射功率是一种简单有效的方法，但是发射功率不能无限的提高，且 提高发射功率会增加系统的成本。 

4. 采用空间分集技术，一条消息的多个副本通过多条不相关的路径被发送给接 收者，接收者通过多个接收信号联合来做决定。空间分集可使不同光束通过不同的 涡旋，使光束之间有相互补偿的可能性，从而降低了接收平面上的光强起伏。美国 AstroTerra 公司在 1997 年分别做了 1、2、4、8 和 16 个孔径的实验来说明独立孔径 的数目与空间分布对降低闪烁的影响。实验在 18 英寸的平面上对称放置 16 个激光 器，结果证明衰落冗余度为 7dB 时，不能补偿 $1 0 ^ { - 6 }$ 的突发误码，连续提高功率余量 不能消除衰落[6]。空间分集技术采用了多个发射端和多个接收端，会使系统的成本成 倍增加。 

5. 运用 ATP（Acquisition, Tracking, Pointing）技术，美国喷气推进实验室（JPL） 2000 年 5 月报道了 46.48 公里的近地大气通信模拟试验，为了降低大气湍流的影响， 其通信通道选择在水平面 2 公里以上，而且应用了跟踪与对准技术来补偿大气对信 号的损失，使起伏的动态范围由原来的 $1 7 { \sim } 2 1 \mathrm { d B }$ 降到 $1 3 \sim 1 4 \mathrm { d B } ^ { [ 7 ] }$ 。在 2000 年悉尼 奥运会上，美国 Terabeam公司成功地进行了图像传送，并在西雅图的四季饭店利用 此技术实现了向客户提供 100Mbps 的数据链接；1994 年美国加利福尼亚的 Thermo Trex 公司在相距 42 公里、海拔高度为 7000 英尺的两座山峰之间进行了数据速率为 1.2Gb/s 的传输实验。他们都成功的运用了ATP技术[8]。 

6. 采用自适应光学系统来补偿波前相位畸变，提高通信质量。由于激光的相干 性高，通过大气时受大气湍流的影响就大，从而导致光波波前相位发生畸变。自适 应光学理论就是对畸变的相位进行重构，从而降低接收面上的光强起伏。目前自适 应光学技术发展已经趋于成熟。在国内，以中国科学院光电技术研究所为代表，其 国自适应光学重点实验室近二十年来，先后成功研制了 19 单元、21 单元、37 单元、 61 单元等多代具有国际先进水平的自适应光学系统，并成功应用于惯性约束聚变、 空间目标的高分辨率观测、大气激光传输校正等，其中 61单元的变形反射镜，通光 孔径为 $1 2 0 \mathrm { m m }$ ，最大变形量是 $\pm 3 \mu \textrm { m } / \pm 3 5 0 \mathrm { v } ^ { [ 9 ] }$ ，代表了自适应光学技术发展的前 沿水平。在国外，美国的Aoptix技术有限公司，于 2000年做过自适应光学应用于光 通信的实验在克服大气光强闪烁和光束扩展方面取得了很好的效果。波士顿大学和 波士顿精密机械有限公司合作生产一种应用于自适应光学的微型变形镜，驱动器单 元数 140，通光孔径只有 $3 . 3 \mathrm { m m }$ ，最大变形量 $2 \mathrm { m m } ^ { [ 1 0 ] }$ ，此项技术使应用于光通信的 自适应光学系统小型化成为可能。 

# 1.3 论文的研究内容和意义

用通信理论来克服大气湍流引起的光信号闪烁还远未成熟，现有的有关高速数 据传输率的商业应用把重点放在使用 OOK 强度调制的光通信系统上。在这样一个 系统中，需要设置一个门限用于解调。大气湍流使得选择一个合适的门限变得非常 困难。在过去几十年里部署的成千上万无线光通信终端采用固定门限OOK系统，因 为其执行起来非常简单。然而，大气湍流对于 OOK 系统依然是一个大问题。已经存 在的系统采用了提高发射功率来克服大气湍流，但是这样做成本太高。

# 2 大气湍流对无线光通信的影响

光波在大气湍流信道中传输的问题也是光波与大气相互作用的一个重要方面。 我们假设湍流不引起光波能量的损耗，而是光波强度、相位和传播方向等参数随着 湍流而起伏。这种起伏是由于不同空气的温度不同导致空气密度的随机变换从而引 起折射指数的变化引起的。大气折射率微小变化的作用类似于处于大气中的小“透 镜”，这些“透镜”的大小近似于湍流漩涡的尺度。光波由于大气湍流的影响，会产 生大气闪烁、相位起伏等湍流效应。光强的起伏是影响无线光通信的接收性的主要 因素，所以本章主要研究大气湍流引起的光强闪烁。 

# 2.1 大气湍流概述

# 2.1.1 大气湍流模型

人类活动和太阳辐照等因素所引起的大气微小温度的随机变化将导致大气风速 的随机变化，从而形成大气的湍流运动。大气温度的随机变化导致了大气密度的随 机变化，从而导致折射率的随机变化。这些变化的累积效应使得湍流大气中的传输 光束波前也将作随机起伏，从而引起光束抖动，强度起伏(闪烁)和像点抖动等一系列 大气湍流效应[13]。描述大气湍流物理特征通常使用大气湍流模型，在大多数情况下 大气湍流是非均匀的，大尺度的湍流单元相对于入射光波一般较大，因此可用短波 近似或几何光学近似，折射效应比较明显，而小尺度的湍流单元与入射光波长相比 拟，衍射效应比较明显。图 2.1 描述了光波束在湍流中传播的机理，这对分析湍流 大气中的波传播至关重要。

由于湍流是一种快速的不规则运动，在这种运动中各种物理量是时间和空间的 随机变量，因此必须用统计规律来描述，有关湍流大气中光的传播理论非常多，其 中有重要影响的是 Tatarskii 的两部著作[14][15]。 

大气的折射率是随着空间、时间和波长而变化的，因此可以用时间、空间和波 长的随机函数来描述大气湍流的折射率[16]： 

$$
n (\mathbf {r}, t, \lambda) = n _ {0} (\mathbf {r}, t, \lambda) + n _ {1} (\mathbf {r}, t, \lambda) \tag {2.1}
$$

对波长的依赖关系可以略去，上式可以写为： 

$$
n (\mathbf {r}, t) = n _ {0} (\mathbf {r}, t) + n _ {1} (\mathbf {r}, t) \tag {2.2}
$$

一般， $n _ { 1 } ( { \bf r } , t ) < < 1$ ， $< n _ { 1 } ( { \bf r } , t ) > = 0$ 。 

大气折射率 $n _ { 1 } ( { \bf r } , t )$ 是一个随机变化的量，一般来说不是各向同性的，因为大气 湍流的大尺度部分往往使气象要素场的均匀性和各向同性遭到破坏，但是在给定的 小区域内依然可以近似地把它看作是均匀各向同性[16]，这样的区域称为惯性子区域。 

人们用雷诺(Reynolds)数来衡量湍流运动的动力学特性，它是一个无量纲数。雷 诺曾预言：液体的流动从层流转向湍流，总是在相同的雷诺数下发生。实际上现在 己经证明存在一个临界 Reynolds 数 $R _ { e _ { c } }$ ，当 $\mathrm { R e } < R _ { e _ { c } }$ 时，大气运动为层流；当 $\mathrm { R e } { > } R _ { e _ { c } }$ 时，大气运动就有可能转化为湍流[17]。但 $R _ { e _ { c } }$ 并不是一个普适常数，它的数值与流体 结构和湍流产生的方式都有关系。 

所谓的雷诺数是在研究流体的动力相似性时引入的，它定义为[17]： 

$$
R _ {e} = \rho \nu R / \mu \tag {2.3}
$$

式中ρ是流体密度，v是流体流动的特征速度，在层流状态时，它是流体流动的 速度，R 是流体流动的特征长度，μ是流体的粘性系数，它是流体流动时，流体层 之间产生的内摩擦力与相应流体层之间的速度差之比， $\mu / \rho$ 称为运动粘性系数。雷 诺数是流体流动的动力学性质的标志。不论流动的形式怎样，甚至是不同的流体， 只要雷诺数相同，则它们都是动力相似的。 

雷诺数的定义来自于流体的惯性力和内摩擦力的比值关系（惯性力是指质量对 加速度的抵抗力，其值为 2 −ma ）。大雷诺数意味着在该流体的运动中，内摩擦力的 

作用是微不足道的[17]。流动的能量不断地输送给扰动运动，于是流体地流动表现为 不稳定状态。而当雷诺数减小时，内摩擦力地作用相应增大。可能出现的扰动很快 会被内摩擦力转化为热能，于是湍流就不可能出现。 

从整体上来看湍流运动，由于内摩擦力将流动能量转化为热能消耗掉了，为了 继续维持这一湍流运动，必须要有能量来补偿耗散损失的能量。我们可以把流体运 动看作是具有速度 ${ \bf { \mathcal { { K } } } V } >$ 的平均运动与速度 ' v 的起伏运动之和。起伏量表示湍流能量， 而维持湍流所需的能量则来自于流体的平均动能。假如我们用 $T ^ { * }$ 表示单位时间内由 平均运动能转化为湍流动能的能量，用ε表示平均耗散率，则湍流的能量 $E ^ { ' }$ 随时间 的变化为[17]： 

$$
\frac {d E ^ {\prime}}{d t} = T ^ {*} - \varepsilon \tag {2.4}
$$

如果 $\boldsymbol { T } ^ { * } = \boldsymbol \varepsilon$ ，则 $E ^ { ' }$ 为常数，表明湍流处于稳定的状态。如果 $\boldsymbol { T } ^ { * } > \boldsymbol \varepsilon$ ，则表明起伏 量随着时间而增长，湍流会继续存在。反之，则表明起伏趋于减小，湍流趋于消亡。 

从上面的分析中，我们可以得到一个结论：只要整体流动的速度大于某一定值， 层流运动就有可能转化为湍流。当这种转化开始后，首先出现的湍流是与流动整体 特征尺度相当的巨大涡旋，它的尺度记作 $\mathrm { L } _ { 0 }$ ，量 $\mathrm { L } _ { 0 }$ 的数值依赖于流动的几何特征。 对于大气湍流来说，它相当于气流离地面的高度，通常在几十米到几百米的范围内， $\mathrm { L } _ { 0 }$ 成为湍流的外尺度。大于的旋涡一般不是各向同性的。以近地面大气湍流为例， 尺度大于 $L _ { 0 } \approx h$ （h 为高度）的旋涡的水平尺度可以延伸得很大，但是，与水平尺度 相比在垂直高度上有明显的不均匀，而尺度小于 $\mathrm { L } _ { 0 }$ 的旋涡，一般是各向同性的。与 外尺度大小相当的旋涡的流速具有很大的起伏能量，它构成了最初的湍流能量。 

当湍流的运动的速度越来越大时，大的湍流涡旋呈现出不稳定性从而分裂成较 小的涡旋，并将能量传递给它们，相应的 Reynolds 数变小，在湍流运动中这个过程 不断持续，Richardson 称之为湍流的级串性质[17]。这时的湍流涡旋尺度越来越小， 当内 Reynolds 数减小到某一数值时，流体粘性影响将成为运动的本质因素，动能全 部转化为热能从而导致无法再分裂为更小的涡旋。这一因素决定了涡旋的最小特征 尺度 l0，我们称之为湍流的内尺度。内尺度的一般量级为数毫米。 

Kolmogorov 在对大气湍流做了一系列研究后，建立了大 Reynolds 数下表征湍流 微结构基本性质的定律，给出了湍流的统计结构函数[18]。 

$$
D _ {r r} = C _ {v} ^ {2} r ^ {2 / 3} \quad l _ {0} <   <   r <   <   L _ {0} \tag {2.5}
$$

$C _ { \nu } ^ { 2 }$ 是依赖湍流耗散能 $\varepsilon ^ { ' }$ 的参量称作速度结构常数，是湍流总能量的测量。r 表 示湍流空间中两点的间距， $l _ { 0 }$ 和 $L _ { 0 }$ 分别为湍流的内尺度和外尺度。 

根据 Kolmogorov 的关于结构函数的“2/3”定律可以导出折射率结构函数[18] 

$$
D _ {n} (r) = <   \left(n _ {1} - n _ {2}\right) ^ {2} > = C _ {n} ^ {2} r ^ {2 / 3} \quad l _ {0} <   <   r <   <   L _ {0} \tag {2.6}
$$

$C _ { n } ^ { 2 }$ 为大气折射率结构常数，r 为统计湍流特征的两点之间的距离。 

大气湍流折射率的统计特性直接影响到激光束的传输特性，通常用折射率结构 常数的数值大小来表征湍流强度，即[19] 

弱湍流 ${ C _ { n } } ^ { 2 } < 1 0 ^ { - 1 6 } ( m ^ { - 2 / 3 } )$ 

中等湍流 2 14 2/3 10 ( ) C m n − − ≈ 

强湍流 $C _ { n } ^ { ~ 2 } > 1 0 ^ { - 1 2 } ( m ^ { - 2 / 3 } )$

# 2.2 大气湍流导致的光强闪烁

大气湍流引起的光强闪烁导致探测器平面上光密度在空间和时间上的变化。它 是由于空气折射率的随机细微变化引起波前失真，从而导致接收相位的随机变化而 造成的[18]。当光束直径 d 大于湍流直径 l 时，光束直径内就会包含许多个湍流漩涡， 每个漩涡各自对光束进行独立的散射和衍射，从而导致光束强度在时间和空间上的 随机起伏，光强忽大忽小——即光强闪烁。由于大气信道是随机的，所以接收到的 光强信号也是一个随机变量，因此必须采用统计平均值来描述其大小，一般使用光 强的概率密度函数来表述。

# 2.2.1 光强闪烁的概率分布模型

目前，在弱起伏大气湍流中常用的描述大气光强闪烁的模型是对数正态分布模 型。该分布模型是依据 Rytov 近似得到的，将对数振幅起伏 $\chi$ 表示为许多独立贡献的 叠加，由中心极限定理得到光强闪烁满足对数正态分布[17]。 

取对数振幅 $\chi$ 为均值为 $< x >$ 、标准方差为 $\sigma$ 的高斯变量，则有 

$$
P _ {\chi} (\chi) = \frac {1}{\sqrt {2 \pi} \sigma} \exp \left[ - \frac {\left(\chi - <   \chi >\right) ^ {2}}{2 \sigma^ {2}} \right] \tag {2.12}
$$

为了求振幅 

$$
A = A _ {0} \exp (\chi) \tag {2.13}
$$

的概率密度函数，引入概率变换 

$$
P _ {A} (A) = P _ {\chi} \left(\chi = \ln \frac {A}{A _ {0}}\right) \left| \frac {d \chi}{d A} \right| \tag {2.14}
$$

而 $\frac { d \chi } { d A } = \frac { 1 } { A }$ 

$$
P _ {A} (A) = \frac {1}{A \sqrt {2 \pi} \sigma} \exp \left[ - \frac {\left(\ln \frac {A}{A _ {0}} - <   \chi >\right) ^ {2}}{2 \sigma^ {2}} \right] (\mathrm {A} > 0) \tag {2.15}
$$

振幅和强度的关系为 $I = A ^ { 2 } = A _ { 0 } ^ { 2 } \exp ( 2 \chi )$ ，则有 

$$
<   I > = <   A ^ {2} > = <   A _ {0} ^ {2} > \tag {2.16}
$$

$$
<   I ^ {2} > = A _ {0} ^ {4} \exp \left(4 \sigma^ {2}\right) \tag {2.17}
$$

所以光强闪烁方差为 

$$
\begin{array}{l} \sigma_ {I} ^ {2} = <   I ^ {2} > - <   I > ^ {2} \\ = A _ {0} ^ {4} \left(\exp (4 \sigma) - 1\right) \tag {2.18} \\ = <   I > ^ {2} \left(\exp (4 \sigma^ {2}) - 1\right) \\ \end{array}
$$

归一化的光强闪烁方差 

$$
\sigma_ {I} ^ {2} = \exp \left(4 \sigma^ {2}\right) - 1 \tag {2.19}
$$

根据实验数据分析和理论近似可以得出，对应于点接收 $( D _ { r } < < \sqrt { L / k }$ ， $\sqrt { L / k }$ 为 菲涅尔半径的尺度)时，在弱湍流区，光强 I 的概率密度为 

$$
P _ {I} (I) = \frac {1}{2 \sqrt {2 \pi} \sigma I} \exp \left[ - \frac {\left(\ln \frac {I}{I _ {0}} - 2 <   \chi >\right) ^ {2}}{8 \sigma^ {2}} \right] (\mathrm {I} > 0) \tag {2.20}
$$

对数强度方差表示为 

$$
P _ {I} (I) = \frac {1}{\sqrt {2 \pi} \sigma_ {\ln I} I} \exp \left[ - \frac {\left(\ln \frac {I}{I _ {0}} + \frac {1}{2} \sigma_ {\ln I} ^ {2}\right) ^ {2}}{2 \sigma_ {\ln I} ^ {2}} \right] \tag {2.21}
$$

归一化的对数强度方差一般表达式为[22] 

$$
\sigma_ {\ln I} ^ {2} = a C _ {n} ^ {2} k ^ {7 / 6} z ^ {1 1 / 6} \tag {2.22}
$$

对于球面波和平面波，a 的取值分别为 0.496 和 1.23 

但是，也有文献指出[24]，根据实际观察，对数正态分布函数低估了概率密度函 数的峰值，与测量数据相比较可以看出，对函数分布的影响也估计不足。在雷达和 通信系统中，探测和衰减概率主要是根据给定概率密度函数的分布来计算的。低估 概率密度函数的分布将会对雷达和通信系统性能产生重要的影响。 

在[25]中的理论表明，强大气湍流引起的光信号闪烁也可以近似为一个对数正态 分布。 

# 2.2.2 大气光强闪烁的日变化

大气信道上折射率的不均匀起伏，导致接收到的光强信号的闪烁。对数光强起 伏方差是表示光强起伏强弱程度的主要特征量。它的定义： 

$$
\sigma_ {\ln I} = <   \left(\ln \left(I / I _ {0}\right) - <   \ln \left(I / I _ {0}\right) >\right) ^ {2} > \tag {2.23}

# 2.2.3 不同天气情况下的光强闪烁

随着天气的变化，光强起伏也会相应的发生变化。总体趋势是晴天的光强起伏 比阴天、雨天大很多，差 5－10 倍左右[26]。其中雨天的光强起伏方差最小，这是因 为雨水降低了地面的温度，削减了温度差异，同时雨滴还减少了空气中的不均匀成 分。这方面都有实验证明，在此不作重点介绍。 

# 3 大气载波激光通信系统原理分析

# 3.1 载波激光调制

# 3.1.1 调制的原理

通信系统的目的是为了传输信息，即要把信息从一点传送到另外一点，通常有 基带传输和调制传输。调制传输最常用的办法是把信息先调制在一个电磁波(载波) 上，然后再把被调制的载波发送到目的地，在目的地接收到电磁波后，对其进行解 调，就可以恢复出原始的发送信息。在大气激光通信中，所使用的载波是光载波， 在发送端把要发送的信息通过某种调制方式调制到光载波上，把加载信息的光载波 发射到大气信道中；在接收端对接收到的光载波信号进行检测，通过解调，最后即 恢复出原始的信息。常用的模拟调制方式有：调幅 (AM)、调频 (FM)、调相 (PM) 等，数字调制方式有：幅移键控（ASK）、频移键控(FSK)、相移键控（PSK）。在光 通信中还有一些不常见的调制方式，如强度调制（IM），极化调制（PLM）等[27][28][29]。 

# 3.1.2 调制的分类

（1）按照激光载波与光源的关系，可以分为内调制和外调制两种[30]。 

内调制是在激光振荡过程中加载调制信号，激光器的振荡参数由调制信号控制， 从而改变激光的输出特性。内调制是通过调制信号来控制激光信号的产生，在所有 内调制中，直接调制具有经济、容易实现等优点，是目前无线光通信中经常采用的 调制方式。目前的内调制主要用在光通信的注入式半导体光源中[30]。 

外调制所加载的调制信号是在激光形成以后，其具体的实现方法是把调制器放 在激光器外的光路上，把调制电压加载在调制器上，使得调制器的某些物理特性发 生相应的变化, 实现对光载波的幅度、相位及频率的调制。调制机理的不同在于调制 的顺序不同。根据调制的物理机制，外调制器可以分为三种：声光(AO)调制器，磁 光（MO）调制器，电光(EO)调制器[30]。 

在各种调制技术中，内调制经济简单且容易实现，缺点是调制速率却受到载流 子寿命及高速率下性能退化的限制。而外调制的调整更加方便，对激光器没有什么 影响，另外，外调制方式不受半导体器件的工作速率的限制，故它比使用内调制的 

调制速率高(约高一个数量级)，调制带宽要宽得多。 

（2）按照调制方式分，激光载波调制可分为：模拟调制和数字调制[29]。 

光载波的模拟调制技术又可以按照光载波被调制的参数分为：强度调制（IM）、 幅度调制(AM)、频率调制(PM)、相位调制(PM)等。 幅度调制(AM)就是光载波的振 幅随着调制信号的变化而变化。相位调制(PM)或频率调制(PM)就是使光载波的相位 或频率随着调制信号的变化规律而改变，它们又统称为角度调制[29]。 

光载波的数字调制方式主要有幅度移位键控调制（ASK）、相位移位键控调制 （PSK）、频率移位键控调制（FSK）。振幅键控调制（ASK）是使用数字电信号来直 接调制激光载波信号强度的技术。“1”符号对应的光载波最大的幅度，“0”符号对 应光载波最小的幅度。相移键控（PSK）是使用数字电信号来调制激光载波的相位， 而所有比特的频率和幅度不变，因此表现为连续的光波。对于二进制 PSK，已调信 号的相位有两种， $0 ^ { \circ }$ 和 $1 8 0 ^ { \circ }$ 。频移键控（FSK）是使用数字电信号来调制光载波 的频率，“1”符号和“0”符号分别对应一种光载波频率[29]。 

在无线光通信中还广泛采用一种在不连续状态下进行调制的数字脉冲调制，这 种调制技术是使用基带数字信号或是调制后的数字信号对激光载波进行光强度调 制。 

脉冲调制的载波是使用一种间断的周期脉冲序列，这种载波信号的某一个参量 按调制信号的规律变化而变化的调制方法叫脉冲调制。即首先使用模拟调制信号对 电脉冲序列的某一参量(幅度、宽度、频率、位置等)进行电调制，使之按照调制信号 的规律变化而变化，成为已调脉冲序列，然后再用这已调电脉冲序列对激光载波信 号进行强度调制，就可以得到相应变化的激光脉冲序列。 

此外还有一种脉冲调制方式是脉冲编码调制。这种调制是首先把模拟信号变换 成电脉冲序列，进而变成代表信号信息的(PCM 数字信号)，再对激光载波进行强度 调制来传递信息的。脉冲编码调制包括了三个过程：抽样、量化、编码[29]。 

抽样量化就是把模拟信号转换成数字信号。抽样定理告诉我们：如果对某一带 宽有限的时间连续信号（模拟信号）进行抽样，且抽样速率满足奈奎斯特采样频率， 那么根据这些抽样值就能够准确地确定原信号。这就是说，若要传输模拟信号，不 一定要传输模拟信号本身，可以只传输按抽样定理得到的抽样值。 

从模拟信号的抽样量化值到代码的转换是由编码器实现的。把量化后的信号变 换成代码的过程称为编码，其相反的过程称为译码。编码器的种类大体上可分为三 

种：逐次比较型、折叠级联型、混合型[29]。

直接检测系统中，光电检测器是最为关键的部分，它的检测灵敏度和输出信噪 比直接决定了系统性能的好坏。

# 2.直接检测系统检测灵敏度

在以上信噪比的分析中，还没有考虑到光电检测器和电路产生的噪声功率对信 噪比的影响，而仅仅考虑了输入到光电检测器的背景噪声光功率的影响。在实际应 用中，还必需考虑到信号的光电流、背景的光电流产生的噪声功率以及光电检测器 暗电流的散弹噪声功率和光电检测器和电路导致的热噪声功率，它们分别为[34]： 

$$
N _ {s} = 2 e i _ {s} \Delta f \tag {3.8}
$$

$$
N _ {b} = 2 e i _ {b} \Delta f \tag {3.9}
$$

$$
N _ {d} = 2 e i _ {d} \Delta f \tag {3.10}
$$

$$
N _ {h} = \frac {4 k T \Delta f}{R _ {L}} \tag {3.11}
$$

以上各式中，Δf 表示检测器的测量带宽， $i _ { s }$ 、 bi 、 $i _ { d }$ 分别表示信号光的电流、 背景光的电流、检测器暗电流， $T$ 为检测器的工作温度， $k$ 为玻耳兹曼常数。 

根据以上各式可以得到光电检测器输出的总的噪声功率为[33] 

$$
\begin{array}{l} N _ {o} = \left[ 2 e \left(i _ {s} + i _ {b} + i _ {d}\right) \Delta f + \frac {4 k T \Delta f}{R _ {L}} \right] R _ {L} \\ = \left\{2 e \left[ \frac {e h}{h n} \left(S _ {i} + P _ {b}\right) + i _ {d} \right] \Delta f + \frac {4 k T \Delta f}{R _ {L}} \right\} R _ {L} \tag {3.12} \\ \end{array}
$$

式（3.12）中 $P _ { b }$ 是背景光的功率。 

而光电检测器输出信号电功率为 

$$
S _ {o} = \left(\gamma S _ {i}\right) ^ {2} R _ {L} = \left(\frac {e \eta}{h \nu} S _ {i}\right) ^ {2} R _ {L} \tag {3.13}
$$

由此可以得出输出的信噪比为 

$$
S N R _ {o} = \frac {S _ {o}}{N _ {o}} = \frac {\left(\frac {e \eta}{h \nu}\right) ^ {2} S _ {i} ^ {2}}{2 e \left[ \frac {e \eta}{h \nu} \left(S _ {i} + P _ {b}\right) + i _ {d} \right] \Delta f + \frac {4 k T \Delta f}{R _ {L}}} \tag {3.14}
$$

在信号的功率很强的情况下，上式的分母中可仅考虑信号光功率产生的散弹噪 

声而可忽略所有其他噪声的影响，于是上式可以简化为 

$$
S N R _ {o} = \frac {\eta S _ {i}}{2 h \nu \Delta f} \tag {3.15}
$$

令 $S N R _ { o } = 1$ ，即可以得到在强信号条件下能够检测到的最小信号功率为 

$$
P _ {s l} = \frac {2 h \nu \Delta f}{\eta} \tag {3.16}
$$

在背景光功率很强的情况下，可仅考虑背景光导致的噪声而忽略其他噪声的影 响，由式（3.14）可以得出以背景光噪声为主要噪声源时能够检测到的最小信号功率 为 

$$
P _ {b l} = \sqrt {\frac {2 h \nu \Delta f P _ {b}}{\eta}} \tag {3.17}
$$

在热噪声很强的情况下，可以仅考虑热噪声而忽略掉其他的噪声，由式（3.14） 可以得到以热噪声为主要噪声源时能够检测到的最小信号功率为 

$$
P _ {h l} = \frac {2 h \nu}{e \eta} \sqrt {\frac {k T \Delta f}{R _ {L}}} \tag {3.18}
$$

通过以上分析，可以对实际采用直接检测系统的检测灵敏度进行估计。根据以 上的分析结果，假设在光信号功率很强时，且光电检测器工作在最理想的状态下， 如果量子效率 $\eta = 8 0 \%$ ，激光器的工作波长 $\lambda { = } 1 5 5 0 n m$ ，检测器的测量带宽 $\Delta f = 1 H z$ ， 由式（3.16）可以得到能够检测到的最小信号功率为 $P _ { s l } = 3 . 2 { \times } 1 0 ^ { - 1 9 } W$ 。但在实际上 这种极限状况是不可能达到的，因为在实际的无线大气激光通信系统中由于长距离 的传输和大气信道的衰减导致接收端信号不可能很大，而且光电检测器的噪声也不 能不考虑，所以这个极限状况只能理解为理想的工作状态。在信号光很弱而背景光 又很强时，假设 $P _ { b } = 0 . 1 W$ ，其它参数如上文，则由式（3.17）可以得出 $P _ { b l } { = } 5 . 6 5 { \times } 1 0 ^ { - 1 1 } W$ 。 在信号光很弱而热噪声又很强时，假设光电检测器的负载电阻为 $R _ { L } = 1 k \Omega$ ，工作温 度为 $T = 3 0 0 K$ ，又有波尔兹曼常量 $k = 1 . 3 9 { \times } 1 0 ^ { - 2 3 } J / K$ ，其它参数不变，则由式（3.18） 可以得出 $P _ { h l } = 3 . 4 3 \times 1 0 ^ { - 1 2 } W$ 。通过比较以上三种情况下的计算结果，我们可以发现： 

光电检测器工作在非理想状况时比工作在接收信号光功率很强的理想状态时能够检 测到的最小信号功率要低 ${ 7 } { \sim } 8$ 个数量级。因此可以得出结论，直接检测方式并不适 合对弱信号光进行检测。 

由以上对直接检测系统的基本原理以及其输出信噪比和检测灵敏度进行的分析 可以得到：在接收信号光较强时可以采用直接检测的方式进行检测，能够达到较高 的输出信噪比和检测灵敏度，而在接收信号光较弱时，则不适于采用直接检测的方 式进行检测，因为此时输出的信噪比和检测灵敏度都较低。

# 3.3.2 相干光检测系统性能分析

在相干光通信系统中，接收端既可以采用相干检测，也可以采用非相干检测， 而相干检测的性能相对要好，所以以下仅对相干光检测的原理进行分析。 

到达接收端的信号光电场可以表示为 

$$
E _ {S} = A _ {S} \exp \left[ - j \left(\omega_ {S} t + \varphi_ {S}\right) \right] \tag {3.19}
$$

$A _ { s }$ 为光信号的幅度， $\omega _ { s }$ 为光信号载波频率， $\varphi _ { s }$ 为光信号相位。本振光电场为 

$$
E _ {L} = A _ {L} \exp \left[ - j \left(\omega_ {L} t + \varphi_ {L}\right) \right] \tag {3.20}
$$

$A _ { L }$ 为本振光的幅度， $\omega _ { L }$ 为本振光频率； $\varphi _ { L }$ 为本振光场相位。如果信号光与本振 光具有相同的偏振态，则两路光场相干后投射到光电检测器上的光强度为 $\left| E _ { s } + E _ { L } \right| ^ { 2 }$ ， 检测到的功率为 $P = K { \left| E _ { s } + E _ { L } \right| } ^ { 2 }$ ， $\mathrm { K }$ 为比例常数，由式（3.19）和（3.20），P(t)可写 为[33] 

$$
P (t) = P _ {S} + P _ {L} + 2 \sqrt {P _ {S} P _ {L}} \cos \left(\omega_ {I F} t + \varphi_ {S} - \varphi_ {L}\right) \tag {3.21}
$$

（1）零差检测 

采用零差检测需要选择的本振光频 $\omega _ { L }$ 与光载波频率 $\omega _ { s }$ 相同，所以 $\omega _ { I F } = 0$ ， 由式（3.21），光电检测器产生的光电流为[33] 

$$
I = \gamma \left(P _ {S} + P _ {L}\right) + 2 \gamma \sqrt {P _ {S} P _ {L}} \cos \left(\varphi_ {S} - \varphi_ {L}\right) \tag {3.22}
$$

$\gamma$ 是光电检测器的光电转换系数。因为大气信道对激光通信的衰减较为严重， 所以一般情况下都满足 $P _ { L } > > P _ { S }$ ，因此 $P _ { s } + P _ { L } \approx P _ { L }$ 。式（3.22）左边第二项包含要 传送的信息。通过对信号电流进行解调，可恢复出原始信号。如果本振光相位和信 号光相位相同，即 $\varphi _ { S } = \varphi _ { L }$ ，则零差检测的信号电流为 

$$
I _ {S} (t) = 2 \gamma \sqrt {P _ {S} (t) P _ {L}} \tag {3.23}
$$

由零差检测的原理，这种检测方式只能对光载波的调幅信号或调相信号进行检 测，而不能检测光载波的调频信号。 

# （2）外差检测

外差检测所选择的本振光频 $\omega _ { L }$ 与信号载波光频 $\omega _ { s }$ 不同，选择合适的本振光频率 使得其差频 $\omega _ { I F }$ 落在微波范围内（GHz 级）。光检测器产生的光电流为[33] 

$$
I = \gamma \left(P _ {S} + P _ {L}\right) + 2 \gamma \sqrt {P _ {S} P _ {L}} \cos \left(\omega_ {I F} t + \varphi_ {S} - \varphi_ {L}\right) \tag {3.24}
$$

$\gamma$ 为检测器的响应度， $P _ { L }$ 和 $P _ { s }$ 是本振光功率和接收端信号功率， IF（中频）频 率 $\omega _ { I F } = \omega _ { S } - \omega _ { L }$ ，通常满足 $P _ { L } > > P _ { S }$ ，因此 $P _ { s } + P _ { L } \approx P _ { L }$ ，等式左边第一项为直流， 容易滤除。含有信息的外差电流为[33] 

$$
I _ {s} = 2 \gamma \sqrt {P _ {s} (t) P _ {L}} \cos \left(w _ {I F} t + \varphi_ {S} - \varphi_ {L}\right) \tag {3.25}
$$

由上式可以看出其包含了信号光功率、频率（与本振光频率的差频）、相位的信 息，所以外差检测方式可以用于光载波调幅、调相、调频等调制方式。 

由式（3.24）和式（3.25）可以看出： 

(1)在接收光信号功率很小时，由于光电检测器的输出电流与 $\sqrt { P _ { L } }$ 成正比，所以 能够通过增大 $P _ { L }$ 来获得足够大的输出电流。本振光在相干检测中起到了放大的作用， 

使得系统获得了混频增益，从而提高了接收灵敏度。 

(2)相干光通信系统的光载波频率和相位必须有相当高的稳定性，因此要求所使 用的光源具备非常高的频率稳定性、非常窄的光谱宽度以及一定的频率调谐范围[4]。 

(3)外差检测和零差检测的检测原理都是基于接收光信号与本振光信号之间的干 涉，必须保持它们之间的相位锁定和偏振方向匹配[35]。 

因为零差检测要求本振光频率与接收光信号频率完全相同，它的相位也要和接 收光信号的相位完全相同，实现起来相当困难。相对来说，外差检测则对本振光与 接收光的频率和相位关系没有严格要求，所以尽管外差检测的信噪比比零差检测低 3dB，但是易于实现。 

根据以上对相干检测原理的分析，可对外差检测系统的输出信噪比和检测灵敏 度进行估计。 

# 1.外差检测光电检测器的输出信噪比

假设本振光不引入噪声，令接收端信号功率、本振光的功率以及噪声功率分别 为 $P _ { s }$ 、 $P _ { L }$ 和 $N _ { i }$ 。则光电检测器面上的总输入功率为 

$$
P _ {i} = P _ {S} + P _ {L} + N _ {i} \tag {3.26}
$$

由光电检测器的平方律特性，输出信号功率为[33] 

$$
\begin{array}{l} P _ {o} = s _ {o} + n _ {o} \\ = \gamma P _ {i} ^ {2} \tag {3.27} \\ = \gamma \left(P _ {S} + N _ {i}\right) ^ {2} + 2 \gamma P _ {L} \left(P _ {S} + N _ {i}\right) + \gamma P _ {L} ^ {2} \\ \end{array}
$$

$\gamma P _ { L } ^ { 2 }$ 项是直流项，一般情况下 $P _ { \mathrm { { } } L } > > P _ { \mathrm { { } } S } + N _ { i }$ ，所以第一项较之第二项可以忽略 不记，光电检测器输出信号通过中频滤波器之后，只剩下第二项，即 $P _ { o } = 2 \gamma P _ { L } ( P _ { S } + N _ { i } )$ ，因此输出信噪比近似为 

$$
S N R _ {o} = \frac {P _ {s}}{N _ {i}} = S N R _ {i} \tag {3.28}
$$

# 2.外差检测的灵敏度

在外差检测下，光电检测器的噪声功率为 

$$
N _ {o} = \left[ 2 e \left(\frac {e \eta}{h \nu} \left(P _ {s} + P _ {L} + P _ {b}\right) + i _ {d}\right) \Delta f _ {I F} + \frac {4 k T \Delta f _ {I F}}{R _ {L}} \right] R _ {L} \tag {3.29}
$$

本振光功率的引入，使得本振光功率导致的散粒噪声大大超过其它散粒噪声及 热噪声，上式可以简化为 

$$
N _ {o} = 2 e \frac {e \eta}{h \nu} P _ {L} \Delta f R _ {L} \tag {3.30}
$$

外差检测的中频信号输出功率为 

$$
P _ {I F} = 2 \left(\frac {e \eta}{h \nu}\right) ^ {2} P _ {L} P _ {S} R _ {L} \tag {3.31}
$$

根据以上两式可求得检测器输出的中频信号功率信噪比为 

$$
S N R _ {0} = \frac {P _ {I F}}{N _ {0}} = \frac {2 \left(\frac {e \eta}{h \nu}\right) ^ {2} P _ {L} P _ {S} R _ {L}}{2 e \frac {e \eta}{h \nu} P _ {L} \Delta f R _ {L}} = \frac {\eta P _ {S}}{h \nu \Delta f} \tag {3.32}
$$

令 $S N R _ { o } = 1$ ，可求得外差检测的灵敏度极限为 

$$
P _ {\min } = \frac {h \nu \Delta f}{\eta} \tag {3.33}
$$

$\eta = 8 0 \%$ ， $\lambda { = } 1 5 5 0 \mathrm { n m }$ ， $\Delta f = 1 H z$ ，可计算得 $P _ { \mathrm { m i n } } = 1 . 6 { \times } 1 0 ^ { - 1 9 } W$ 。 

# 3.4 总结

通过以上的分析，可以看出采用非相干光通信直接检测系统在输入光功率远大 于背景噪声功率的情况下，输出信噪比位输入信噪比的一半，而采用相干光通信外 差检测系统的输出信噪比近似于输入信噪比。在 $S N R _ { o } = 1$ ， $\eta { = } 8 0 \%$ ， $\lambda { = } 1 5 5 0 \mathrm { n m }$ ， $\Delta f = 1 H z$ 时：强信号条件下非相干光直接检测系统能够检测到的最小信号功率为 $3 . 2 \times 1 0 ^ { - 1 9 }$ ，相干光外差检测系统能够检测到的最小信号功率为 $1 . 6 \times 1 0 ^ { - 1 9 }$ 。而在弱信 号条件下，相干光外差检测系统的性能要远远好于非相干光直接检测系统，因此非 相干光直接检测系统适合于在强信号条件下使用。虽然相干光通信系统的性能要优 于非相干光通信系统，但是非相干光通信系统实现起来简单，因而成本更低。

在一个通过大气湍流信道的无线光通信系统中，接收光信号强度 $P ( u , t )$ 可写为： 

$$
P (u, t) = A (u, t) P _ {s} (t) \tag {4.1}
$$

大气湍流引起的信号闪烁 $A ( u , t )$ 是一个平稳随机过程， $\mathbf { u }$ 是样本空间的一个事 件， $P _ { s } ( t )$ 是没有湍流时的接收光强度。令 $A ( u , t )$ 的自相关函数为 $R _ { \cal A } ( \tau )$ ，假定 $R _ { A } ( 0 )$ 

归一化，转化后电信号的平均功率也用归一化来分析。当 $R _ { A } ( 0 )$ 归一化，样本在任何 时间的概率分布由一个参数决定，定义为闪烁水平 $\sigma$ 。在[36]中使用没有经过调制的 激光束测量了 1Km、 $2 . 5 \mathrm { K m }$ 近地面水平湍流路径功率谱密度，测得的功率谱密度带 宽小于 1KHz。 

随机过程 $A ( u , t )$ 是一个对数正态分布过程[37],[25]。令 $x ( u , t )$ 为高斯平稳随机过程， 其自相关函数为 $R _ { x } ( \tau )$ ， $R _ { x } ( 0 ) = \sigma ^ { 2 }$ 。定义 

$$
A (u, t) = k e ^ {x (u, t)}, k > 0 \tag {4.2}
$$

在任何时间 $t = t _ { 0 }$ ,样本 $A ( u , t = t _ { 0 } )$ 是一个随机变量，其概率密度函数（pdf）为： 

$$
f (A) = \frac {1}{\sqrt {2 \pi} \sigma A} \exp \left\{- \frac {\left(\log A - m ^ {\prime}\right) ^ {2}}{2 \sigma^ {2}} \right\}, \tag {4.3}
$$

$$
m ^ {\prime} = \log k, A \geq 0.
$$

A 的 $\mathbf { n }$ 阶中心矩 $E \left\{ A ^ { n } \right\} = k ^ { n } e ^ { n ^ { 2 } \sigma ^ { 2 } / 2 }$ ，为了比较误码率的性能，特别当采用编码时， 探测器输出端 A 的二阶中心矩，因此电信号的功率是单位量。归一化 $E ( A ^ { 2 } )$ ， $k = \exp \left\{ - \sigma ^ { 2 } \right\}$ 。随机变量 $A ( u , t = t _ { 0 } )$ 的概率密度函数为： 

$$
f (A) = \frac {1}{\sqrt {2 \pi} \sigma A} \exp \left\{- \frac {\left(\log A + \sigma^ {2}\right) ^ {2}}{2 \sigma^ {2}} \right\} \tag {4.4}
$$

$$
E \left\{A \right\} = \exp \left\{- \frac {\sigma^ {2}}{2} \right\} \tag {4.5}
$$

# 4.2 采用ASK（OOK）强度调制光通信系统误码率分析

# 4.2.1 采用OOK强度调制光通信系统误码率分析

幅移键控的原理是用基带数字信号对载波的幅度进行调制，其中最简单的方法 就是开关键控 OOK。 

在现有的通过大气的光通信系统中，OOK是主要的调制方式。众所周知，大气 湍流使得这种系统的性能严重恶化[38]。这部分指出造成性能恶化的根源。 

在 OOK 系统中，信息数据序列被转换为电信号 

$$
z (u, t) = \sum_ {i = - \infty} ^ {\infty} d _ {i} g \left(t - i T _ {s}\right) \tag {4.6}
$$

$d _ { i } \in \{ 1 , - 1 \}$ 是第 i 位数据符号的信号幅度， $g ( t )$ ：门脉冲， $T _ { s }$ ：符号时间，这个 电信号驱动激光器。发射激光的强度为： 

$$
s (t) = 1 + \sum_ {i = - \infty} ^ {\infty} d _ {i} g \left(t - i T _ {s}\right) \tag {4.7}
$$

接收光束的强度为： 

$$
P (t) = \frac {P}{2} A (u, t) + \sum_ {i = - \infty} ^ {\infty} \frac {P}{2} A (u, t) d _ {i} g \left(t - i T _ {s}\right) \tag {4.8}
$$

P 是当没有湍流时的最大接收强度。因此接收电信号为 

$$
r (t) = K \left\{A (u, t) + \sum_ {i = - \infty} ^ {\infty} A (u, t) d _ {i} g \left(t - i T _ {s}\right) \right\} + n (t) \tag {4.9}
$$

K 是常量，由接收光强度和光电转换效率决定， $n ( t )$ 是加性高斯白噪声（AWGN） [11],[38]。不失一般性，接收电信号可写为 

$$
r (t) = A (u, t) + \sum_ {i = - \infty} ^ {\infty} A (u, t) d _ {i} g \left(t - i T _ {s}\right) + n (t) \tag {4.10}
$$

无湍流时， $A ( u , t ) = 1$ ，接收电信号 

$$
r (t) = 1 + \sum_ {i = - \infty} ^ {\infty} d _ {i} g \left(t - i T _ {s}\right) + n (t) \tag {4.11}
$$

直流分量可以滤掉而不影响信号的频谱。采用 OOK 调制光通信系统误码率 （BER）为 

$$
P _ {e} = Q \left(\sqrt {\frac {E _ {b}}{\sigma_ {g} {} ^ {2}}}\right) \tag {4.12}
$$

$E _ { b }$ 为（4.11）中第二项中每一位的能量， ${ \sigma _ { g } } ^ { 2 }$ 为加性高斯噪声的能量[28]。信噪 比 $\nu = E _ { b } / N _ { 0 }$ ， ${ \sigma _ { g } } ^ { 2 } = N _ { 0 } / 2$ 。 

当有大气湍流时，如果发送逻辑“1”，接收信号被湍流干扰，在这种情况下，接 收信号为 

$$
\left. r (t) \right| _ {s _ {1}} = 2 A (u, t) + n (t) \tag {4.13}
$$

探测器输出电信号的条件概率密度函数为： 

$$
p \left(r \mid s _ {0}\right) = \frac {1}{\sqrt {2 \pi} \sigma_ {g}} \exp \left\{- \frac {r ^ {2}}{2 \sigma_ {g} {} ^ {2}} \right\} \tag {4.14}
$$

$$
p \left(r \mid s _ {1}\right) = \frac {\exp \left\{- \sigma^ {2} / 2 \right\}}{2 \pi \sigma \sigma_ {g}} \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \left[ \frac {\ln^ {2} x}{2 \sigma^ {2}} + \frac {(2 r - x) ^ {2}}{2 \sigma_ {g} {} ^ {2}} \right] \right\} d x \tag {4.15}
$$

$s _ { 0 }$ ：发送逻辑 $^ { \circ } 0 ^ { \circ }$ ， $s _ { 1 }$ ：发送逻辑“1”， $\sigma$ ：闪烁水平。方程（4.15）是由对数正 态分布函数和高斯密度函数卷积得到的。在 $s _ { 0 }$ 和 $s _ { 1 }$ 时的概率密度函数是不对称的。 为了使得错误概率最小，必须精确知道湍流水平和加性噪声来选择一个合适的门限， 在[39]和自由空间光通信实践中证实那是不可能的。 

令探测门限 $\mathrm { T } { > } 0$ ，在大气湍流作用下 OOK 系统误码率为： 

$$
P _ {e} (T) = p _ {0} P \left(r > T / s _ {0}\right) + \left(1 - p _ {0}\right) P \left(r <   T / s _ {1}\right) \tag {4.16}
$$

$p _ { 0 }$ ：发送“0”的概率，将（4.14）、（4.15）代入（4.16），OOK 系统误码率为： 

$$
P _ {e} (T) = p _ {0} Q \left(\frac {T}{\sigma_ {g}}\right) + \frac {\left(1 - p _ {0}\right)}{\sqrt {2 \pi} \sigma} \exp \left\{- \frac {\sigma^ {2}}{2} \right\} \times \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \frac {\ln^ {2} x}{2 \sigma^ {2}} \right\} Q \left(\frac {2 x - T}{\sigma_ {g}}\right) d x \tag {4.17}
$$

$$
Q (x) = \int_ {x} ^ {\infty} \frac {1}{\sqrt {2 \pi}} e ^ {- \frac {t ^ {2}}{2}} d t
$$

取 $\mathrm { P _ { 0 } } \mathrm { = } ( 1 \mathrm { - } \mathrm { P _ { 0 } } ) \mathrm { = } 0 . 5$ ，可得 

$$
P _ {e} (T) = \frac {1}{2} Q \left(\frac {T}{\sigma_ {g}}\right) + \frac {1}{2 \sqrt {2 \pi} \sigma} \exp \left\{- \frac {\sigma^ {2}}{2} \right\} \times \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \frac {\ln^ {2} x}{2 \sigma^ {2}} \right\} Q \left(\frac {2 x - T}{\sigma_ {g}}\right) d x \tag {4.18}
$$

当信噪比（SNR）v 很高时， 

$$
\lim  _ {v \rightarrow \infty} P (r <   T / s _ {1}) = \lim  _ {v \rightarrow \infty} \int_ {- \infty} ^ {T} f (x) * f _ {g} (x) d x \tag {4.19}
$$

$$
\lim  _ {v \rightarrow \infty} P (r <   T / s _ {0}) = \lim  _ {v \rightarrow \infty} \int_ {T} ^ {\infty} f _ {g} (x) d x = 0 \tag {4.20}
$$

$f ( x )$ ：闪烁随机变量概率密度函数， $f _ { g } \left( x \right)$ ：加性噪声概率密度函数。在高 SNR 时，BER 由闪烁水平和门限决定。BER 极限为： 

$$
\lim  _ {v \rightarrow \infty} P _ {e} (T) = \left(1 - p _ {0}\right) Q \left(\frac {\ln 2 - \ln T}{\sigma} - \sigma\right) \tag {4.21}
$$

采用OOK 固定门限直接解调的系统通过大气湍流信道，即使SNR 非常高，BER 也不能无限小。

BER 极限是闪烁水平 $\sigma$ 和门限 T 的方程。低的门限可能导致在 SNR 低时 BER 性能很低，在 SNR 高的时候性能好。令门限 $T = e ^ { - \sigma ^ { 2 } / 2 }$ 为正态闪烁过程，不同闪烁水 平下的 BER 极限如表 4.1。 

光通信系统采用固定门限OOK强度调制不能克服大气湍流[39]，这可以从频域得 到解释。由（4.10）接收电信号的功率谱密度函数为： 

$$
R (f) = A (f) + A (f) ^ {*} Z (f) + N (f) \tag {4.22}
$$

$A ( f )$ 是湍流 $A ( u , t )$ 的功率谱密度函数， $Z ( f )$ 是传输信号 $Z ( u , t )$ 的功率谱密度 函数， $A ( f ) ^ { * } Z ( f )$ 是卷积， $N ( f )$ 噪声功率谱密度函数。湍流 $A ( u , t )$ 和传输信号 $Z ( u , t )$ 是独立的基带随机过程。 $A ( f ) ^ { * } Z ( f )$ 和 $Z ( f )$ 都在基带，接收机很难分辨这两个随机 过程，换句话说，在（4.10）或（4.22）第一部分会对解调产生干扰，因此闪烁过程 $A ( u , t )$ 严重影响解调性能。

在采用 ASK 强度调制时，主要采用的是二进制振幅键控，所以下面主要讨论 

2ASK强度调制。 

二进制幅移键控的时域表达式为 

$$
e _ {2 A S K} (t) = b (t) \cos \left(\omega_ {c} t + \varphi_ {c}\right) \tag {4.23}
$$

其中 b(t)为基带信号，表达式为 

$$
\mathrm {b (t)} = \sum_ {n = - \infty} ^ {\infty} a _ {n} \mathrm {g (t - n T} _ {s}) \quad \text {其 中} a _ {n} = \left\{ \begin{array}{l l} 0, & \text {概 率 P} \\ 1, & \text {概 率 1 - P} \end{array} \right. \tag {4.24}
$$

发射激光强度为 

$$
s (t) = 1 + \sum_ {n = - \infty} ^ {\infty} a _ {n} g \left(t - n T _ {s}\right) \cos \left(\omega_ {c} t + \varphi_ {c}\right) \tag {4.25}
$$

接收信号 

$$
r (t) = A (u, t) + \sum_ {n = - \infty} ^ {\infty} a _ {n} g \left(t - n T _ {s}\right) A (u, t) \cos \left(\omega_ {c} t + \varphi_ {c}\right) + n (t) \tag {4.26}
$$

当 $\omega _ { c }$ 的频率很高时，可以通过高通滤波器滤掉（4.26）的第一部分，这样得到 的接收信号为 

$$
r (t) = \sum_ {n = - \infty} ^ {\infty} a _ {n} g \left(t - n T _ {s}\right) A (u, t) \cos \left(\omega_ {c} t + \varphi_ {c}\right) + n (t) \tag {4.27}
$$

式（4.26）波形经过相乘器和低通滤波器后，在抽样判决器得到的输出波形为 

$$
r (t) = \left\{ \begin{array}{l l} A (u, t) + n (t), & \text {发 送 ＂} 1 \text {＂} \\ n (t), & \text {发 送 ＂} 0 \text {＂} \end{array} \right. \tag {4.28}

p \left(r \mid s _ {0}\right) = \frac {1}{\sqrt {2 \pi} \sigma_ {g}} \exp \left\{- \frac {r ^ {2}}{2 \sigma_ {g} {} ^ {2}} \right\} \tag {4.29}
$$

$$
p \left(r \mid s _ {1}\right) = \frac {\exp \left\{- \sigma^ {2} / 2 \right\}}{2 \pi \sigma \sigma_ {g}} \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \left[ \frac {\ln^ {2} x}{2 \sigma^ {2}} + \frac {(r - x) ^ {2}}{2 \sigma_ {g} {} ^ {2}} \right] \right\} d x \tag {4.30}
$$

$s _ { 0 }$ ：发送逻辑“0”， $s _ { 1 }$ ：发送逻辑“1”， $\sigma$ ：闪烁水平。令探测门限 $\mathrm { \Delta T > 0 }$ ，在大 气湍流作用下 2ASK 系统误码率为： 

$$
P _ {e} \left(T ^ {\prime}\right) = p Q \left(\frac {T ^ {\prime}}{\sigma_ {g}}\right) + \frac {(1 - p)}{\sqrt {2 \pi} \sigma} \exp \left\{- \frac {\sigma^ {2}}{2} \right\} \times \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \frac {\ln^ {2} x}{2 \sigma^ {2}} \right\} Q \left(\frac {x - T ^ {\prime}}{\sigma_ {g}}\right) d x \tag {4.31}
$$

取 $\mathrm { P = } 1 { \mathrm { - } } \mathrm { P = } 0 . 5$ ，可得

P _ {e} \left(T ^ {\prime}\right) = \frac {1}{2} Q \left(\frac {T ^ {\prime}}{\sigma_ {g}}\right) + \frac {1}{2 \sqrt {2 \pi} \sigma} \exp \left\{- \frac {\sigma^ {2}}{2} \right\} \times \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \frac {\ln^ {2} x}{2 \sigma^ {2}} \right\} Q \left(\frac {x - T ^ {\prime}}{\sigma_ {g}}\right) d x \tag {4.32}
$$

# 4.3 采用载波PSK强度调制光通信系统误码率分析

# 4.3.1 采用载波BPSK强度调制光通信系统误码率分析

可以通过分离（4.10）或（4.22）第一部分的虚假信息来提高通过大气湍流信道 光通信系统的性能，分离应该在解调器输入之前完成。既然采用强度调制，（4.10） 和（4.22）的第一部分通常存在。在实践中，强度调制因为简单而被经常采用。一种 方法是移动（4.10）和（4.22）中第二部分的频域，使其远离随机过程 $A ( u , t )$ 。这一 部分分析采用载波强度调制光通信系统通过大气湍流信道时的性能。 

考虑在图 4.1 中的光通信系统采用载波 PSK 调制，在电调制器，数据序列采用 PSK 调制，可以使用现有的微控制器芯片以很低的成本完成这个过程。PSK 信号上 变频到中频（IF） $f _ { c }$ 。由于闪烁信号 $A ( u , t )$ 的带宽只有几千赫兹，可以选择足够高的 $f _ { c }$ 来确保（4.10）和（4.22）中第一和第二部分在频域不重叠。在现有的射频技术下， 可以选择足够大的 $f _ { c }$ 来支持每秒几个 G 比特的数据速率。除非要求的数据速率非常 高， $f _ { c }$ 不应该取太大，这样可以降低终端的成本。上变频的 PSK 信号调制激光器的 强度。传输光强为： 

$$
s (t) = 1 + \alpha \left[ s _ {i} (t) \cos \omega_ {c} t - s _ {q} (t) \sin \omega_ {c} t \right] \tag {4.33}
$$

$s _ { i } ( t ) = \sum _ { j } g ( t - j T _ { s } ) \cos \Phi _ { j }$ 是同相信号， $s _ { i } ( t ) = \sum _ { j } g ( t - j T _ { s } ) \sin \Phi _ { j }$ 是正交信号， $0 < \alpha \leq 1$ ， $\omega _ { c } = 2 \pi f _ { c }$ ， $\Phi _ { j }$ 第 j 位相位， $g ( t )$ 门脉冲， $T _ { s }$ 符号时间，为避免非线性幅 度满足 $\sqrt { { s _ { i } } ^ { 2 } + { s _ { q } } ^ { 2 } } \le 1$ 。当采用强度调制，发射信号的直流部分大于 0。 

接收光强度为： 

$$
P (t) = I _ {0} A (u, t) \left\{1 + \alpha \left[ s _ {i} (t) \cos \omega_ {c} t - s _ {q} (t) \sin \omega_ {c} t \right] \right\} \tag {4.34}
$$

$I _ { 0 }$ 是常量，光电探测器输出电信号为： 

$$
I (t) = A (u, t) + \alpha A (u, t) \left[ s _ {i} (t) \cos \omega_ {c} t - s _ {q} (t) \sin \omega_ {c} t \right] + n _ {i} (t) \cos \omega_ {c} t - n _ {q} (t) \sin \omega_ {c} t \tag {4.35}
$$

$\mathbf { n } _ { i } ( t )$ 和 $\boldsymbol { \mathrm { n } } _ { q } ( t )$ 是方差为 ${ \sigma _ { g } } ^ { 2 }$ 的加性白高斯随机过程。这个系统和在上一节对应的 采用 OOK 系统传输相同的光能量。接收信号的功率谱密度为： 

$$
I (f) = A (f) + \frac {B \left(f - f _ {c}\right) + B \left(f + f _ {c}\right)}{2} + \frac {N \left(f - f _ {c}\right) + N \left(f + f _ {c}\right)}{2} \tag {4.36}
$$

$$
B (f) = A (f) * Z (f) \tag {4.37}
$$

假定 $f _ { c } > B _ { A } + B _ { B }$ ， $f _ { c }$ 中频（IF）, $B _ { \scriptscriptstyle A }$ ： $A ( f )$ 单边带带宽， $B _ { { } _ { B } }$ ： $B ( f )$ 单边带带 宽。式（4.30）中第一项可以被带同滤波器滤掉，滤波后的输出信号为： 

$$
I _ {1} (t) = \alpha A (u, t) \left[ s _ {i} (t) \cos \omega_ {c} t - s _ {q} (t) \sin \omega_ {c} t \right] + n _ {i} (t) \cos \omega_ {c} t - n _ {q} (t) \sin \omega_ {c} t \tag {4.38}
$$

把滤波后的信号下变频到基带，输入点解调器的同相信号为： 

$$
r _ {i} (t) = \alpha A (u, t) s _ {i} (t) + n _ {i} (t) \tag {4.39}
$$

正交信号为： 

$$
r _ {q} (t) = \alpha A (u, t) s _ {q} (t) + n _ {q} (t) \tag {4.40}
$$

为了得到（4.39）和（4.40）需要采用载波相位恢复。OOK 系统不需要采用载 波相位恢复。 

当所采用调制方式为载波 BPSK 时，由式（4.39）和（4.40）可得 

$$
p \left(r \mid s _ {1}\right) = \frac {\exp \left\{- \sigma^ {2} / 2 \right\}}{2 \pi \sigma \sigma_ {g}} \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \left[ \frac {\ln^ {2} x}{2 \sigma^ {2}} + \frac {(\alpha r - x) ^ {2}}{2 \sigma_ {g} {} ^ {2}} \right] \right\} d x \tag {4.41}
$$

$$
p \left(r \mid s _ {0}\right) = \frac {\exp \left\{- \sigma^ {2} / 2 \right\}}{2 \pi \sigma \sigma_ {g}} \int_ {- \infty} ^ {0} \frac {1}{x ^ {2}} \exp \left\{- \left[ \frac {\ln^ {2} x}{2 \sigma^ {2}} + \frac {(\alpha r + x) ^ {2}}{2 \sigma_ {g} {} ^ {2}} \right] \right\} d x \tag {4.42}
$$

取 $\mathrm { P } _ { 0 } { = } ( 1 { \ - } \mathrm { P } _ { 0 } ) { = } 0 . 5 ,$ ， $\alpha = 1$ ，由式（4.41）和（4.42）可得 

$$
P _ {e} = \frac {1}{\sqrt {2 \pi} \sigma} \exp \left\{- \frac {\sigma^ {2}}{2} \right\} \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \frac {\ln^ {2} x}{2 \sigma^ {2}} \right\} Q \left(\frac {x}{\sigma_ {g}}\right) d x \tag {4.43}
$$

# 4.3.2 采用载波M元PSK强度调制光通信系统误码率分析

对于采用 QPSK 调制的光通信系统，BER 为[40]： 

$$
P _ {e} = \frac {\exp \left\{- \sigma^ {2} / 2 \right\}}{2 \sqrt {\pi} \sigma} \int_ {0} ^ {\infty} x ^ {- 2} \exp \left\{- \frac {\ln^ {2} x}{2 \sigma^ {2}} \right\} Q \left(\frac {\alpha x}{\sqrt {2} \sigma_ {g}}\right) d x \tag {4.44}
$$

采用 $\mathrm { M } { > } 4$ 阶 PSK，定义接收信号向量 $r = \left[ r _ { i } r _ { q } \right]$ 。令 $V = \sqrt { { r _ { i } ^ { 2 } + { r _ { q } } ^ { 2 } } }$ ， $\theta = \arctan \frac { r _ { q } } { r _ { i } } ~ ,$ 联合概率密度为[40]： 

$$
p (V, \theta) = \frac {V}{2 \pi \sigma_ {g} ^ {2}} \exp \left\{- \frac {V ^ {2} - 2 r V \cos \theta + r ^ {2}}{2 \sigma_ {g} ^ {2}} \right\} \tag {4.45}
$$

r 是没有加性白高斯噪声时的接收信号。由此得出 V, θ ，r 的联合概率密度为[40]： 

$$
p (V, \theta , r) = \frac {V}{(2 \pi) ^ {3 / 2} \sigma_ {g} ^ {2} \sigma} \times \exp \left\{- \frac {V ^ {2} - 2 r V \cos \theta + r ^ {2}}{2 \sigma_ {g} ^ {2}} - \frac {\left(\ln r + \sigma^ {2}\right) ^ {2}}{2 \sigma^ {2}} \right\} \tag {4.46}
$$

误码率为[40]： 

$$
P _ {e} = 1 - \int_ {- \pi / M} ^ {\pi / M} \int_ {0} ^ {\infty} \frac {1}{2 \pi \sigma} \exp \left\{- \frac {r ^ {2}}{2 \sigma^ {2} g} - \frac {\left(\ln r + \sigma^ {2}\right) ^ {2}}{2 \sigma^ {2}} \right\} \tag {4.47}
$$

$$
\left\{\frac {1}{\sqrt {2 \pi}} + \frac {r \cos \theta}{\sigma_ {g}} \left[ 1 - Q \left(\frac {r \cos \theta}{\sigma_ {g}}\right) \right] \times \exp \left\{\frac {r ^ {2} \cos^ {2} \theta}{2 \sigma_ {g} ^ {2}} \right\} \right\} d r d \theta
$$

# 4.4 采用载波FSK强度调制光通信系统误码率分析

由于采用多元载波 FSK 强度调制光通信系统比较复杂，且由前面的讨论可知， 多元载波调制的性能比二元载波调制的性能要差，所以这一部分我主要讨论二元载 波FSK强度调制光通信系统的性能。 

二进制FSK的时域表达式为[29] 

$$
e _ {2 F S K} (t) = b (t) \cos \left(\omega_ {1} (t) + \varphi_ {1}\right) + \overline {{b (t)}} \cos \left(\omega_ {2} (t) + \varphi_ {2}\right) \tag {4.48}
$$

其中 b(t)为基带信号，表达式为 

$$
\mathrm {b} (\mathrm {t}) = \sum_ {n = - \infty} ^ {\infty} a _ {n} \mathrm {g} (\mathrm {t} - \mathrm {n T} _ {s}) \quad \text {其 中} a _ {n} = \left\{ \begin{array}{l l} 0, & \text {概 率 P} \\ 1, & \text {概 率 1 - P} \end{array} \right. \tag {4.49}
$$

发射激光强度为 

$$
s (t) = 1 + \sum_ {n = - \infty} ^ {\infty} a _ {n} g \left(t - n T _ {s}\right) \cos \left(\omega_ {1} t + \varphi_ {1}\right) + \sum_ {n = - \infty} ^ {\infty} \overline {{a _ {n}}} g \left(t - n T _ {s}\right) \cos \left(\omega_ {2} t + \varphi_ {2}\right) \tag {4.50}
$$

为简单起见 

$$
s (t) = 1 + \sum_ {n = - \infty} ^ {\infty} a _ {n} g \left(t - n T _ {s}\right) \cos \left(\omega_ {1} t\right) + \sum_ {n = - \infty} ^ {\infty} \bar {a} _ {n} g \left(t - n T _ {s}\right) \cos \left(\omega_ {2} t\right) \tag {4.51}
$$

接收信号 

$$
\begin{array}{l} r (t) = A (u, t) + \sum_ {n = - \infty} ^ {\infty} a _ {n} g \left(t - n T _ {s}\right) A (u, t) \cos \left(\omega_ {1} t\right) + \tag {4.52} \\ \sum_ {n = - \infty} ^ {\infty} \overline {{a _ {n}}} g (t - n T _ {s}) A (u, t) \cos (\omega_ {2} t) + n (t) \\ \end{array}
$$

式（4.52）的第一部分可以通过一个低通滤波器滤掉，得到的接收信号为 

$$
\begin{array}{l} r (t) = \sum_ {n = - \infty} ^ {\infty} a _ {n} g \left(t - n T _ {s}\right) A (u, t) \cos \left(\omega_ {1} t\right) + \tag {4.53} \\ \sum_ {n = - \infty} ^ {\infty} \overline {{a _ {n}}} g (t - n T _ {s}) A (u, t) \cos (\omega_ {2} t) + n (t) \\ \end{array}
$$

采用同步检测法，假定在（0，Ts）时间所发送的码元为“1”，则这时送入抽样 判决器进行比较的两路信号的波形分别为 

$$
\left\{ \begin{array}{l} x _ {1} (t) = A (u, t) + n _ {1} (t) \\ x _ {2} (t) = n _ {2} (t) \end{array} \right. \tag {4.54}
$$

$n _ { 1 } ( t )$ 、 $n _ { 2 } ( t )$ 是方差为 $\boldsymbol { \sigma ^ { 2 } } _ { g }$ 的高斯随机过程，故抽样值 $x _ { 1 } ( t ) = A ( u , t ) + n _ { 1 } ( t )$ 是均值 为 $A ( u , t )$ 、方差为 $\boldsymbol { \sigma ^ { 2 } } _ { g }$ 的正态随机变量；而抽样值 $x _ { 2 } ( t ) = n _ { 2 } ( t )$ 也是均值为 0、方差为 $\boldsymbol { \sigma ^ { 2 } } _ { g }$ 的正态随机变量。由于此时 ${ \bf { X } } _ { 1 } { < } { \bf { X } } _ { 2 }$ 将造成将“1”码错误判决为 $^ { 6 6 } 0 ^ { 9 }$ 码，故这时 的错误概率 $\mathrm { P _ { e l } }$ 为（ $A ( u , t )$ 由 a 代替） 

$$
\begin{array}{l} P _ {e 1} = P \left(x _ {1} <   x _ {2}\right) = P \left[ \left(a + n _ {1}\right) <   n _ {2} \right] \tag {4.55} \\ = P \left(a + n _ {1} - n _ {2} <   0\right) \\ \end{array}
$$

令 $z = a + n _ { 1 } - n _ { 2 }$ ，则 $\mathbf { Z }$ 也是正态随机变量，且均值为 a，方差为 $\boldsymbol { \sigma ^ { 2 } } _ { z }$ ， $\sigma _ { \mathrm { ~ } z } ^ { 2 } = 2 \sigma _ { \mathrm { ~ } g } ^ { 2 }$ ， 因此 $\mathbf { Z }$ 的概率密度函数 $\operatorname { f } ( z )$ 为 

$$
f (z) = \frac {1}{\sqrt {2 \pi} \sigma_ {z}} \exp \left\{- \frac {(z - a) ^ {2}}{2 \sigma_ {z} ^ {2}} \right\} = \frac {1}{2 \sqrt {\pi} \sigma_ {g}} \exp \left\{- \frac {(z - a) ^ {2}}{4 \sigma_ {g} ^ {2}} \right\} \tag {4.56}
$$

又由前面可知 $A ( u , t )$ 的概率密度函数为 

$$
f (A) = \frac {1}{\sqrt {2 \pi} \sigma A} \exp \left\{- \frac {\left(\log A + \sigma^ {2}\right) ^ {2}}{2 \sigma^ {2}} \right\} \tag {4.57}
$$

由（4.56）和（4.57）可知联合概率密度函数为 

$$
p \left(r \mid s _ {1}\right) = \frac {\exp \left\{- \sigma^ {2} / 2 \right\}}{4 \pi \sigma \sigma_ {g}} \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \left[ \frac {\ln^ {2} x}{2 \sigma^ {2}} + \frac {(r - x) ^ {2}}{4 \sigma_ {g} {} ^ {2}} \right] \right\} d x \tag {4.58}
$$

由于发送“0”被判为“1”和发送“1”被判为“0”的概率相等，所以两种情 况下的误码率相等，令 $\mathrm { p } _ { 0 } { = } 0 . 5$ 可以得到总的误码率为 

$$
P _ {e} = \frac {1}{\sqrt {2 \pi} \sigma} \exp \left\{- \frac {\sigma^ {2}}{2} \right\} \int_ {0} ^ {\infty} \frac {1}{x ^ {2}} \exp \left\{- \frac {\ln^ {2} x}{2 \sigma^ {2}} \right\} Q \left(\frac {x}{\sqrt {2} \sigma_ {g}}\right) d x \tag {4.59}
$$

# 5 大气湍流信道编码性能分析

大气湍流引起的光强闪烁会明显降低无线光通信的通信质量，由上一章的讨论 可知采用载波 BPSK 强度调制可以提高系统的性能，由于无线光通信系统的发射功 率严重受限，信噪比不能无限提高，所以为了进一步降低误码率提高通信质量，需 要采用错误纠正码。卷积码和维特比译码算法能有效纠正随机错误，并且复杂度比 较低。但是由于大气信道的记忆性会严重降低卷积码的纠错性能，所以要采用交织 编码来抑制其影响。 

# 5.1 卷积码的性能分析

衡量卷积码性能的有效方法是比特差错概率[41]。转移函数T(D,N)中因子N的指 数代表与全零路径汇合于某节点的所选错误路径中的差错信息比特数。无记忆信道 卷积码性能上限为[42] 

$$
P _ {B} <   \sum_ {i} C _ {i} P _ {i} \tag {5.3}
$$

$C _ { i }$ 由 $\left\{ d T ( D , N ) \right\} / d N \Bigr | _ { N = 1 } = \sum _ { i } C _ { i } D ^ { i }$ 决定， $P _ { i }$ 是第 i 位错误的概率。通用性能上限 [42] 

$$
P _ {B} <   \frac {d T (D , N)}{d N} \left| N = 1, D = D _ {0} = \sum_ {i} C _ {i} D _ {0} ^ {i} \right. \tag {5.4}
$$

$$
D _ {0} = \sum_ {r _ {i} \in R} \sqrt {P \left(r _ {i} \mid - 1\right) P (r \mid 1)} \tag {5.5}
$$

R 为接收信号空间， $P ( r _ { i } | \boldsymbol { x } )$ 是发送 $x \in \{ - 1 , 1 \}$ 时，接收信号的条件概率分布。 

和无线光通信系统的数据速率相比大气湍流相当缓慢，这表明在连续的数据位 里闪烁几乎是相同的。换句话说，就是信道具有记忆性。它不仅会引起随机错误， 而更主要的是造成突发错误。随机错误的特点是码元间的错误互相独立，即每个码 元的错误概率与它前后码元的错误与否无关。而突发错误则不然，一个码元的错误 往往影响前后码元的错误概率，或者说，一个码元产生错误，则后面几个码元都可 能发生错误。 

最初在加性白高斯噪声信道，卷积码的解码算法不能利用信道的记忆性，也不 

适于大气湍流信道。卷积码和维特比译码算法能有效纠正随机错误，并且复杂度比 较低。 

当瞬间闪烁很严重，会产生区间误差，严重降低译码器的性能。可以采用交织 来改变位序列编码信息块。尽管接收到的比特流可能有连续的位错误，交织处理器 改变接收信息块的位序列，使得错误的发生是随机的[43]。 

在有大气湍流时，块交织被认为可以改变维特比译码器的性能[44]。交织可以使 得接收序列符号独立，对于有很长记忆性的信道，这种假设对于找到卷积编码系统 的性能边界很重要。当交织消除信道记忆性，可以采用维特比算法解码，式（5.3） 中的边界可以采用。 

对于采用OOK调制的光通信系统，当有湍流时，条件概率函数不均匀如（4.14） 和（4.15）。定义一个码字集 $\{ C _ { 0 } , C _ { 1 } , C _ { 2 } , C _ { 3 } \}$ ， $C _ { 0 } = ( 0 , 0 , 0 )$ ， 1C =（ ，0 0 1，）， $C _ { 2 } = ( 0 , 1 , 0 )$ ， $C _ { 3 } = ( 0 , 1 , 1 )$ 。令 $P _ { i j }$ 为发送 $C _ { i }$ 接收信号为 $C _ { j }$ 的概率。尽管 $C _ { 1 }$ 、 $C _ { 2 }$ 和 $C _ { 0 }$ 、 $C _ { 3 }$ 的汉明距 离相等，当有大气湍流时 $P _ { 0 3 } \neq P _ { 1 2 }$ ， $P _ { 0 3 } \neq P _ { 3 0 }$ 。当大气湍流很强而接收器噪声功率很 低时这一状况更加明显。当信噪比很高，发射 $^ { 6 6 } 0 ^ { 9 }$ 的错误概率几乎为0，发送“1” 的错误概率由（4.20）决定。OOK 信号的不对称特性增加了解码算法的复杂度。编 码 OOK调制的性能分析牵涉到所有路径，非常复杂[44]。 

对于采用载波 PSK 强度调制的光通信系统，大气湍流影响直流分量，信息部分 在频域是独立的。接收信号的条件概率密度函数为：

# 5.3 系统改进模型

通过以上分析和仿真的结果可以看到，采用卷积编码和维特比译码可以提高系 统的误码率性能。但是由于大气湍流信道引起的光强闪烁的变化要远远慢于信息传 输的码率，也就是说大气湍流信道具有记忆性，而卷积码的解码算法只能用于无记 忆性信道。为了能够采用卷积编码和维特比译码算法必须采用交织技术，将相邻的 比特流离散化，将突发错误变成随机错误，为此将第四章所采用的大气无线光通信 系统做了相应的改动，如图 5.9。信源编码后先采用卷积编码然后采用交织编码，由 电调制器调制后送到光调制器发射出去。接收端解调后先进行解交织，然后采用维 特比译码，即可得到需要的信息。

# 6.1 论文工作总结

本论文的主要研究内容是分析大气湍流信道中采用各种载波调制方式的无线光 通信系统的性能。 

论文首先介绍了大气无线光通信在克服大气湍流的研究现状。大气信道中对接 收光信号能量影响最大的是大气湍流引起的光强闪烁，所以文中重点讨论大气湍流 信道光强闪烁特性。通过分析可知在弱湍流下，光强闪烁满足对数正态分布，也有 理论表明强大气湍流引起的光信号闪烁也可以近似为一个对数正态分布。 

论文对激光载波的调制解调方法、检测接收技术及复用方式进行了分析。通过 对非相干光通信系统和相干光通信系统的分析，得出的结论是相干光通信系统的性 能要优于非相干通信系统。 

论文分析了在大气湍流信道中，光强闪烁对采用 OOK、2ASK、BPSK、2FSK 等载波强度调制误码率性能的影响，通过仿真分析得到采用载波 BPSK 调制系统的 性能最佳。文章从理论上探讨为什么载波 BPSK 强度调制系统的性能要优于采用 OOK强度调制系统的性能。 

最后探讨怎样进一步降低系统的误码率，通过分析比较采用交织卷积编码和没 有采用交织卷积编码系统的误码率性能可以看出，采用交织卷积编码可以很好的提 高系统的误码率性能。 

# 6.2 下一步工作展望

论文分析了采用（1/3，3）的卷积码系统的误码率系统，没有进一步探讨采用其 他卷积码的误码率性能，也没有对比仿真采用不同信道编码对系统误码率的影响。 

因为大气湍流信道具有记忆性，不能直接使用卷积编码和维特比译码，为了消 除大气信道的记忆性必须使用交织编码。文章没有讨论使用不同交织编码对系统性 能的影响。 

因此，下一步工作的重点是怎样选择合适的交织编码和信道编码，怎样实现交 织编码和信道编码。
