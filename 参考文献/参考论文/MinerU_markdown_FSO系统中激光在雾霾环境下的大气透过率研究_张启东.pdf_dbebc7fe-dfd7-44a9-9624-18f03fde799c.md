Atpresent,the fogand haze weather becomes one quite harmful climate conditionat most areaofChina.Duringthe application of wireless light immunization system under thisair condition,thefogand hazeparticles would cause theattenuation oflaser,reduce the light signal ofsystem receiver,hence decrease thepropertyof BERperformance in the system.Inthe extremecondition,itmight result to“Of-target”,whichseriously impact the usageofwireless opticalcommunication system:This essaymainly focuses on the scattering characteristicsof particlesand thepenetrationrate oflaserunder thefogand haze weather by modelinganalysis. Thereceivingefficiency oflaserat different communicationdistance would beevaluated inthe air condition. 

Thisthesis analyzes the scattering characteristics for incident waves with different wavelength based onMie ScateringLawusing the simulationmethod fortypicalparticles including fogparticles,dustparticles,bituminouscoalparticles,PM2.5particlesand PM10 particles inthe fogand hazeweather.Besides,with thecombiningof thespectral distribution functionandMonte-Carlomethod,therelationshipinbetweenofthedensity ofparticlesand themaximum transmission distance of particles,transmission attenuation coefficient, transmissionrate hasbeenevaluated.Themodellingillustratesthat the extinctioncoefficient of particles increaseswith thescaleparametersand finally approaches to2,it isshown that the totalenergy atenuationcaused byparticles tolaser isabout twice asmuchas that caused by geometric crosssectionenergyattenuation..Fordifferent particles,theforward scatterismore obvious thanbackscatter,whilethe forward scatteringdominates the general scattering.When thewavelength ofincident waveishigh,theattenuationcoefficient oflaterisrelativelylow, 

hence the corresponding maximum transmission distance of laser and the transmission rate would be high. The numerical values of laser propagation parameters at specific particle concentration are given. 

Based on the simulation, the link of 80m, 340m and $2 5 0 0 \mathrm { m }$ has been selected for the measurement of the power attenuation during the transmission of laser at the real haze weather condition. Referring to the definition connection and difference of fog environment and haze environment, the data which perfectly fits the fog environment and haze environment has been selected to draw the curve of system receiption power and the laser transmission that changes with the visibility. Finally, the basic principles of laser transmission in the fog and haze weather in Xi’an area have been derived. It could be concluded that all links of laser can be corresponded with specific empirical model during the transmission of laser in the smog weather based on the comparison of fog environment measuring curve and typical fog attenuation empirical model. It is demonstrated that the empirical model is worthy of reference for further researches at different visibility and transmission distance.

# 1 绪论

# 1.1 研究背景及意义

随着通信技术壁垒的不断突破，促使国家和人民对于高速率传输海量信息的通信需求 与日俱增。利用激光可以高速传输的优点，将信息加载到激光上，得以让无线光通信在军 事、通信、气象和保密等领域大展拳脚[1]。无线光通信是一种将大容量信息加载在激光上， 并在自由空间内传输的新型高速通信技术，又称为“自由空间光”(Free Space Optical, FSO) 通信[2]。无线光通信具有高速传输、架设自由、维护成本低和无须频率许可等优点，在抢 险救灾保障通信、无线电管控区域(机场、监狱)通信和高质量保密通信等方面具有广阔的 应用前景。 

1978 年以后，我国各省市的经济总量大幅提高，工业生产规模逐年扩大，随之而来 的却是常年的高污染天气。目前，雾霾天气已经成为以河北、山西、陕西等工业能源大省 为代表，中国大部分地区秋冬季节的主要天气。很多省市甚至一连数十日出现重度雾霾天 气，对城镇建设、日常出行等广大民众及民生事业发展造成诸多危害。2013 年，“雾霾” 一词跃然纸上，走进民生并成为当年热点。2014 年 2 月，习近平主席在北京做出了大力 改善大气环境、逐步提升空气质量的重要指示。表 1-1所示为西安地区近年来各季节平均 出现霾天气的日数及全年占比，其中西安市平均每年中有 $4 7 . 1 \%$ 天为霾天气，并且以秋冬 季节为主[3]。

无线光通信系统在复杂大气环境中部署应用时，随机分布的颗粒物(如：气溶胶、雾、 霾、大气分子等)会由于吸收和散射作用，严重降低到达系统接收端的激光功率[4]。而由 水汽粒子和固体颗粒物构成的雾霾环境对激光的传输尤为敏感，形态与成份各异的多种雾 霾粒子对激光造成的衰减效应也各不相同。

# 1.2 国内外研究进展

# 1.2.1 大气粒子光学特性的国内外研究进展

在对 FSO 通信系统中激光传输特性的研究方面，多数分析模型都是建立在完整的粒 子光学特性研究上。国内外学者较多采用模拟仿真与实测实验等手段，对大气环境中存在 的气溶胶粒子及各类细微颗粒进行了光学特性方面的研究，构建了一套较为完备的理论架 构。1820 年，英国物理学家 Rayleigh 提出了著名的 Rayleigh 散射理论，从此拉开了粒子 光学散射特性分析的大幕。1905年，Gustav Mie[5]提出了一种解决均匀球状粒子散射问题 的散射理论，表明了粒子散射光强的空间分布及其与激光波长、粒径尺寸、复折射率之间 的复杂关系。1994 年，Kawata 等人[6]通过对多次散射模型进行分析，利用地球反射率极 化和方向性探测器数据库提出了一种得到气溶胶颗粒光学特性参数的方法。1996 年， Kopeika 等人[7]通过粒子测量系统和气象参数对粒子的粒径分布进行了测量，并计算了气 溶胶粒子的相关衰减系数，利用MODTRAN软件计算了由分子吸收产生的消光系数。2001 年，Glen等人[8]研究了碳粒子和 $\mathrm { ( N H _ { 4 } ) _ { 2 } S O _ { 4 } }$ 粒子在不同混合状态下的光学性质。结果表明， 在不同环境中，粒子处于内混合和外混合两种状态时具有不同的光学性质，干燥环境下光 学特性差异超过 $2 5 \%$ ，潮湿环境下差异超过 $5 0 \%$ 。2008 年，Pierre[9]对常用的气溶胶浑浊 度系数、大气水汽浓度及城市气溶胶光学厚度之间的转换函数进行了仿真研究，结果表明 该转换函数的整体性能相较于辐射传输模型提高了 $3 \%$ 。黄建平[10]和张苗[11]等人分别在 2009年和2011年对我国的西北地区和武汉城区进行了气溶胶光学特性的实地测量。其中， 黄建平利用天空背景辐射计和平面平行辐射传输模式研究了气溶胶光学厚度、平均不对称 因子等特性；张苗等人着重于测定气溶胶的实际散射系数与吸收系数。2011 年，董浩等 人[12]通过将单波长激光雷达的背景散射信号和 Fascode复杂气溶胶模型结合，提出了一种 多波长激光大气气溶胶消光系数的实时反演方法，并将反演结果与实时测量的数据进行比 较，验证了该方法的可行性。2012 年，刘西川等人[13]以 Mie 散射为基础，仿真分析了当 入射光处于可见光和近红外波段内的粒子散射特性。结果表明，粒子的散射衰减能力与其 数密度成正比；而粒子散射时的不对称因子只与其符合的谱分布函数有关，与数密度无关。 2013 年，Naohiro等人[14]结合高光谱成像仪与分光辐射计的数据，研究了当气溶胶粒子处 于对流层时的物理特性和光学特性。2013 年，蒋大刚等人[15]针对弱湍流到强湍流环境下 大气参数的测量，采用菲涅尔透镜与大口径光学仪器接收激光光斑，通过对光斑的辐照度 与漂移量进行分析，提出了一种大气消光系数和大气折射率结构常数 $C _ { \mathrm { n } } ^ { 2 }$ 的同步测量技术。 2014 年，蒋大刚等人[16]对光学仪器的光圈进行改进，提出了利用孔径可变的光学仪器测 量大气消光系数和 $C _ { \mathrm { n } } ^ { \ 2 }$ 的可变孔径法。2017年，程晨等人[17]在可见光到红外波段计算了具 有一定谱分布的硫酸盐气溶胶、海盐气溶胶和水溶性气溶胶随相对湿度和波长变化的光学 参数。通过对光学特性的分析得出，在使用 Monte-Carlo 方法模拟大气的传输过程时，考 虑相对湿度和波长变化对模拟结果的影响十分必要。 

针对大气粒子的光学特性研究，主要是研究粒子的各项衰减系数。目前，较多地从物 理模型、数据库和实验测量数据出发，对粒子的光学特性进行反演和分析。本文将以 Mie 散射理论为主，着重分析雾霾环境中的典型粒子在不同入射激光波长时的粒子散射特性。 

# 1.2.2 激光大气透过率的国内外研究进展

在研究激光大气传输时的透过率问题方面，国内外学者主要从粒子的光学特性出发， 以气溶胶数据库为基础，辅以各类大气传输仿真软件(MODTRAN、HITRAN2008 等)对激 光大气透过率进行研究。1997年，Strow等人[18]通过将气溶胶光学数据库中提取的光学厚 度进行奇异值分解，并在数据库中对温度和气压参数进行检索，提出了一种快速计算近红 外大气透过率的方法。所提方法有效地降低了计算复杂度，极大地提高了计算效率。2009 年，陈山等人[19]在分析 $_ { \mathrm { H } _ { 2 } \mathrm { O } }$ 、 $\mathrm { C O } _ { 2 }$ 等气体分子的吸收衰减及复杂大气环境散射衰减效应 的基础上，提出了一种大气透过率简算模型。2010 年，董浩等人[20]以大气传输模型为基 础，提出了一种实时反演多波长激光大气透过率的数学模型。将反演结果与实测数值对比 分析，验证了该模型可以高精度地实时反演可见和红外波段的大气透过率。2011 年，Aliass 等人[21]利用大气传输仿真软件 MODTRAN定性研究了马来西亚部分地区的大气透过率。 结果显示，乡村地区的气溶胶吸收系数明显小于城镇地区，对当地的烟感测绘和气溶胶建 模具有重要意义。2013 年，王晓蕊等人[22]研究了发射天顶角、激光波数、高度及降雨量 对激光大气透过率的影响。研究结果显示，激光透过率与能见度数值成正比；当能见度与 发射天顶角一定时，激光透过率与传输高度成正比；而当天顶角增加到 $8 9 ^ { \mathrm { { o } } }$ 时，激光透过 率在激光传输达到一定的高度和能见度数值时逐渐降低。2014 年，王奇杰等人[23]利用 HITRAN2008 数据库与空军地球物理实验室提供的大气模型计算大气分子透过率，利用 最小二乘法将计算得到的透过率拟合到不同的大气模型下，得到了含有拟合系数的修正 Malkmus 模型。研究结果表明，修正模型的均方误差小于传统 Malkmus 模型 1-2 个数量 级，且修正模型均适用于不同的大气分子模型。2016 年，郭静等人[24]提出了一种适用于 雨滴环境的大气透过率修正模型，仿真了不同降雨环境下波长为 $0 . 5 3 2 \mu \mathrm { m }$ 和 $1 . 0 6 4 \mu \mathrm { m }$ 激 光的衰减系数。结果表明，降雨环境中的激光衰减不仅与降雨量有关，还与光束的发散角 和链路长度有关。通过实验，证明了改进后的修正模型可以更好地为雨中激光传输应用提 供理论依据和技术支撑。 

由于大气湍流这一不可忽视的大气现象会对激光造成光束扩展、漂移和光强起伏等影 响，使得无线光通信系统性能受限，严重时会中断通信。在这种通信背景下，国内外部分 学者将大气透过率与大气湍流的强弱等方面结合起来，对大气湍流介质中的激光传输透过 率进行了研究。2009年，Alaineejad等人[25]研究了圆形和椭圆对称性部分相干平顶光束在 湍流介质中传输时的平均透过率。研究发现，激光大气透过率与平顶光束阶数 $N$ 成反比。 而对于完全相干平顶光束存在临界值 $R _ { c }$ ，当接收器的径向集电孔径 $R { < } R _ { c }$ 时，透过率与平 顶光束阶数成正比关系。当 $R { > } R _ { c }$ 时，透过率与阶数成反比。2012 年，Yalçın Ata 等人[26] 研究了在非Kolmogorov 湍流介质中运行密集波分复用系统时的透过率。其结果显示，在 

波长一定的情况下，当非 Kolmogorov 湍流呈现较高的幂律时，透过率明显降低。而当湍 流强度较高时，对于密集波分复用系统和非 Kolmogorov 湍流的所有幂律，透过率均有所 降低。2015 年，Yalçın Ata等人[27]利用扩展Huygens–Fresnel原理和斜程 Hufnagel-Valley 模 型计算了在大气湍流介质中斜程传输多高斯波束时的平均透过率。研究结果显示，在通信 链路达到 $2 0 \mathrm { k m }$ 时，通过使用具有较大平坦度的大尺寸平顶高斯光束和较大外尺寸的环形 光束，可以有效地提高湍流环境下的透过率。2017 年，李寒山等人[28]在研究大气湍流和 激光衰减效应的基础上，提出了湍流介质衰减效应下的信噪比函数和误码率函数，推导出 大气湍流介质中传输误码率与不同光强波动系数的最佳阈值之间的关系。 

# 1.2.3 雾霾环境中激光传输特性的国内外研究进展

在研究雾霾环境中的激光大气透过率问题时，国内外学者在仿真研究和实验测定方面 做了很多工作。仿真研究主要利用 Mie散射理论、Monte-Carlo方法和粒子谱分布函数等 研究雾粒子和霾粒子的光学特性，并在此基础上进一步分析大气透过率的算法模型。2002 年 3月至2003年4月，Fischer[29]利用实验装置测量了激光在美国华盛顿地区雾环境中的 传输特性，并将实测数据与经验模型进行对比。研究证明在能见度 $V { < } 6 5 0 \mathrm { m }$ 时，Kruse 模 型与实测值相差较大；当能见度 $V { > } 6 5 0 \mathrm { m }$ 时，Kruse 模型与实测值匹配程度较高。2009 年，柯熙政等人[30]分析了激光在雾环境中的衰减计算公式，根据 Adarsh Deepak 等人提出 的前向散射系数得到衰减特性的修正模型，并指出紫外光和可见光的衰减基本无差异。 2010 年，张文刚等人[31]利用 MODTRAN软件研究了激光透过率、能见度与雾层厚度之间 的关系，通过对仿真数据进行拟合，推导出利用能见度和雾层厚度反演激光透过率的数学 模型。2011年，孙超等人[32]以雾滴分布模型、粒径分布与能见度之间的经验关系为基础， 计算分析了衰减特性与大气能见度和链路长度之间的关系。2012 年，Ijaz 等人[33]在实验 室中利用透明玻璃舱模拟了雾环境，对比分析了不同能见度下的雾环境对波长为 $0 . 8 3 \mu \mathrm { m }$ 和 $1 . 5 5 \mu \mathrm { m }$ 激光的衰减影响。实验结果表明，当玻璃舱中的雾环境从浓雾变为中雾时，波 长为 $0 . 8 3 \mu \mathrm { m }$ 激光的衰减更强。2012年，Xu 等人[34]利用上海城区连续测量的气溶胶数据， 对气溶胶粒子的散射效率和单次反照率等光学参数进行了分析。对比不同天气下的 PM2.5 浓度和黑碳浓度数据，指出 PM2.5 粒子和黑碳粒子是雾霾环境中的主要粒子，为本文后 续的研究提供了一定的理论基础。2013年，Zhao 等人[35]利用北京城郊地区PM2.5 粒子的 质量浓度、散射系数以及黑碳浓度的观测数据，研究了北京地区雾霾天气下 PM2.5 粒子 的光学特性。结果表明，粒子的光学特性受天气变化的影响较为明显，PM2.5 粒子浓度、 散射系数等光学参数在雾霾天气时明显高于非雾霾天气。而不同气象条件下的雾霾天气， 造成雾霾形成过程中 PM2.5 粒子浓度和消光强度、散射系数以及吸收系数的空间分布呈 现不同特点。2014年，Guo 等人[36]基于Mie散射理论分析了不同粒径尺寸及复折射率下， PM2.5 粒子在单颗粒和混合颗粒两种状态下的消光特性。结果表明，单体 PM2.5 粒子的 消光系数会随粒径大小和对应波长下的复折射率的变化而变化；粒子在混合状态下的消光 

系数随粒子粒径的增大而快速提高，最终接近于某一常数。2014 年，董飞彪等人[37]采用 蒙特卡洛多次散射模型仿真分析了波长为 $1 . 0 6 4 \mu \mathrm { m }$ 的脉冲激光在雾环境中的传输衰减特 性，并对脉冲激光信号在不同雾环境和接收视场角时的脉冲响应特性进行了仿真分析。 2014 年，魏海亮等人[38]根据电介质极化理论推导出海雾介质等效介电常数的一般计算模 型，并进一步得到激光雾中传输衰减的算法模型。2016 年，吴呈龙等人[39]研究了辐射雾 对激光的吸收特性，其研究覆盖了高达 $3 0 \mathrm { T H z }$ 的激光波段。结果显示，当能见度相同时， 高能激光在雾中的衰减效率主要与雾滴粒子介电常数的虚部成正比；随着激光波长的降 低，辐射雾对激光的吸收特性与雾滴粒子等效介电常数随激光频率的变化保持较好的一致 性。2017 年，童广德等人[40]以近程雷达应用为背景，提出了一种快速计算激光多次散射 的数学模型。仿真结果显示：雾滴粒子的二次等效散射截面与雾滴粒子浓度的 2/3 次方成 正比，而二次散射系数随能见度的衰减相较于一次散射系数更快。2018 年，王婷等人[41] 在 Mie 散射理论的基础上，计算了波长为 $1 . 0 6 4 \mu \mathrm { m }$ 、 $3 . 8 \mu \mathrm { m }$ 、 $1 0 . 6 \mu \mathrm { m }$ 的激光在平流雾与辐 射雾中的消光系数和多粒子体系下的散射相函数。并用 Monte-Carlo 方法分析了考虑多次 散射效应时，能见度、接收屏边长与链路长度对激光大气透过率的影响。研究表明，当能 见度与传输距离均为 $1 \mathrm { k m }$ 时， $1 . 0 6 4 \mu \mathrm { m }$ 激光在平流雾中的接收屏粒子数与接收屏边长成 正比；对于前向散射能力较强的粒子，用 Monte-Carlo 方法计算得出的多次散射下的透过 率值更为准确。2018 年，王浩等人[42]以非致命激光致盲武器在雨雾天气下的打击效果为 背景，建立了波长为 $0 . 5 3 2 \mu \mathrm { m }$ 的激光在雨雾环境中的衰减模型。仿真结果显示，激光的衰 减程度与链路长度和强度成正比。

# 2 雾霾环境及其粒子分布特性

雾霾是雾与霾的组合词，多见于中国内陆地区，我国已将雾和霾一起作为灾害性天气 进行预报。为了更好的在雾霾环境中应用无线光通信系统，研究雾霾环境中的粒子光学特 性，就必须对雾和霾两种环境进行针对性研究。本章将对雾环境和霾环境的基本特征及其 粒子的物理特性和分布特性进行概述，为后续的仿真分析提供一定的理论基础。 

# 2.1 雾霾环境的基本特征

# 2.1.1 雾与霾的基本特征

雾是由近地面悬浮着的水滴和冰晶粒子经过长时间沉降形成的一种气溶胶系统。根据 形成雾环境的过程划分，可以将其分为以下几类[43,44]：(1)辐射雾：由于白天气温较高， 地表会受到太阳持续不断的热辐射。当太阳落山后，地表逐渐冷却，造成近地面潮湿空气 中的水汽冷凝析出所形成的雾；(2)平流雾：潮湿空气中的水汽受到流经的低温环境的影 响而形成的雾；(3)蒸发雾：水面上积聚的水蒸气在冷空气的作用下冷凝析出形成的雾；(4) 锋面雾：在冷暖空气的交界面附近，较大的温差导致温暖的潮湿空气冷凝析出形成的雾。

# 2.2 雾霾粒子的基本物理特性

# 2.2.1 雾粒子的基本物理特性

# (1) 雾滴的大小

雾环境的发展趋势和稳定程度决定了雾滴粒子的尺寸大小。一般情况下，对于形成初 期的雾环境而言，雾滴的大小分布较为均匀。而对于即将消散的雾环境，雾滴粒子之间的 尺度差距会变大，且谱分布会变得较宽[58]。 

通过大量观测实验数据表明[59]：当能见度较低时，雾滴粒子的粒径取值在 $1 \mu \mathrm { m } \cdot 6 \mu \mathrm { m }$ 之间；当能见度较高时，雾滴粒子的半径在 $2 5 \mu \mathrm { m }$ 左右。而当环境温度 $T { > } 0 ^ { \circ } \mathrm { C }$ 时，雾滴半 径 $r$ 通常在 $7 \mu \mathrm { m } - 1 5 \mu \mathrm { m }$ 之间；当环境温度 $T { < } 0 ^ { \mathrm { { o } } } \mathrm { { C } }$ 时， $r$ 通常在 $2 \mu \mathrm { m } - 5 \mu \mathrm { m }$ 之间。 

# (2) 雾滴的复折射率

对雾滴粒子而言，随着凝聚核的不同，粒子的成分也会有所差异，对应的复折射率也 会有所不同。如式(2-4)所示为粒子折射率的定义表达式。 

$$
m = \frac {C}{C _ {p}} \tag {2-4}
$$

其中， $C$ 是光在真空中的传播速度， $C _ { p }$ 是光在介质中的传播速度。通常将 $m$ 写为复数形 式，如式(2-5)所示： 

$$
m = m ^ {\prime} - n ^ {\prime} \sqrt {- 1} = m ^ {\prime} - n ^ {\prime} \mathrm {i} \tag {2-5}
$$

其中， $m$ '和 $n$ '分别反映了粒子的散射作用和吸收作用。当 $n { \stackrel { \prime } { = } } 0$ 时，说明粒子只对激光存 在散射作用而无吸收作用。 

由于雾滴一般是由水滴粒子构成，所以雾滴的复折射率通常由水滴的复折射率来代 替。图2-1给出了部分波长对应的雾滴粒子复折射率曲线，如图所示为不同激光波长时， 雾滴粒子复折射率的实部和虚部[58]。

# 2.3 雾霾粒子的分布特性

# 2.3.1 雾粒子的分布特性

目前常用的雾滴谱分布函数为 Gamma 分布，如式(2-7)所示[30]。该分布适合于大陆与 海洋环境下的雾，用单一的能见度即可确定。 

$$
n (r) = a r ^ {\alpha} \exp (- b r ^ {\beta}) \tag {2-7}
$$

其中， $n ( r )$ 表示为单位体积、单位半径间隔内的雾滴粒子总数， $r$ 为雾滴粒子半径， $a$ 和 $^ b$ 为确定雾滴谱形状的参数。在大多数情况下，取 $\scriptstyle { a = 2 }$ 、 $\beta { = } 1$ ，即： 

$$
n (r) = a r ^ {2} \exp (- b r ^ {1}) \tag {2-8}
$$

$$
a = \frac {9 . 7 8 1}{V ^ {6} W ^ {5}} 1 0 ^ {1 5} \tag {2-9}
$$

$$
b = \frac {1 . 3 0 4}{V W} 1 0 ^ {4} \tag {2-10}
$$

其中， $V$ 为雾的能见度，单位为 km；W 为含水量，单位为 ${ \mathrm { g } } / { \mathrm { m } } ^ { 3 }$ 。 

平流雾(advection fog, adv. fog)的雾滴尺寸分布与能见度的经验公式为[40]： 

$$
n (r) _ {\text {a d v .}} = 1. 0 5 9 \times 1 0 ^ {7} V ^ {1. 1 5} r ^ {2} \times \exp (- 0. 8 3 5 9 V ^ {0. 4 3} r) \tag {2-11}
$$

辐射雾(radiation fog, rad. fog)的雾滴尺寸分布与能见度的经验公式为[40]： 

$$
n (r) _ {\text {r a d .}} = 3. 1 0 4 \times 1 0 ^ {1 0} V ^ {1. 7} r ^ {2} \times \exp (- 4. 1 2 2 V ^ {0. 5 4} r) \tag {2-12}
$$

# 2.3.2 霾粒子的分布特性

分布特性能够直观的反应霾粒子的物理特性，类型不同的霾粒子之间也存在较大差 异。霾粒子具有多种谱分布函数，在研究中常选用具有一定分布特性的经验公式进行表达。 常用的粒子谱分布为：幂指数分布、Junge分布、Γ 分布和对数正态分布[63]。 

# (1) 幂指数分布

幂指数分布如式(2-13)所示： 

$$
n (r) = a r ^ {(- \alpha + 1)} \tag {2-7}
$$

其中， $n ( r )$ 代表粒子数密度分布， $r$ 为粒子半径。粒子半径为 $0 . 0 1 \mu \mathrm { m } \cdot 1 \mu \mathrm { m }$ 的霾粒子与平流 层中的气溶胶粒子基本符合幂指数分布。当粒子半径 $r$ 为 $0 . 1 \mu \mathrm { m } - 1 \mu \mathrm { m }$ 时，参数 $\scriptstyle { a = 3 }$ ；当粒 子半径 $r$ 为 $0 . 0 2 \mu \mathrm { m } \cdot 0 . 1 \mu \mathrm { m }$ 时，参数 $\scriptstyle { a = 4 }$ 。 

# (2) Junge 分布

Junge 分布，又称荣格分布，其形式如式(2-14)所示[64]： 

$$
n (r) = \frac {d N}{d r} = C r ^ {- \tau} \tag {2-14}
$$

其中， $N$ 为粒子整体数目； $C$ 为不同粒子浓度对应下的常数； $\tau$ 的值决定了曲线走势的陡 缓程度。一般将Junge分布表示为对数形式： 

$$
n (r) = \frac {d N}{d \ln r} = 0. 4 3 4 C r ^ {- \tau - 1} \tag {2-15}
$$

其中，当 $\tau \in ( 3 \mathrm { , 4 } )$ 时，该分布符合霾粒子分布；当 $\tau { \approx } 2$ 时，该分布符合雾粒子分布。 

# (3) Γ 分布

Γ 分布如式(2-16)所示[65]： 

$$
n (r) = \frac {d N (r)}{d r} = a r ^ {\alpha} \exp (- b r ^ {\beta}) \tag {2-16}
$$

其中， $N$ 为单位空间内的粒子总数， $r$ 为粒子半径，a、b、α、 $\beta$ 则需要根据粒子种类不同 而确定相对应的参数取值。若引入如式(2-17)所示的粒子众数半径 $r _ { c }$ ： 

$$
r _ {c} ^ {\beta} = \frac {\alpha}{b \beta} \tag {2-17}
$$

则式(2-16)可改写为： 

$$
n \left(r _ {c}\right) = \frac {d N \left(r _ {c}\right)}{d r} = a r _ {c} ^ {\alpha} \exp \left(- \frac {\alpha}{\beta}\right) \tag {2-18}
$$

霾粒子在大气中通常具有随机分布的特性，并且随时间、地点及形成原因有很大差异。 目前，常用的霾粒子谱分布为对数正态分布，其概率谱密度函数如式(2-19)所示[66]：

$$

其中， $r$ 为粒子半径， $r _ { m }$ 为粒子平均半径， $\sigma$ 为标准差。

# 3.1 大气激光散射理论

光的散射是指激光在大气介质中传输时，部分光束偏离原传播方向的现象，偏离原方 向的光波称为散射波[67]。

在Mie散射理论中引入了粒子的粒径尺度参数 $X$ 为：

其中， $\lambda$ 为射激光波长， $r$ 为散射粒子半径。 

当偏振光 $I _ { 0 }$ 照射到散射粒子上时，球形粒子的散射光强为[72]： 

$$
I = \frac {\lambda^ {2}}{4 \pi^ {2} l ^ {2}} I _ {0} \left[ i _ {1} (\theta) \sin^ {2} \varphi + i _ {2} (\theta) \cos^ {2} \varphi \right] \tag {3-7}

而非偏振光照射到散射粒子上时，球型粒子的散射光强为：

其中， $l$ 为散射粒子到观测点的距离， $\theta$ 为散射角， $S _ { 1 }$ 和 $S _ { 2 }$ 为散射光振幅函数的垂直和 水平分量，振幅 $i _ { 1 } ( \theta )$ 和 $i _ { 2 } ( \theta )$ 是描述散射面水平强度分量和垂直强度分量的散射光强函数。 

式(3-8)中的 $S _ { 1 } ( \theta )$ 、 $S _ { 2 } ( \theta )$ 为散射光振幅函数，如式(3-9)和式(3-10)所示： 

$$
S _ {1} (\theta) = \sum_ {n - 1} ^ {\infty} \frac {2 n + 1}{n (n + 1)} \left(a _ {n} \pi_ {n} + b _ {n} \tau_ {n}\right) \tag {3-9}
$$

$$
S _ {2} (\theta) = \sum_ {n - 1} ^ {\infty} \frac {2 n + 1}{n (n + 1)} \left(a _ {n} \tau_ {n} + b _ {n} \pi_ {n}\right) \tag {3-10}
$$

其中， $a _ { n }$ 和 $b _ { n }$ 称为 Mie 散射系数，如式(3-11)所示： 

$$
\left\{ \begin{array}{l} a _ {n} = \frac {\psi_ {n} (X) \psi_ {n} ^ {\prime} (m X) - m \psi_ {n} ^ {\prime} (X) \psi_ {n} ^ {\prime} (m X)}{\xi_ {n} (X) \psi_ {n} ^ {\prime} (m X) - m \xi_ {n} ^ {\prime} (X) \psi_ {n} ^ {\prime} (m X)} \\ b _ {n} = \frac {\psi_ {n} ^ {\prime} (X) \psi_ {n} (m X) - m \psi_ {n} (X) \psi_ {n} ^ {\prime} (m X)}{\xi_ {n} ^ {\prime} (X) \psi_ {n} (m X) - m \xi_ {n} (X) \psi_ {n} ^ {\prime} (m X)} \end{array} \right. \tag {3-11}
$$

其中， $m$ 为复折射率， $X$ 为粒子尺度参数。 $\psi _ { n } ( z )$ 、 $\xi _ { n } ( z )$ 分别为半整数阶贝塞尔函数与第 一类汉克尔函数， $\psi _ { n } ^ { ' } ( z )$ 、 $\xi _ { n } ^ { ' } ( z )$ 分别为 $\psi _ { n } ( z )$ 、 $\xi _ { n } ( z )$ 的一次导数，如式(3-12)所示：

\left\{ \begin{array}{l} \psi_ {n} (z) = z j _ {n} (z) = z \sqrt {\frac {\pi}{2 z}} J _ {n + 0. 5} (z) \\ \xi_ {n} (z) = z h _ {n} ^ {(1)} (z) \\ \psi_ {n} ^ {\prime} (z) = z j _ {n - 1} (z) - n j _ {n} (z) \\ \xi_ {n} ^ {\prime} (z) = z h _ {n - 1} ^ {(1)} (z) - n h _ {n - 1} ^ {(1)} (z) \end{array} \right. \tag {3-12}
$$

其中， $J _ { \mathrm { n + 0 } . 5 } ( z )$ 表示为半奇阶的第一类贝塞尔函数， ${ h _ { n } } ^ { ( 1 ) } ( z )$ 表示为半奇阶的第二类汉克尔函 数，如式(3-13)所示。 

$$
\left\{ \begin{array}{l} h _ {n} ^ {(1)} (z) = j _ {n} (z) + i y _ {n} (z) \\ y _ {n} (z) = \sqrt {\frac {\pi}{2 z}} Y _ {n + 0. 5} (z) \end{array} \right. \tag {3-13}
$$

$$
j _ {n + 1} (z) = \frac {2 n + 1}{z} j _ {n} (z) - j _ {n - 1} (z) \tag {3-14}
$$

$$
y _ {n + 1} (z) = \frac {2 n + 1}{z} y _ {n} (z) - y _ {n - 1} (z) \tag {3-15}
$$

$j _ { n } ( z )$ 和 $y _ { n } ( z )$ 的初始值为： 

$$
\left\{ \begin{array}{l} j _ {0} (z) = \frac {\sin z}{z}, j _ {1} (z) = \frac {\sin z}{z ^ {2}} - \frac {\cos z}{z} \\ y _ {0} (z) = - \frac {\cos z}{z}, y _ {1} (z) = - \frac {\cos z}{z ^ {2}} - \frac {\sin z}{z} \end{array} \right. \tag {3-16}

# 3.1.2 多次散射

当激光传输介质较为稠密时，介质内部的粒子除了会受到入射光的照射，还会受到光 源附近粒子所发出的散射光的照射。此时，整个环境中的散射辐射不仅包含了入射光的一 次散射，还包含了其他粒子所造成的二次散射、三次散射等，这种现象被称为光的多次散 射[77]。直接对多次散射进行分析时需要建立在大量的假设基础上，不能严密地表现真实 情况。目前主流的方法是利用光的波粒二象性，将光波的多次散射问题转化为光子的碰撞 传输问题，通过 Monte-Carlo 方法进行解决。

Monte-Carlo方法[78]是一种以概率模型为基础，通过计算模拟环境下的实验结果作为 所求问题近似解的方法。Monte-Carlo 多次散射模拟是把散射问题看成是光子与介质中不 

同粒子不断发生的碰撞问题，并将每一次碰撞视为一次弹性散射过程，且每次散射都只与 前一次散射过程有关[73]。这种方法可以有效避免求解多次散射过程中的复杂传输方程。 图3-4所示为Monte-Carlo 方法求解入射光在介质中传输时的多次散射过程[78]。其中，激 光沿 $z$ 轴射入厚度为 $h$ 的介质中，S1、S2、S3等为光子在介质中发生多次散射的各散射点。 激光沿 $z$ 轴射入厚度为 $h$ 的介质中，根据粒子在介质中的空间位置 $O \ b { = } ( x , y , z )$ 、运动方向 $J { = } ( \mu x , \mu y , \mu z )$ 和权重值 $\omega$ 描述光子在介质空间中的运动状态。其中， $\mu _ { x } , \mu _ { y }$ 和 $\mu _ { z }$ 分别为光 子前进方向 $J$ 与 $x , y , z$ 轴夹角的余弦值。假设光子从坐标轴原点出发，沿图中所示的 $\mathbf { Z }$ 轴 正方向射入。得到粒子的初始位置 $O _ { 0 } { = } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) { = } ( 0 , 0 , 0 )$ ，初始运动方向 $J _ { 0 } { = } ( \mu x _ { 0 }$ , $\mu y _ { 0 } , \mu z _ { 0 } )$ ，初始权重 $\mathrm { \omega } _ { \mathrm { \omega } } = 1$ 。 

描述光子在介质内部的运动过程，可以通过对其散射相函数进行采样来获取每次散射 时的散射角。对于球型粒子而言，由 Mie散射理论求出的散射相函数以级数的形式表示， 不利于从中抽样得出粒子散射方向。而 Henyey-Greenstein 相函数表达形式简单，也能较 为完备的表述实际散射情况[79]。本文采用 Henyey-Greenstein 函数近似地描述光子的散射 相位函数，表达式如式(3-18)所示。 

$$
F (\theta) = \frac {1 - g ^ {2}}{3} \tag {3-18}
$$

$$
(1 + g ^ {2} - 2 g \cos \theta) ^ {\overline {{2}}}
$$

其中，g 为散射不对称因子， $\xi$ 为(0,1)上均匀分布的随机数。对(3-18)采样即可求得每次 碰撞后的散射角 $\theta ^ { [ 8 0 ] }$ ： 

$$
\theta = \left\{ \begin{array}{c c} \cos^ {- 1} \left\{\frac {1}{2 g} \left\lfloor (1 + g ^ {2}) - \frac {\left(1 - g ^ {2}\right) ^ {2}}{\left(1 - g + 2 g \xi\right) ^ {2}} \right\rfloor \right\} & g \neq 0 \\ \cos^ {- 1} (2 \xi - 1) & g = 0 \end{array} \right. \tag {3-19}
$$

由于粒子是各向同性散射，其方位角 $\varphi$ 在 $( 0 , 2 \pi )$ 上均匀分布，因此方位角的抽样值可 由式(3-20)给出。 

$$
\varphi = 2 \pi \xi \tag {3-20}
$$

设光子在进行 $n$ 次碰撞后的空间位置 $O _ { n } { = } ( x _ { n } , y _ { n } , z _ { n } )$ ，则其 $n { + 1 }$ 次碰撞后的空间位置 $O _ { n + 1 } { = } ( x _ { n + 1 } , y _ { n + 1 } , z _ { n + 1 } )$ 与 $O _ { n } { = } ( x _ { n } , y _ { n } , z _ { n } )$ 满足式(3-21)至式(3-23)关系： 

$$
x _ {n + 1} = x _ {n} + L \cdot \mu x _ {n} \tag {3-21}
$$

$$
y _ {n + 1} = y _ {n} + L \cdot \mu y _ {n} \tag {3-22}
$$

$$
z _ {n + 1} = z _ {n} + L \cdot \mu z _ {n} \tag {3-23}
$$

其中，μxn、μyn、 $\mu z _ { n }$ 表示第 $n$ 次碰撞时光子前进方向与 $x , \ y , \ z$ 轴的夹角余弦值， $L$ 为光 子在介质中自由传输时的路程长度。由式(3-21)至式(3-23)可知，光子第 $n { + 1 }$ 碰撞都与第 $n$ 次碰撞时的光子运动状态有关。光子的自由运动路径长度如式(3-24)所示： 

$$
L = - \frac {\ln (\xi)}{Q _ {e x t}} \tag {3-24}
$$

其中， $\xi$ 取(0,1)上均匀分布的随机数， $Q _ { e x t }$ 为粒子的消光系数，如式(3-25)所示： 

$$
Q _ {e x t} = Q _ {s c a} + Q _ {a b s} \tag {3-25}
$$

其中， $Q _ { s c a }$ 为粒子的散射系数， $\mathcal { Q } _ { a b s }$ 为粒子的吸收系数。 

光子第 $n { + 1 }$ 次碰撞的运动方向 $J _ { n + 1 } { = } ( \mu x _ { n + 1 } , \mu y _ { n + 1 } , \mu z _ { n + 1 } )$ 与第 $n$ 次碰撞的运动方向 $J _ { n } { = } ( \mu x _ { n } , \mu y _ { n } , \mu z _ { n } )$ 有关，二者的关系由式(3-26)至式(3-28)给出。 

$$
\mu x _ {n + 1} = \frac {\sin \theta_ {n} \left(\mu x _ {n} \mu z _ {n} \cos \phi_ {n} + \mu x _ {n} \sin \phi_ {n}\right)}{\sqrt {1 - \mu z _ {n} ^ {2}}} + \mu x _ {n} \cos \theta_ {n} \tag {3-26}
$$

$$
\mu y _ {n + 1} = \frac {\sin \theta_ {n} \left(\mu y _ {n} \mu z _ {n} \cos \phi_ {n} + \mu y _ {n} \sin \phi_ {n}\right)}{\sqrt {1 - \mu z _ {n} ^ {2}}} + \mu y _ {n} \cos \theta_ {n} \tag {3-27}
$$

$$
\mu z _ {n + 1} = - \sin \theta_ {n} \cos \phi_ {n} \sqrt {1 - \mu z _ {n} ^ {2}} + \mu z _ {n} \cos \theta_ {n} \tag {3-28}
$$

当光子的运动方向接近 $n$ 轴，即 $| \mu z _ { n } | \approx 1$ ，此时光子的第 $n { + 1 }$ 次的碰撞方向为： 

$$
\mu x _ {n + 1} = \sin \theta_ {n} \cos \phi_ {n}
$$

$$
\mu y _ {n + 1} = \sin \theta_ {n} \sin \phi_ {n} \tag {3-29}
$$

$$
\mu z _ {n + 1} = \cos \theta_ {n} \frac {\mu z _ {n}}{\left| \mu z _ {n} \right|}
$$

并且光子在每次碰撞后的能量会逐渐衰减，即每次碰撞后的权重会逐渐减小。每次碰 撞发生后光子的权重变化由式(3-30)给出： 

$$
\omega_ {n + 1} = \omega_ {n} \cdot \frac {Q _ {s c a}}{Q _ {e x t}} \tag {3-30}
$$

设光子的初始权重为 $\omega _ { 0 } { = } 1$ ，由式(3-31)给出第 $n$ 个光子透射出介质的概率为： 

$$
P _ {n} = \sum_ {i = 0} ^ {M - 1} P _ {n} ^ {i} \tag {3-31}
$$

其中， $M$ 为第 $n$ 个光子在传输过程中与介质内部粒子碰撞的总次数。对 $N$ 个光子进行追 踪得到光子透过率的平均值即为整体介质的激光透过率 T，如式(3-32)所示： 

$$
T = \frac {1}{N} \sum_ {n - 1} ^ {N} P _ {n} = \frac {1}{N} \sum_ {n = 1} ^ {N} \sum_ {i = 0} ^ {M - 1} P _ {n} ^ {i} \tag {3-32}
$$

由此得到激光光束的多次散射衰减系数： 

$$
\mu_ {e} = - 1 0 \lg T \quad (\mathrm {d B / k m}) \tag {3-33}
$$

$$

# 3.2 大气激光吸收理论

大气对激光的吸收作用主要包含气体分子吸收和气溶胶粒子吸收两部分，如式(3-35) 

所示[81]。 

$$
\alpha (\lambda) = \alpha_ {A} (\lambda) + \alpha_ {a} (\lambda) \tag {3-35}
$$

其中， $a _ { A } ( \lambda )$ 和 $a _ { a } ( \lambda )$ 分别为由气体分子和气溶胶粒子吸收作用引起的激光衰减系数。 

大气气体分子的吸收作用是将入射激光的能量转化为大气气体分子运动所需的能量。 气体分子吸收衰减主要与入射激光的波长密切相关。有研究表明，在可见光和 $1 . 0 6 \mu \mathrm { m }$ 波 段内，气体分子的吸收几乎可以忽略不计。而对于长波波段，例如 $1 0 . 6 \mu \mathrm { m }$ ，大气气体分 子的吸收衰减较为严重[82]。除此之外，吸收衰减还与激光传输的海拔高度有关，因为在 近地面附近，激光的大气吸收效应与水汽和气体分子的浓度成正比。

# 4.1.1 雾滴粒子的单次散射特性分析

针对粒子的单次散射问题，主要从以下几个参量展开讨论：消光效率因子 $Q _ { e x t }$ 、散射 效率因子 $Q _ { s c a }$ 、吸收效率因子 $\mathcal { Q } _ { a b s }$ 、不对称因子 $g$ 、单次反照率 $w$ 、 散射相函数 $p ( \theta )$ 。结 合 Mie 散射理论给出各参量的表达式如式(4-3)至式(4-8)[13,30]： 

$$
Q _ {e x t} = \frac {2}{X ^ {2}} \sum_ {n = 1} ^ {\infty} (2 n + 1) \operatorname {R e} \left(a _ {n} + b _ {n}\right) \tag {4-3}
$$

$$
Q _ {s c a} = \frac {2}{X ^ {2}} \sum_ {n = 1} ^ {\infty} (2 n + 1) \left(\left| a _ {n} \right| ^ {2} + \left| b _ {n} \right| ^ {2}\right) \tag {4-4}
$$

$$
Q _ {a b s} = Q _ {e x t} - Q _ {s c a} \tag {4-5}
$$

$$
g = \frac {4}{X ^ {2} Q _ {s c a}} \left\{\sum_ {n = 1} ^ {\infty} \frac {n (n + 2)}{n + 1} \operatorname {R e} \left(a _ {n} a _ {n} ^ {*} + b _ {n} b _ {n} ^ {*}\right) + \sum_ {n = 1} ^ {\infty} \frac {2 n + 1}{n (n + 1)} \operatorname {R e} \left(a _ {n} b _ {n} ^ {*}\right) \right\} \tag {4-6}
$$

$$
w = \frac {Q _ {s c a}}{Q _ {e x t}} \tag {4-7}
$$

$$
p (\theta) = \frac {4 \pi \sigma}{\sigma_ {s}} = \frac {4 \pi \sigma}{\pi r ^ {2} Q _ {s}} = \frac {\left| S _ {1} \right| ^ {2} + \left| S _ {2} \right| ^ {2}}{\sum_ {n = 1} ^ {\infty} \left[ (2 n + 1) \left(\left| a _ {n} \right| ^ {2} + \left| b _ {n} \right| ^ {2}\right) \right]} \tag {4-8}
$$

其中， $X$ 为粒子的尺度参数， $a _ { n }$ 和 $b _ { n }$ 分别为 Mie 散射系数， $S _ { 1 }$ 和 $S _ { 2 }$ 分别为散射光振幅函 数， $\sigma$ 为粒子谱分布参数。本节将选取可见光到远红外波段中较为典型的 2 个波长： $1 . 0 6 \mu \mathrm { m }$ 和 $1 0 . 6 \mu \mathrm { m }$ 展开雾滴粒子的单次散射特性分析。

# 4.1.3 雾环境中的传输衰减经验模型分析

# (1) Kim 模型

Kim 模型的雾衰减系数为[33,85]： 

$$

# 5.4 本章小结

激光在雾霾环境中传输时，会受到大气中悬浮的水滴粒子、PM2.5 和PM10 等大气污 染粒子的消光作用，造成较为严重的激光功率衰减。本章根据实验要求及预期目标，设计 并测量了激光在雾霾环境中传输时的功率衰减。通过大量实验数据，得出激光在西安地区 雾霾环境中传输的基本规律。通过将雾环境实测数据与雾衰减经验模型进行对比可知，当 激光在雾环境中进行传输时，各激光链路均有与之相对应的经验预测模型。表明雾衰减经 验模型在不同能见度和传输距离时具有一定的参考价值。 

# 6 总结和展望

# 6.1 工作总结

本文主要研究了无线光通信系统中激光在雾霾环境下的大气透过率。论文对雾霾环境 中悬浮的水滴粒子、沙尘粒子、烟煤粒子、PM2.5 粒子和PM10粒子的散射特性进行了分 析；设计并测量了激光在雾霾环境中传输时到达接收端的功率变化；通过对实验数据整合 处理并与经验模型进行对比，分析了激光在实际雾霾环境中的传输衰减特性，得出激光在 西安地区雾霾环境中传输的基本规律。对本论文中的主要研究工作及成果总结如下： 

(1) 本文对雾霾环境及其粒子的基本物理特性进行了概述，介绍了雾霾环境中的激光 衰减理论，着重阐述了本文仿真分析时用到的 Mie散射理论和Monte-Carlo 方法。在此基 础上，对雾霾环境中的 5 种典型粒子：雾滴粒子、沙尘粒子、烟煤粒子、PM2.5 粒子和 PM10粒子进行了粒子散射特性分析。结果表明：粒子的消光系数随着尺度参数的不断增 大，最终均趋近于2，说明粒子对激光造成的总能量衰减大约为几何截面能量衰减的 2倍； 在整体散射作用中，前向散射相较于后向散射占主体地位。 

(2) 结合粒子的粒径分布函数，利用 Monte-Carlo 方法仿真计算了不同入射波长时， 粒子浓度与激光最大传输距离、衰减系数和透过率之间的关系。结果表明，随着粒子浓度 的增加，粒子的消光系数不断提高，对应的激光最大传输距离和大气透过率不断降低。在 大气透过率的数值方面，当雾滴粒子浓度为 $5 0 0 \mu \mathrm { g } / \mathrm { m } ^ { 3 }$ 时， $1 . 0 6 \mu \mathrm { m }$ 激光的透过率为 $7 5 . 7 3 \%$ ， $1 0 . 6 \mu \mathrm { m }$ 激光的透过率为 $8 3 . 2 2 \%$ ；当沙尘粒子浓度达到 $3 0 0 \mu \mathrm { g } / \mathrm { m } ^ { 3 }$ 时， $0 . 6 3 3 \mu \mathrm { m }$ 激光的透过 率为 $7 8 . 0 1 \%$ ， $1 . 3 \mu \mathrm { m }$ 激光的透过率 $8 3 . 3 \%$ ；当烟煤粒子浓度达到 $5 0 \mu \mathrm { g } / \mathrm { m } ^ { 3 }$ 时， $0 . 6 3 3 \mu \mathrm { m }$ 激 光的透过率为 $0 . 0 0 1 \%$ ， $1 . 3 \mu \mathrm { m }$ 激光的透过率 $0 . 5 1 \%$ ，且透过率随粒子浓度继续增大趋近于 0；当 PM2.5 粒子和 PM10 粒子的浓度达到 $1 7 0 \mu \mathrm { g } / \mathrm { m } ^ { 3 }$ 时， $0 . 5 5 \mu \mathrm { m }$ 波长对应的透过率分别 为 $8 5 \%$ 和 $9 1 . 8 \%$ 。 

(3) 本文选取 $8 0 \mathrm { m }$ 、 $3 4 0 \mathrm { m }$ 和 $2 5 0 0 \mathrm { m }$ 链路对激光在西安地区实际雾霾环境中传输时的 功率变化进行了测量，得到系统接收功率与激光大气透过率随实际能见度变化的曲线图。 将雾环境中的实测曲线与常用雾衰减经验模型进行对比分析可知：随着能见度的增加， $8 0 \mathrm { m }$ 实测曲线逐渐趋近于 Kreid 模型，并在 $V { > } 8 . 3 \mathrm { k m }$ 时与 Kreid 模型基本重合； $3 4 0 \mathrm { m }$ 实 测曲线整体与Vasseur 辐射雾模型保持较高的一致性，并在 $V { \mathrm { < } } 9 \mathrm { k m }$ 时与 Vasseur 辐射雾模 型基本重合；在 $2 5 0 0 \mathrm { m }$ 激光链路中，当 $V { < } 5 . 7 \mathrm { k m }$ 时，实测曲线大于 Kim 模型；当 $5 . 7 \mathrm { k m } < V < 7 \mathrm { k m }$ 时，实测曲线与 Kim 模型基本重合；当 $7 \mathrm { k m } < V < 9 . 2 \mathrm { k m }$ 时，实测曲线处于 Kim 模型与 Naboulsi 模型之间。为未来无线光通信系统在雾霾环境中的应用提供了一定 的参考价值。 

# 6.2 展望

研究雾霾环境对激光造成的传输衰减问题，对于未来无线光通信系统在雾霾环境中的 

应用及组网具有重要的参考价值。论文只是在该问题上取得了一些初步成果，还有很多问 题有待进一步解决。未来工作主要应该围绕以下几方面进行： 

(1) 对于粒子的散射特性，本文仅从球型粒子出发，研究了单次散射和多次散射问题。 未来应该针对其他形状复杂、成分各异的粒子进行仿真研究。 

(2) 本文仅采用单一波长对激光传输的物理特性进行了测量，未来应该设计多波长测 量系统，结合不同的调制解调技术，测量无线光通信系统在雾霾环境中的信噪比、误码率 等系统性能。
