# 无线光通信关键技术研究

# 摘要

无线光通信是无线通信与光通信相交叉融合的一种通信技术，为日益紧张的有限射频资源提供了无限的可替代的频谱。光通信具有高带宽、低辐射、抗干扰等特点，因此越来越受到国内外的重视，成为未来无线通信领域研究的重要方向之一。 

本论文从系统研发角度出发，阐述了无线光通信的原理和关键技术，概括分析了无线光通信的特点和优势，归纳总结了无线光通信系统设计中的几个关键问题，设计实现了两套无线光通信实验系统：紫外LED通信实验系统和可见光视频演示系统，并进行了实测分析。 

论文参考国内外已有的无线光通信案例，并借鉴传统无线射频技术，针对上述系统设计中的技术难点做了研究，其中包括：对于无线光通信发射光源的研究和改进；对接收端探测性能的改进；信号噪声的数字滤波处理以及对无线光通信中信道差错编码技术的研究。在实验系统的基础上，将理论分析与实验测试相结合，将控制光源进行占空比发光的方案在紫外LED通信试验系统和可见光系统上进行了论证和实际测试；对影响数字滤波性能的各项参数进行了仿真和实际测试；利用紫外通信实验实际测试了双接收对于系统的增益，进而验证多接收的性能；将信道编码技术（RS纠错编码）应用于无线光通信，并进行了误比特率的测试，进一步提升了系统的通信性能。 

最后，通过对以上几个关键技术的总结和具体实验结果的分析与比较，论文提出了进一步改善性能的方案。 

关键词：无线光通信 占空比发光 数字滤波 多接收 RS编码

# 第一章 绪论

无线光通信作为一种新型的基于光信号传播的通信系统，为日益紧张的有限射频资源提供了无限的可替代的光谱资源。该技术利用光载波进行通信，具有带宽高，辐射低，不干扰电磁设备工作的特点，被誉为是一种未来绿色的通信手段，因此广泛地受到国内外通信研究的重视，成为未来无线通信领域研究的重要方向。本论文主要从技术角度出发，针对基于LED的无线光通信系统中的关键技术进行了研究。 

# 1.1 研究背景与研究现状

近年来，空间光通信技术迅速发展，基于红外波段的自由空间光通信（FSO）技术已经从理论研究过渡到实际应用，而波长较短的可见光频段和紫外频段也越来越受到人们的关注，正逐步地从工业生产、安全防伪、医疗保健、环境保护等领域延伸到无线通信领域。 

无线光通信技术区别于传统的射频通信技术，有其自身的特点：从红外波段到紫外波段，无线光通信有着丰富的免费频谱资源，为信息传输提供了超大容量的频带；其信息传输保密性好，并且无电磁污染，不会干扰到其他电磁敏感设备，本身也不会受到电磁辐射的影响；设备体积小，系统轻巧便携，不受地理因素的影响，架设灵活；功耗低，只需几伏，本地供电方式多种多样；此外处在不同波段的红外、可见光、紫外还分别有其自身的特点。因此，无线光通信是未来短距离无线通信强有力的竞争者，对其的研究有很大的实用价值。 

目前红外光通信（Free Space Optical，FSO）技术已经趋于成熟，各种应用产品已经投入市场。本论文依托国家自然科学基金项目，主要研究可见光波段和紫外光波段的新的基于LED的无线光通信系统中的关键技术问题。 

# 1.1.1 国外现状

# 1.1.1.1 可见光通信

在可见光通信方面，基于照明LED基础上的可见光通信技术有其特有的优势，近几年来该技术受到国内外各个领域的高度重视，随着相关领域科技水平的进步和各国科研力量的投入增大，可见光通信正步入一个高速发展的时期。 

国际上，发达国家对可见光通信的研究比较早，其中日本方面的进展较为瞩目。可见光通信最早由日本首先提出，在日本政府的支持下获得了较快的发展，如今可见光通信已然成为未来“绿色通信”的发展目标之一。2000年，日本KEIO 

大学和索尼计算机科学研究所共同提出了基于LED的无线信息传输思想，可见光通信由此受到关注[1]。2003年可见光通信协会（VLCC）成立，2008年可见光通信小组IEEE802.15.7成立。目前，利用照明LED系统进行可见光通信的实验已经得到证实：在使用波分复用（WDM）技术的情况下，能够实现速率为803Mbps的可见光通信，LED通信已经成为未来发展绿色通信的一个重要方向，在室内接入网的方案设计中，可见光通信有突出的竞争力。 

日本方面，2000年，日本中川实验室的Tanaka等人首先对可见光LED的无线通信系统进行了数学分析和理论建模，随着可见光通信速率的提升，他们认为多径效应和码间干扰（ISI）是影响系统通信性能的最主要的因素[2]，之后他们还采用不同的调制技术对系统性能进行了仿真[3]。2002年，Komine等人从不同方面对可见光通信系统进行了分析，包括：系统光源属性、噪声的分布模型以及信道的模型等，对比了不同调制方式下的误码率，并对信号传播的多径效应做了深入的研究[4]。2003年，Tanaka等人提出了可见光LED通信系统所需的LED的基本要求，同年Komine提出利用现有的电力传输网，将电力线通信与可见光通信相结合的思想[5,6]。2004年，Komine等人深入研究了可见光的信道传输特性以及LED光源布局对于通信的影响，分析了可见光链路中的遮挡效应和墙壁、物体的反射效应的影响，码间干扰的影响，在不同的数据传输速率和不同的探测器接收视场角的情况下，给出了各个因素对于系统性能的影响曲线[7]。2006年，Sugiyama等人提出了应用于可见光通信的改进了的PPM调制——SCI-PPM调制[8]。2007年，Hidemitsu和Sugiyama等人综合考虑照明和通信两个因素，提出了可以控制照明亮度的调制方法：脉冲宽度调制(PWM)控制和调制深度(modulation Depth)控制，同时控制照明和通信[9]。在实验实现上，2008年，日本太阳诱电株式会社在“日本高科技博览会”上演示了最大数据传输速率达到100Mbps的高速VLC系统。最近两年，中川实验室对可见光通信的研究主要集中在OFDM调制技术以及自适应技术上，进一步提升可见光通信的速率。 

欧洲对于可见光通信的研究稍晚于日本，但最近几年已经推出了可喜的成果。2006年，德国不来梅国际大学的Afgani实验测试了OFDM技术在可见光LED通信系统中的应用，实验证明，在使用高亮度的LED光源，并且在短距离的通信范围内（一米的范围内），利用OFDM技术能有效削减峰均功率比(Peak-to-average Power Ratio)，实现高速通信[10]。2007年，Haas提出在白光通信系统中应用基于正交相移键控(QPSK)调制COFDM(coded OFDM)技术，有效的降低了系统的误比特率[11]。同年，牛津大学的O'Brien所在的课题组用多谐振均衡技术解决了LED光源窄带宽的问题，从而提升了LED光源的可调制带宽，基带调制速率最高达 $25\mathrm{MHz}^{[12]}$ 。2008年，德国海因里希·赫兹通信工程研究所的Langer等人就宽 

带可见光通信的物理层基础问题进行了初步研究[13]，同年，Gruber就实验实现了101Mbps的白光LED通信。近两年，Hass所在课题组主要在研究对称削峰OFDM(ACO-OFDM)和直流偏置OFDM(DCO-OFDM)在可见光系统上的应用，并已取得了一定的成果[14]。 

2008年，美国投资了一项1.82亿美元为期10年的“智慧照明(Smart Lighting)”计划[15]，用于探索无线通信与可见光通信的融合，该计划有美国仁斯利尔理工大学，波士顿大学等超过30所大学的参与，同时也有超过20家的国际化大公司对此计划表示支持。 

总体来说，可见光通信在国外的发展比较先进，目前已经在实验室实现了速率率达百兆比特量级的数据通信系统。而低速的应用也已出现在市场上。可见光通信小组（VLCC）以及IEEE802.15.7工作小组的成立，以及越来越多的政府部门和商业公司的加入，可见光通信技术正在步入一个快速发展的阶段。 

# 1.1.1.2 紫外光通信

相对于可见光通信的飞速发展，紫外光通信技术虽然也已得到了国内外科研领域的高度重视和政府资金的投入，但目前紫外光器件的发展水平有限，相比较于其他光通信技术，如FSO和可见光通信等，紫外光通信的发展仍远远落后。这其中不仅包括通信理论的研究和理论模型的完善，还包括紫外光源、接收探测器等关键器件的发展，此外，针对紫外光通信的编码调制方式等核心的通信技术也不够健全，因此，紫外光通信技术的研究仍处在一个较初级的阶段。 

国际上，美国在紫外光通信的研究方面起步较早。早在1932年，美国就开始研究紫外光在大气中的传输特性，并开始了相关紫外产品的制造。1968年，第一套紫外光系统在麻省理工学院的林肯实验室搭建，该系统主要用于紫外信道的基本特性的测试，随后，该实验室预言了紫外光通信在10英里范围内的可能性[16]。在理论模型的研究方面，1974年，佛罗里达大学最早开始了紫外信道模型的研究，用多重散射理论计算紫外C波段的光在地球表面的传播特性[17]。此后，基于蒙特卡罗方法的单散射模型和多散射模型相继被提出，但由于紫外光传输的特殊性，此套理论模型还处在不断的完善中[18]。在实验测试方面，虽然美国较早地研制出了通信性能处于国际领先地位的紫外通信模型，但是紫外通信器件的发展较为缓慢，其实际应用还不够成熟，因此紫外通信一直处于实验室研究阶段。近几年来，随着工艺水平的进步，紫外器件已经出现了较多的产品，从 $350\mathrm{nm}$ 的紫外A波段LED到 $265\mathrm{nm}$ 的紫外B波段LED，甚至 $150\mathrm{nm}$ 深紫外的C波段LED都已经推向市场，并且LED的量子效率和发光功率还在不断的提升。探测器方面的发展，也已出现了灵敏度高，响应速度快，视场角宽广的紫外光探测器。在此基础之上，2006年，麻省理工大学林肯实验室研制了一套分布式的紫外LED无人 

传感通信系统，此系统的LED光源发射光功率为 $40\mathrm{mW}$ ，通信速率2.4Kbps，通信距离 $10\mathrm{m}$ ，实现了用紫外光的语音通信[19]。此外，加州大学河滨分校的WIT实验室也一直在从事紫外光通信的理论研究和实验原型系统的研发，从2007，该实验室先后进行了基于蒙特卡洛法的紫外光信道的单散射模型分析和多散射模型分析，并实际搭建了多套实验测试系统，包括紫外LED通信测试系统和紫外激光通信系统等，该系统可实现发送端 $40^{\circ}$ ，接收端 $20^{\circ}$ ，距离 $35\mathrm{m}$ ，误码率小于 $10^{-3}$ 的非视距紫外光通信。利用该系统完成了紫外实际信道损耗的测试、信道脉冲的时延特性测试以及不同几何仰角下的紫外通信误比特率测试[20]。 

日本在紫外光器件的研究方面一直处于国际领先的水平。早在2002年，日本日亚化学工业就研制出了中心波长为 $380\mathrm{nm}$ ，发射光功率为 $100\mathrm{mW}$ 的紫外LED。与此同时，中心波长为 $400\mathrm{nm}$ ，外部量子效率达 $31\%$ 的紫外A波段LED也被日本的新能源产业技术综合开发机构（NEDO）主持开发。日本滨松公司在探测器方面的制造一直处于国际领先地位，目前已推出了多种产品，包括紫外波段的光电倍增管（PMT）、雪崩光电二极管、微通道板光电倍增管（MCP）等不同灵敏度的光探测器，并且还专门研制了配合探测器的放大管座，PMT和MCP可以将经过光电转换后的电流信号放大至原来的 $10^{7}$ 倍。紫外光源和紫外探测器件的进步有力地推动了紫外光应用于通信领域的步伐。 

总体来说，在紫外光通信的研究方面，国际上发达国家的技术先进，研究一直处于领先地位。美国主要在短距离紫外通信方面进行了大量的研究，在进行原理探索的同时也完成了从理论到实用系统的研发。日本方面针对紫外光器件的研究比较多，并且正在进一步的提升各种器件的性能。从以上紫外光通信的发展来看，紫外通信的研究目前都在向着小型化，低功耗的方向发展，但受限于目前的紫外光器件水平，距离真正推向市场，实现产品化还有一段距离。 

# 1.1.2 国内现状

# 1.1.2.1 可见光通信

从上一节资料的分析可以看出，日本和欧洲等国先后投入了大量的人力物力在可见光通信的研究，已经有了相当多的理论和技术积累，并完成了从理论到实际应用设计的过渡，已经在某些领域实施了试点系统。但是，以上这些研究成果和具体的技术细节还都处于保密状态，我国在此方面的研究还处于初步阶段，技术积累和实验部分还不够，因此，发展可见光通信还需要靠自身的努力和不断的尝试，目前国内已有几所大学的部分实验室有对可见光通信开始了的相关的研究。 

国内对于可见光通信的研究起步较晚，但也已取得了一定的成果。目前研究无线光通信的机构主要有：西安理工大学，浙江大学，暨南大学，长春理工大学 

等多所院校。早在1999年，香港大学的GPang等人研制了一台可见光LED通信实验原型机，通过实验，该原型机可以实现距离 $20\mathrm{m}$ ，通信速率为10kbps的语音通信[21]。2006年，暨南大学的陈长缨教授等人设计实现了可见光高速通信实验系统，该系统采用照明LED作为通信光源，实现了点对点的通信方式，通信速率可达10Mbps,实验结果表明，基于照明LED的可见光通信有较好的通信性能，未来在接入网领域有较强竞争力[22]。此后，暨南大学继续展开对白光LED通信系统中各项关键技术的研究，并在2008年，该课题组克服了通信中可见光LED自身功率小，无法满足照明要求等难题，设计出一套4Mbps，通信距离超过2.5米的视频传输系统原型，继续丰富了国内对于可见光LED通信应用于室内接入链路的实验研究[23]。同年，浙江大学的王永波等人深入研究了可见光通信中的照度分布情况，对白光LED光源的布局以及其产生的照明分布进行了分析，通过对通信系统中光源的分布进行设计，使得接收面上的各点的所达到的光探测强度最佳[24]。 

虽然国内的可见光通信研究起步晚，但经过多家研究机构的努力，已经取得了喜人的成果，为可见光的进一步研究打下了基础。 

# 1.1.2.2 紫外光通信

紫外光通信方面，相对于国外研究，国内紫外光通信的起步较晚，主要的研究单位包括北京邮电大学、重庆大学、国防科技大学以及北京理工大学等院校和中科院的一些研究机构，由于国内的紫外器件生产水平较低，还不具备生产某些关键器件，因此国内的研究大部分都还是集中在紫外信道理论模型的构建和调制解调方法的研究上面，也有部分实验室采用紫外汞灯实现了实际的通信实验测试。 

在紫外器件的研发方面，中科院半导体所、上海光机所等多家单位已经投入了大量的科研力量在紫外半导体关键器件的研发上，并已经取得了可喜的成果。其他学院研究机构也都从各个方面开展了紫外光通信的研究。北京理工大学在紫外光通信的应用方面的研究较早，自1999年开始，北京理工大学一直在持续进行有关紫外光应用方面的实验，基于紫外低压汞灯研制了紫外光点对点通信实验系统[25]。国防科技大学主要致力于紫外光通信理论的研究，包括用蒙特卡罗方法构建紫外光非视距传输模型以及基于离散坐标法的紫外光信道传输模型，紫外光调制解调技术的研究等，自此基础上，实际搭建了测试平台，因此，国防科技大学在理论和实验方面都积累了丰富的经验。重庆大学实现了基于紫外LED的通信实验系统，并实现了短距离、低速率的语音通信和数据通信，并在此基础之上，进一步对紫外光的调制和编码技术进行了研究[26]。中科院空间科学与应用研究中心主要针对紫外光通信中的多媒体业务传输技术进行研究。电子科技大学在紫外光通信系统的传输模型、调制编码技术上都有相应的研究成果[27]。西安理工大学是国内较早探索紫外光组网技术的单位之一，主要研究紫外光通信与其他通信方 

式之间的网络融合问题[28]。北京邮电大学在研究基于紫外LED的通信技术上有着丰富的经验，其中在紫外光通信系统的设计、关键技术的研发、实验测试验证以及信道理论的建模等方面都取得了一定的成绩，先后构建了基于蒙特卡罗法的紫外光大气传输单散射信道模型和多散射模型，并在此基础上对系统设计、信号处理算法、通信协议等方面都做了广泛的调研和深入的研究[29]。 

总之，国内研究机构在紫外光通信方面的研究还以理论居多，随着国内器件技术水平的提高，各个研究机构开始向实际实验平台的搭建和实验测试转变。 

# 1.2 论文研究目的及意义

首先，在信息化技术的布局中，无线频谱资源日益紧张，无线接入网络正是受限于电磁频谱这一有限资源的影响，无法满足消费者对于更高宽带的需求，使得无线技术的发展捉襟见肘，而光谱频谱资源成功的用于通信无疑是给无线通信领域的雪中送炭。其次，随着信息化程度的不断提高，越来越多的无线电子设备接入到网络，频谱复用的空间越来越狭小，频谱的拥挤愈加显现。而频谱拥挤就会导致非常严重的后果：不同的电子设备之间相互影响，甚至导致有的设备陷于瘫痪。再次，常规电磁波段的频率信道很容易受到捕获和窃听。目前百米以内的高速无线接入网主要以无线射频技术WiFi为主，而WiFi所用的信号属于射频频段，其对建筑物的穿透性较强，因此，只要在通信范围附近就可以截获到射频信号，并从中窃听到通信的内容。由此可见，射频通信在保密性方面有着巨大的缺陷。而无线光通信的提出，克服了以上传统无线通信的弊端，对于接入网带宽的扩充、减少设备之间的互相干扰和实现保密通信等方面提供了有力的解决方案。 

无线光通信通常可以按其工作的不同波段进行划分。红外波段通信，红外波段已被广泛应用到了光纤传输网络中，目前的无线红外波段通信又叫自由空间光通信（FSO），其发展较早，且已有了较为成熟的器件，包括激光器和探测器等设备，并不断的发展和投入市场，红外通信为无线光通信的发展开辟了先河。虽然FSO通信较常用于较远距离的高速通信场合，但是需要复杂的跟踪和对准机制，其器件的功耗较大，复杂环境下的通信可靠性受到影响，且不利于小型化和可移动便携性。 

有灯光的地方即可有可见光的通信。随着固体照明的兴起，白光LED技术获得了较快的发展，与传统的照明灯具相比，白光LED具有更高的电光转换效率、更长的使用寿命、更低的功耗等优点，白光LED照明有取代传统照明之势。另一方面，白光LED具有可调制信号的特点，其目前的基带调制速率可达40MHz，因此，将白光LED与通信相结合，实现可见光通信将有无限宽广的市场前景。 

可见光通信（Visible Light Communications，VLC）技术是以白光LED作 

为光源，以大气为信道，将基带信号调制在LED发出的可见光上进行通信的无线光通信技术。它既可以作为白光LED灯照明之外的第二功能存在，也可以专门用于通信。可见光通信是一种灵活、便捷、高隐蔽的无线局域网通信技术。首先，由于白光对人眼的安全性，室内白光LED灯的功率之和可以高达数十瓦，甚至上百瓦，这就使可见光通信具备了非常高的信噪比，为其高速通信打下了良好的基础，非其他技术可比。目前利用高阶调制技术，实验的通信速率已达530Mbits/s，理论速率可达1.7Gbits/s以上，远高于射频WiFi等技术的理论和实际速率。其次，由于可见光对于物体的不可穿透性，因此可见光通信具有高度的保密性。第三，由于可见光信号和射频信号相互不干扰，所以它可以应用在电磁敏感环境中，如机舱、医院等。第四，由于频谱无需授权即可使用，所以可见光通信应用灵活，可以单独使用，也可以作为射频无线设备的有效备份。 

由此可见，可见光通信具备带宽大、隐蔽性好、抗电磁干扰等特点，在有高宽带需求的区域通信中有较大的应用潜力，并且在有照明的空间中有着无可比拟的优势，研究可见光通信技术不仅可以替代传统射频通信，对于能源节约，绿色通信等方面有着巨大的价值。 

相对于其他波段的光通信，紫外光波段应用于通信也受到了国内外的追捧。主要是由于紫外光的自然背景噪声较小，可以得到较高的信噪比，可以适应全天候的工作环境，另一方面是因为大气分子对于紫外光的传输有散射作用，不需要严格的对准即可实现通信。但是，由于紫外信号受到大气媒介的强吸收作用，本身又具有散射传播的特性，加上目前的紫外LED光源速率受限（1Mbps），通信距离限制在百米量级。 

紫外光的特殊性使得紫外通信成为一种特殊环境下的可替代射频通信的无线通信方式，如在电磁屏蔽环境中，在难以实现严格的视距对准的情况下，紫外光通信有着特别的优势。因此，研究紫外光通信，可以填补在短距离，低速率，低功耗，无需严格对准，适应全天候工作条件的无线光通信应用技术的空白。 

本论文充分调研了国内外无线光通信技术的研究成果，吸收借鉴了大量的理论和实际系统的设计方案，为了进一步的提升无线光通信的性能，加快其实用化的进程，针对系统设计中的关键技术进行了研究，并最终设计和研制了两套无线光通信原型系统：实现了小型化、低功耗、高灵敏度的基于紫外LED的通信实验原型系统，以及基于可见光LED的高速通信视频演示系统。这两套系统同时支持语音通信和数据通信，可以完成多方面的实际信道测试，为调制编码技术的实际验证提供了平台，为理论联系实际进一步的提升系统通信性能，开展各项技术研究打下了基础。

# 第二章 无线光通信系统设计

与传统的射频通信相比，无线光通信最大的特点是以光波为载体来传播信号，光波的频谱更高，此外，光具有波粒二象性，既可以看作是一种电磁波，又可以被看作是粒子在空间传播，因此，无线光通信系统的设计与传统射频通信有很大的区别。 

# 2.1 光谱特性的研究

鉴于紫外波段的大气传输特性，紫外光通信选择使用“日盲区”波段的紫外光（C波段）作为信道载波。首先，C波段的紫外光与臭氧分子极易融合，大气中的臭氧层隔断了阳光中的该波段的能量，形成一个天然的屏蔽层，即使在白天通信，在地表传输的C波段的信号也只有较低的自然背景噪声，干扰明显低于比其他波段的光。其次，C波段紫外光在具有一定空气粒子浓度的环境下是散射传播的，即收发双方不需要严格对准即可建立通信[30]。因此，C波段紫外光的特点使得通信接收端信号处理的设计复杂度大大降低。另外，应该注意到C波段的光还受到一定的大气吸收作用，信号能量衰减较大，通信距离受限发射端的发射光功率。 

# 2.2 无线光通信的特点

无线光通信相比于无线射频通信来说，其最大的特点是利用光载波来承载信息进行通信，因此与射频通信相比有多个不同的特点。 

# 2.2.1 无限的频谱资源

射频通信通常是将基带信号调制到射频频段，然后利用射频天线将电磁波发射出去，以交变电磁场的形式在自由空间以光速传播，接收端将收到的射频信号进行反调制，还原成基带信号。射频通信的范围一般是指频率为300MHz-300GHz的无线电波。无线光通信的基本技术是光电转换，即通过光发射机将基带信号调制到光路上，光信号在大气中进行传播，接收端光接收机将探测到的光信号进行光电变换，并恢复出基带信号。目前所研究的无线光通信范围有紫外、可见光和红外，波长从100nm到4000nm。因此，相对于日益紧张的射频资源来说，光通信有着无限的频谱资源。 

# 2.2.2 通信保密性

射频信号在空气中向四周传播时，经大气层外缘的电离层反射，可以形成较远距离的传输能力，同时在碰到不同介质时会发生电磁波的反射、折射、穿透等现象，因此射频通信在发送端到接收端之间的路径多变，窃听者可以很容易从传播路径中截取射频信号，进而破译出信息。 

无线光通信的信息载体为光波，而光波更容易受到大气中各种粒子吸收作用和散射作用的影响，能量损耗较大，因此无线光通信主要受限于光发射机的功率和大气状况。目前，无线光通信中一般使用指向性较好的光发射机（如激光器， 

一定发散角的LED等)，使光源发出能量集中的光，从而延长通信距离。光波传输时，向其他方向散射的能量很小，受吸收作用的影响几乎损耗殆尽，并且光波不容易穿透障碍物，因此无线光通信的链路主要是直视路径（其他反射路径，折射路径，散射路径很小），因此窃听者不容易截取信号，可以实现较好的保密性。 

# 2.2.3 设备简单，应急通信

传统射频通信设备复杂，建设周期长。射频天线尺寸较大，供电条件要求严格，其架设也直接影响到通信的性能。无线光通信设备简单，光收发机体积小，功耗低，易于架设，短时间内就可以建立链路，建设成本较低。因此，在不具备有线接入条件、较少的资金和人力或者恢复通信链路故障时，无线光通信可以简单，快速，有效的解决问题。 

此外，无线光通信还是解决“最后一公里”问题，有效提高用户接入端的传输容量和速度的方案。 

# 2.3 系统设计因素

目前无线光通信多采用光强度调制/直接探测（IM/DD）系统，本论文经过长时间的充分调研，阅读了大量国内外的参考文献和技术资料，为系统的设计打下了坚实的理论基础。并在此之上，对实验系统方案进行设计，其中主要考虑的因素有以下几个方面： 

# （1）信道特性：

信道特性是指光载波在大气信道中传输时，空气介质对于光的能量损耗、散射、吸收、反射等的影响。不同的气象条件对于光载波的传输衰减，大气闪烁，粒子吸收，散射效应，背景噪声等的影响剧烈程度也都不同。因此，研究无线光的大气信道传输特性可以发现不同气象条件下通信质量的规律，进而指导无线光通信系统的设计和应用场合的选择。 

研究光通信大气传输理论的方法有多种，主要的有单次散射近似方法、离散坐标法、光线追踪法和蒙特卡罗方法。其中，单次散射近似方法在大气衰减系数与传输距离之积小于0.1时较准确，因此此种方法的应用范围有限；而离散坐标法针对散射传输的理论目前还不够成熟，其完善还面临许多问题；基于随机过程的蒙特卡罗方法应用最广泛，可以模拟光在复杂环境下多次散射的结果[18]，因此本论文的系统设计参照了大量的基于蒙特卡罗法的散射模型的仿真结果，再根据实际实验的需求，对系统的具体技术细节进行改进。 

# (2) 传输可靠性:

无线光通信信道情况复杂，相比于射频通信受气象条件的影响更大，因此需 

要采取必要的发射接收技术来确保数据传输的可靠性。这与射频通信的情况类似，一方面可以采用大功率的光发射机来克服大气信道带来的衰减，提高接收端的信噪比。另一方面可以采用灵敏度较高，探测视场角较大的探测器，以感应微弱信号。 

# （3）传输速率

传输速率是衡量各类通信性能的一个重要指标。无线光通信相对于其他通信而言，最大的优势之一就是有无限的频谱带宽，并且不受其他通信方式的干扰。对于LED光源而言，目前基带的调制速率达到了几十兆bps。提升系统的传输速率，既可以采用复杂的调制技术来提升单一载波的速率，又可以采用波分复用技术提高系统的整体容量。 

# （4）组网技术

目前来说，无线光通信的组网方式一般有三种：点到点方式、点到多点（星型）和网状网通信。而当前各研究机构多采用点到点的光通信方式，归其原因：首先，无线光通信技术还处于实验室研究阶段（红外通信应经推广），理论架构还不够完善，有待于进一步的探索。其次，相应的国际通信标准还不够完整，不同厂商的产品型号参差不齐。再次，目前的光通信应用还限于离散的各个点，没有大规模的投入市场。因此，无线光通信距离组网还有一段路程，但相应的组网方案研究已经在进行中。本论文从实际出发，系统方案的设计目的只是为了提升通信的性能，所以仍采用了点到点的拓扑结构。 

# 2.4 无线光通信系统方案设计

通过以上调研及分析，结合无线光通信的特点，本论文设计了基于LED光源的无线光通信系统方案，具体方案如图2-2所示： 

![](file:///Users/louisyu/MinerU/无线光通信关键技术研究_刘义乐.pdf-7666a461-ad34-4d41-ba08-bee2be763aca/images/865d518653347f8dfc18ab3e1d6ea1ff3acd9b2d73d4922f5ba76543651f56fd.jpg)



图2-2 无线光通信系统方案


整个通信系统总体可分为信息发送部分（源端）和信息接收部分（宿端）两大部分。发送源端由数据缓存、信道编码、信道调制器、占空比调节器和阵列驱动光源五个大的功能模块组成；接收端系统的设计相比而言较为复杂，由PMT探测接收器、A/D转换电路及数字滤波器、判决恢复电路、同步解调、信道译码等部分功能模块组成。此外除了收发两端的设备外，在光路部分还设计了光学模块。 

在发送端，缓存器将数据源发来的信息数据缓存，随后将数据按组送入信道编码器，进而对编码处理后的数据进行调制，转换成适合光路传输的基带信号（本系统进行了DPM/PPM/OOK调制处理）。再将基带信号经过占空比调整后送入LED驱动电路驱动LED光源发光。 

在光路，设计光学系统进行光学聚焦和滤波处理。光学聚焦可以提升光发射机的通信性能，可以选用透镜或者球面聚焦系统。滤波系统一般选用与光源波长匹配的滤光片，可以滤除到达接收端探测器的其它波段的杂光，有效降低探测器前段的信噪比，简化了探测接收电路的设计。 

在接收端，首先利用探测器将探测到的光信号转变为电信号，经过整形放大电路后进行A/D变换，经过采样，量化后的数字信号被送入数字滤波器进行处理，以减低噪声的干扰。数字滤波器的输出经过判决和占空比恢复，转换成已调制的基带信号，系统检测同步字节完成同步，进行信号的解调和信道译码，最终将数据送入终端设备，完成通信。 

其中信道编码器的作用是提高数据传输的可靠性。信道编码的过程是在源数据流中插入一些额外的码元，与所要传递的信息构成一定的逻辑关系，从而在接收端对接收数据进行判错和纠错处理，以降低信息错误的概率，进而确保信息的可靠传输。信道编码是无线光通信的重要关键技术之一，这部分内容在论文的第六章做了详细介绍。 

本论文广泛调研了LED的发光特性，针对调节LED的发光占空比做了深入的研究和大量的实验测试，在系统中加入占空比调整模块的作用可以较大程度的提升光源的瞬间发光功率和发光效率，提升通信性能。论文的第三章对此模块的功能做了分析。 

将分集接收技术应用于无线光通信，是本论文系统方案设计中的又一个关键技术。多接收将给系统带来接收增益，增加接收信噪比，减少误比特率，提升系统性能。 

数字滤波器的作用是将经过A/D转换后的信号进行处理，减小其中的噪声分量的影响，进而降低信号判决、恢复错误的概率。第五章节重点介绍了数字滤波器的设计及其实验验证的结果。

# 第三章 占空比变换对LED光源性能的提升

通信中所用的光源有多种：激光器，LED和其它低压光灯，其中光源的性能（调制速率，功率）直接决定了整个系统的有效通信距离和通信质量。激光器调制速率高，但功率有限，光在无线信道中的衰耗很大，使用激光器作为光源必然限制了通信的距离。而低压光灯则正好相反，它可以发出大功率的光强，但调制的速率很低[31]。LED的特性介于两者之间，LED的响应速度可达到兆bps，而且随着LED作为绿色照明的兴起，大功率的LED开始被推出。除此之外，LED还具有结构简单、可靠性强、外观结构小巧、易于驱动、功耗低、价格便宜、寿命长等特点，特别适合应用于无线光通信系统中。 

# 3.1 LED发光特性

发光二极管（LED）的简单来说就是PN结，P型半导体内有许多可移动的空穴和固定不动的负离子，N型半导体内有许多可移动的电子和固定的正离子，对于P型半导体而言空穴是多数载流子，对于N型半导体而言电子是多数载流子。 

正常情况下，在PN结的接触面，空穴和电子相遇而复合，只剩下固定不动的正、负离子，从而形成空间电荷区，此时P端和N端之间产生了一定的势垒电压（P端为负，N端为正，称之为反向电压），阻碍PN结内的多数载流子继续扩散，达到一个平衡的状态。当在PN结的两端加上正向偏压时（P端接正极，N端接负极），正向偏压会抵消PN结的势垒电压，P区和N区的多数载流子又可以在外加电压的电场中开始向对方一侧扩散，同时外加电源给PN结两端源源不断的送入电子和空穴对，使其复合产生光子发光。当在PN结两端加上反向偏压时，会增大PN结的势垒电压，使扩散平衡区域加大，阻碍多数载流子的扩散，减少空穴和电子的复合，此时不发光。 

LED 看似结构简单，但其工作特性却受诸多因素的影响，如半导体材料的不同，温度的变化，外形结构的设计等。 

# 3.1.1 LED的发光效率

发光效率又称量子效率，包括内部量子效率与取出效率两部分，是两者的乘积。其中，内部量子效率是LED组件自身电能转化为光能的效率，与器件的材料选择、垒晶组成及空间结构相关。空穴与电子复合一部分转化为光，一部分转化为热能，转化为光的部分比上消耗的电能即为内部量子效率。取出效率指所有空穴与电子复合产生的光子经过吸收、折射、反射后，发射到达器件外部的比率。 

# 3.1.2 LED的光学特性

LED半导体材料掺杂成分的不同，导致其禁带宽度不同，产生不同的能隙，从而制成可发出不同波长光的LED。但是，半导体的能隙不是固定不变的，随着温度的升高能隙变小，所发出的光的波长也会变化，一般情况下，LED发射光的波长随着温度的上升而增加，出现红移现象。

LED是电流控制型器件，其负载特性为PN结的UI曲线，如图3-1所示。

加正向偏压时，电流随电压有较大的变化，LED的发光亮度与正向电流成正比。加反向偏压时，在不击穿的情况下，漏电流很小，当反向偏压超过击穿电压时，LED被损坏。此外，发光亮度也受温度条件的影响，空穴和电子对的复合效率随着温度的升高而降低，导致发光强度减弱。因此LED的散热设计相当重要。 

# 3.2 占空比调节

大气对光的传输有强烈的吸收作用，目前单只LED的发光功率在毫瓦级别，这必然限制了通信的有效传输，因此，提升光源的发光功率，进而提升通信系统的性能是无线光通信首先要解决的问题。 

# 3.2.1 调整占空比对LED的影响

本论文依据LED的发光特性，控制LED的驱动电流的持续时间，从而调整LED的发光时间（即调整发光的占空比)，以达到提高LED发光功率的目的。 

首先，一般LED都有标准的正向偏压，正常发光时，流入LED的电流恒定，在PN结内的空穴和电子对成一定比例的复合发光。当我们对LED驱动的输入信号进行占空比的调整后，驱动将原本连续恒定的电流变为离散的电流脉冲送给LED，LED仅在电流脉冲为高的瞬间发出光能量。根据LED的发光机理，当瞬间 

给PN结加上正向偏压和注入电流时，LED并不马上发光，由于惯性PN结需要一定的时间来压缩空间电荷区域，此时，PN结还未导通，P极和N极在不断的注入空穴和电子。当空间电荷区减小到一定程度，PN结导通，瞬间PN结内的电子和空穴浓度达到最大值，此时电子和空穴复合，从而瞬间激发出更多的光子。在相同的电压下，瞬间产生的输出光功率比连续发光时的功率有明显的提升。在通信中，一般利用LED的亮、灭来分别表示信息位的‘0’和‘1’，如果每次只用LED瞬间发出的强光来代表信息位，即可完成信息的传输，而不需要在整个信息位的周期持续的发光，因此，占空比调节既提高了LED的发光功率，又降低了器件的功耗。 

其次，LED连续发光时，电子空穴对复合所产生的热量将不断囤积，LED组件的温度将会持续升高，而高温对于LED的各项发光性能都有削弱，不仅导致电子空穴对的复合效率下降，而且还容易出现波长红移，使LED的发光质量下降。进行占空比变换后的LED发光不是连续性的，每个信息位周期内，LED仅仅发出瞬间的光，其他时间都处于未导通状态，因此LED组件热量的产生直接被减少，LED有了充分的时间散热，其发光性能受到较小的影响。

# 3.3.2 可见光LED通信

可见光通信的需要兼顾到照明的功能，因此系统设计中选用白光LED作为光源。常用的白光LED其实是由其他颜色的光混合得到的。白光LED主要由两种方式产生：一种是通过将蓝光LED芯片和荧光粉封装在一起，当LED上电发射蓝光时，荧光粉吸收蓝光并产生出黄光，利用透镜将蓝光与黄光混合，便可得出肉眼所需的白光。这种单芯片结构的白光LED成本低，亮度高，但是受荧光物质发光原理的影响，它必须受到蓝光LED芯片照射后才能发光，所以不适合做高速数据传输使用；另外一种是混合了红、蓝、绿三种基色的LED芯片，控制它们的发光强度，以获得期望的白光效果。我们称这种方法是多芯片类型白光LED，多芯片结构相对于单芯片结构发光速度更快，更适合用于可见光通信系统。通过改变三种颜色的比例，可以得到不同种类的颜色输出。应用于白光通信时，通过同步调制红、绿、蓝LED芯片发光，我们可以得到调制后的白光，同时也能用做照明使用。白光LED的种类和原理如表3-1所示：

# 3.4 本章小结

压缩占空比一方面提高了发射端的瞬间光功率，当发射端在相同功耗下，做了占空比变换的信号能使接收端检测到峰值功率更大的脉冲。其次，在信号波形发生畸变的情况下，对未做占空比变换的信号进行判决，畸变会直接导致判决错误。反之经过占空比变换后的信号，由于是先检测上升沿，再恢复信号，只要上升沿检测正确，所恢复后的信号必然得到正确的结果。最后，由于占空比的变换需要判定脉冲的上升沿，判定时钟需大于数据速率，判定过程必然会引入延时，所以当判定时钟越高，延时越小。本论文中当紫外光通信的数据速率为10Kbps，将单个周期的脉冲压缩为2us时，采用50Mbps的判定时钟（周期为20ns），判定延时为20ns，这相对于10Kbps的数据周期（0.1ms）来说可以忽略。 

因此，采用占空比变换不仅可以换来LED瞬间发光功率的提升，还可以降低信号畸变对于接收端判决的影响，使系统有了容忍无线信道畸变的能力，提升了系统性能。当然这种性能的提升需要付出高速电路为代价，比较适用于中低速的无线光通信环境。 

# 第四章 无线光通信中的数字滤波

无线光信号在大气中传输，受到强的大气损耗，经过一段距离后，接收端探测器探测到的光脉冲信号变得十分微弱，因此接收端需用高灵敏度的探测器进行探测。同时，探测器也必然将同波段或相邻波段的自然背景噪声也一同做了光电转换，形成混有杂声的信号，所以接收端探测器的输出是有效信号和背景噪声信号的叠加。除了自然背景噪声外，电路中的热噪声，器件的开关噪声都会一同混入有用信号中，影响信息的正确接收判决。 

无线光通信实验测试中，有用信息以高低能量脉冲的形式存在，混杂在信号中的噪声则多为杂乱无章的高斯噪声毛刺，为了消除噪声毛刺对于脉冲判决的影响，论文设计了数字平滑滤波器以降低噪声的影响[32]。 

# 4.1 数字平滑滤波器的设计

本论文设计了数字平滑滤波器如图4-1所示，主要包括两部分：A/D采样电路和数字滤波器。其中，利用A/D采样电路对探测器送出的模拟信号进行过频采样、量化，送入FPGA内的数字平滑滤波器进行处理，最终实现降噪功能。 

A/D电路为多路并行高速采样，量化的模拟转数字电路，本论文所用的A/D电路支持100MHz的高频采样速率，8bit数字量化。数字平滑滤波器的原理即均值滤波：对某一区间段的信号值做平均算法，然后用所得的平均值加权作为均值滤波器的输出。其中均值滤波算法一般有两种：一种是分组求平均，取一段区间的数据，计算它们的平均值，最后连续输出这一平均值来代替原来区间段的所有数据。另一种是滑动求平均法，首先设定某一长度的滑动移位寄存器（通常一组滑动移位寄存器的长度不宜大于最小脉冲的宽度），将采样所得数据依次送入移位寄存器，同时，对当前移位寄存器中的一组数据求平均值，依次输出当前求得的平均值作为均值滤波器的输出。 

本论文选择采用过频采样和滑动求均值滤波技术相结合的数字滤波方法。具体如图4-1所示。 

![](file:///Users/louisyu/MinerU/无线光通信关键技术研究_刘义乐.pdf-7666a461-ad34-4d41-ba08-bee2be763aca/images/0795701f354a93ad850dcb70a9cb13a01caa9bf282ef0d6ac3fa6534b3e41fff.jpg)



图4-1数字平滑滤波器


设A/D采样电路的输入的信号为 $y(t)$ （其包含了有用信号和噪声信号），采用 $m$ 倍于有用信号最小脉宽的频率对信号进行过频采样，即一个最小的脉冲间隔内A/D电路将会连续输出 $m$ 个采样值： $y_{1},y_{2}\dots y_{m}$ ，将其依次送入滑动平均移位寄存器内做均值算法，如果滑动移位寄存器的长度为 $n$ （ $n$ 小于等于 $m$ ），则均值滤波器最终得到输出信号为： $Y = \sum_{i = 1}^{n}y_{i} / n$ 。数字滤波器可以有效的去除有用信号上叠加的背景噪声毛刺，达到平滑信号，改善信噪比，减少信号误判的概率。

图中结果有所不同的是，由于A/D采样时钟的间隔和数据脉宽的间隔有差异，在不做占空比调整时，A/D电路在每个脉冲周期内可以采到更多的数据，经过数字滤波平均后的信号的峰值仍然保持，如图4-2中，原始信号的的峰值为0.25，经过数字滤波处理后的信号峰值基本还保持0.25，峰值功率并没有被衰减。当信号做占空比调整时，脉冲周期被压缩变窄，A/D电路在每个脉冲周期内采集到的高电平值次数有限，当数字滤波器进行平均计算时，在所有的高电平都进入移位寄存器前，平均的结果是一个逐渐递增的过程，当所有高电平样值已经充满移位寄存器时，才能得到与原始信号一样的峰值输出。因此当占空比脉宽过窄或者采样间隔多大时就会造成对高电平的采样次数过少，而至移位寄存器中的高电平数值无法被充满，则所得的输出信号的峰值就会被削弱。如图4-3的实验中，原始信号中单个脉冲的宽度为1us，则其一个高电平周期内被A/D电路采样了8次，在求平均值时，又对其求了 $n = 8$ 的平均值，因此，当实际中采样电路的时钟相位稍有偏差，便得不到8个高的电平，其滑动平均滤波的输出峰值将被削弱。图4-3中的原始信号的峰值为0.2，在经过数字滤波处理后，其峰值为0.15，被削弱了。 

因此在设计数字滤波时，不仅要考虑到自身的采样速率，平均阶数的选取，更要结合信号的实际情况（如占空比变换的情况），为了不削弱滤波后信号的峰值功率，在设计数字滤器时要确保能采到多于平均阶数的高电平值（一般建议选2倍以上）。 

# 4.3 实验测试

无线光通信中，噪声的来源主要为阳光中的同频段分量，用数字滤波器对这些噪声进行处理，可以降低信号的误判概率，提升系统通信性能。为了实际测试数字滤波器的性能，本论文借助于紫外光实验平台，设计实验测试滤波的性能及其对误码率的影响。 

本论文所设计的紫外光通信系统工作在紫外的日盲区域，接收端用有高倍放大能力的光电倍增管（PMT）进行微弱光信号的探测。实际中，虽然系统工作在日盲波段，但在不发射信号光的情况下，PMT仍然能探测到微量的该波段的自然背景噪声。当通信距离延长，信号光在大气中的路径损耗增加，高压管座需要对信号做高倍放大时，这些噪声的影响就突出出来。

# 第五章 多接收在无线光通信中的性能

分集接收，是采用分集的方法对收到的承载相同信息的若干个不相关的之路上的信号，通过合并技术将它们合为一路信号输出。分集接收可以在接收端利用多个接收器探测到多路径的独立信号，可以抗信道深衰落的影响，并获得分集增益，降低接收端对信噪比的要求。这种技术已广泛应用于第二代和第三代移动通信系统中。 

分集合并的方式一般有三种：最大比值合并，等增益合并，选择式合并。其中，最大比值合并的性能最好，但其实现较为复杂；选择式合并的实现最为简单，但效果最差；等增益合并的性能和复杂度都介于两者之间。并且随着接收器个数的增多，等增益合并的性能可近似于最大比值合并。 

# 5.1 多接收系统设计

在无线光通信中，大气信道对信号的传输产生随机性的影响，导致多径衰落，为了降低这种影响，在接收端系统设计时引入了多接收功能模块[32]。多接收的设计框架如图5-1所示：

接收端用多个探测器对光信号进行探测，并将探测到的多路信号分别单独进行放大，A/D转换以及数字滤波处理，最后送入分集接收器进行合并处理，得到合并信号。其中，信号放大电路为直接将PMT探测并经过光电转换后的电信号进行模拟放大处理，放大后的信号送入A/D采样电路。根据奈奎斯特定理，信号进行A/D变换时的采样速率必须大于2倍的信号中的最高频率，才能使采完样的数字信号完整地保留原始信号中的信息。在实际中一般都大于2倍的值。本论文中A/D电路同样采用过频时钟对放大后的模拟波形进行采样（例如，紫外通信系统中，单个窄脉冲的宽度为2us，所采用的采样频率为8MHz，是16倍的过频采样。可见光通信系统中，做完占空比的单个脉冲为20MHz，所采用的A/D采样频率为 

65MHz，为3倍频采样）。数字滤波器主要对采样后的数据做了线性变化，以降低噪声分量的影响。最终分集合并模块将多路信号进行合并，形成一路输出。 

分集接收有多种合并方式。无线光通信的载波波长为um级别，实际实验中，探测器之间的距离必然会远远大于这个波长，因此可以认为每个探测器接收恢复后的信号都是相互独立的一路信号。其次，无线光通信距离较短，虽然不同的探测器所对准的通信路径不同，但收发之间的信道情况基本趋于一致，所以信道的信噪比近似相等。考虑到不同合并方式所得到的增益效果和实现的复杂度，论文采用等增益方式合并。在几路信噪比相差不大的情况下，等增益合并可获得合并增益，即使在某一路径出现突发的深衰落的情况下，其他路径也可以进行衰落补偿。 

# 5.2 多接收的实现与实验测试

多接收的具体实现需要多套接收探测设备同时进行工作，其中最重要的部分是分集合并方式的设计，本节将具体说明此模块的实现，并用做具体实验验证多接收的性能。 

# 5.2.1 等增益合并方式的FPGA实现

本论文选用在FPGA器件上实现多路信号的合并，其中，等增益合并的难点是多路信号的相位同步。 

在无线光通信中，光子可以沿多个路径传播，到达接收端时由于路径的差异，从而产生不同的相位差，可见多径效应是限制通信速率和通信距离的一个重要因素。目前的LED通信中，受器件自身的影响，通信速率受限，本论文中所设计的紫外光通信速率为10Kbps，通信距离 $100\mathrm{m}$ ，可见光通信速率为10Mbps，通信距离 $3\mathrm{m}$ 。当系统工作在10Mbps时，由于多径效应出现 $180^{\circ}$ 相位差的多径路径差在 $30\mathrm{m}$ 左右，这一距离远远超出了系统所设计的 $3\mathrm{m}$ 的收发通信距离。与此同时，无线通信链路中视距链路占主要部分，与视距路径相差 $30\mathrm{m}$ 的通信链路的能量在接收信号中所占的比例完全可以忽略不计。 

在本论文所涉及的紫外和可见光系统中，可以认为多接收的每条路径的差值在容忍范围内，相位偏移忽略不计。因此在等增益合并中大大简化了具体设计。多路模拟电信号经过A/D转换和数字滤波处理后，送入分集合并模块进行等增益合并。图5-2是双接收情况下等增益合并方式在FPGA中实现后的仿真结果：

# 第六章 RS纠错编码在无线光通信中的应用

无线光通信情况复杂，其性能受大气环境、光学器件和自身信号特性等诸多因素的影响，为了提升无线光通信系统的通信性能，除了传统的提升发射机的发光功率和接收端的信号噪声处理外，还可以通过改变系统所使用的编码方式来改变信号的可靠性，改变系统的调制方式以改变信号的传输特性，进而改善接收端解调和译码后信号的信噪比状况，提升通信性能。 

目前，国内外对于无线光通信中的各种强度调制解调方式的研究较多，提出了OOK/PPM/DPIM/DPPM/M-DPPM等多种不同调制特性和不同阶数的调制方式[33]。相对而言，在信道纠错编码方面，针对无线光通信的研究还比较少。因此，研究一种适合于无线光通信系统的高性能的信道编码，进一步提升通信可靠性，对于无线光通信有重要的意义。 

# 6.1 信道编码

信道编码是按照某种算术的编码规则，在等待发送的信息码元中增加一些码元（监督码元或校验码元），接收端利用这些增加的码元进行解码，可以检查和修正一些由于传输而导致的码元错误，最终提高传输的可靠性。信道编码有检错功能和纠错功能，在无线光通信中，本论文选择功能更强，有纠错能力的差错控制编码。常用的差错控制编码方式主要有三种：检错重发方式，前向纠错方式和混合纠错方式。考虑到检错重发和混合纠错方式都是建立在交互式的全双工通信协议之上，本论文选择了实现更为简单的前向纠错控制编码应用于通信系统中，初步探索纠错编码在无线光通信中的性能。 

FEC 前向纠错编码(Forward Error Correction)是单方向的差错控制编码，其采用预先确定的算法增加通信数据的冗余信息度，在接收方一旦错误被检出，其接收器将根据传输的冗余信息，计算数据中的错误并纠正。 

移动通信中使用到的前向纠错编码（FEC）包括BCH编码，奇偶校验码，CRC码，配合交织使用的卷积码以及Turbo码等，其纠错性能的好坏与实现复杂度成正比。考虑到无线光通信的特点，信号受大气干扰易出现突发的连续错误，本论文选用BCH码应用于无线光通信。 

# 6.2 RS纠错编码的FPGA实现

对于线性编码而言，其最大的汉明距离为校验元数加1，而RS码正好如此，因此，RS码为极大最小距离可分码。RS码不仅能纠正随机的错误，还能纠正突 

发的错误[34]。 

基于LED的无线光通信为异步串行通信，需要从数据中提出同步时钟。如果构建较长的编码，产生一串较长的帧数据，在接收端从数据中恢复出的时钟很可能在长的数据帧结尾处发生相位偏移，不利于数据同步。另外构建长的数据帧也必然会导致数据发送缓存和接收缓存的时延过大，所以无线光通信中应该选用较短的数据帧进行数据的传输[35]。 

# 6.2.1 RS编码器设计

本论文选用RS（18，10）用于紫外通信编码。RS(18,10)的编码效率为 $5 / 9$ 最小码距 $d_{\mathrm{min}}$ 为9，每一组RS码可以纠正 $\leqslant 4$ 的符号错误。其生成多项式为： 

$$
g (x) = \prod_ {i = 1} ^ {8} \left(x + a ^ {i}\right) = x ^ {8} + 2 2 7 x ^ {7} + 4 4 x ^ {6} + 1 7 8 x ^ {5} + 7 1 x ^ {4} + 1 7 2 x ^ {3} + 8 x ^ {2} + 2 2 4 x + 3 7 。
$$

其中 $a^i$ 是 $GF\left(a^{8}\right)$ 域中的元素，本原多项式为： $P(x) = x^{8} + x^{4} + x^{3} + x^{2} + 1$ 。假设输入数据信息为： $m(x) = \sum_{i=0}^{9} m_i x^i$ ，那么校验多项式 $r(x) = x^8 m(x) \oplus g(x)$ ，将 $r(x)$ 与信息多项式 $m(x)$ 结合，得到 RS 系统码 $c(x) = x^8 m(x) + r(x)$ 。

（1）8个缓存器的初始值全为零，开关1闭合，开关2打开。第一个时钟开始，data_in第一位数据直接赋给输出RS_code的同时也与缓存8中零值异或后送入反馈电路，同 $g(x)$ 的系数（图中标出的乘法器因子）做 $GF(a^s)$ 域上的乘法运算，并与前一个寄存器的当前值异或后将结果缓存入后一级的寄存器，等待第二个时钟周期到来。 

（2）前10个周期内，一组RS码字依次输入，编码器重复以上动作。第10周期结束，此时，编码电路已完成了模 $g(x)$ 的除法功能，寄存器内所剩下的就是10位RS信息码元的校验位。 

（3）从第11个周期开始，开关1断开，开关2闭合。寄存器temp8-templ的值依次赋给RS_code输出,这8个校验位与原始的一组10个信息码字一起，组成一组RS(18，10)码输出。编完一组码后，寄存器需要全部清零，然后重复以上的动作，完成对下一组RS码的编码。 

# 6.2.2 RS译码器设计

RS 译码的基本原理是：首先将接收到的完整的一组系统码 $\{\mathrm{Rn} - 1, \mathrm{Rn} - 2, \dots \dots, \mathrm{R2}, \mathrm{R1}, \mathrm{R0}\}$ 存入缓存，本论文中 $n = 18$ ，通过计算接收多项式： 

$$
R (y) = R _ {n - 1} y ^ {n - 1} + R _ {n - 2} y ^ {n - 2} + \dots \dots + R _ {1} y + R _ {0}, \quad (y = \alpha^ {i}, i = 1, 2 \dots 8)
$$

其中 $a^i$ 属于伽罗华域 $GF\left(a^8\right)$ ，得到伴随式 $S_{i} = R(\alpha^{\prime}), i = 1,2,\dots 8$ ，如果 $S_{i} = 0$ ，则认为接收无误，若存在 $S_{i} \neq 0$ ，则由BM迭代算法确定错误位置多项式 $\sigma(x)^{[34]}$ ： 

$$
\sigma (x) = \prod_ {i = 1} ^ {\prime} \left(1 - x X _ {i}\right) = 1 + \sigma_ {1} x + \dots + \sigma_ {t} x ^ {\prime}, \quad (t = 4)
$$

其中 $\sigma_{t}$ 是由BM算法得到的错误多项式的系数。Chien搜索法将 $x = \alpha^{255 - 18 + i},i = 1,2,\dots 18$ 代入多项式 $\sigma (x)$ 计算，若 $\sigma (x) = 0$ ，则表示第i位有错误。最后Forney算法根据错误位置和伴随式解得对应的错误值，将错误纠正。 

![](file:///Users/louisyu/MinerU/无线光通信关键技术研究_刘义乐.pdf-7666a461-ad34-4d41-ba08-bee2be763aca/images/aa8b6ed0b5dbd8b8af0c7fc05c31640d98f62ae0f113b7d69febb8b8cfb40c09.jpg)



图6-2RS(18,10)译码器设计


RS 译码器的设计主要分为四个步骤，如图 6-2 所示。伴随式的计算，BM 算法得出错误方程，Chien 搜索计算错误位置，Forney 算法计算错误值[34]。译码器采用流水结构，各模块的功能相互独立，但由它的上一个模块的计算结果进行触发。Chien 算法模块根据 RS 码的纠错能力对计算结果进行判定，如果超出了纠错范围，就立刻终止译码，不再进行错误值计算，直接输出结果以节省系统的功耗。这种系统设计可以进行连续性的译码，系统的处理延迟由时延最大的模块决定。

# 6.3 实验设计与测试

无线光通信易受室外不同时段气象条件的影响，为了保证测试条件相对稳定，本论文基于图2-2的紫外光通信实验平台，选择在室内环境下进行测试（长 $\times$ 宽 $\times$ 高 $= 70\mathrm{m}\times 8\mathrm{m}\times 5\mathrm{m})$ 。 

# 6.3.1 实验设计

发射端光源选择中心波长为 $265\mathrm{nm}$ ，波束宽度为 $7^{\circ}$ 的半球面封装LED，输出光功率范围为 $0\sim 15.5\mathrm{mW}$ ，本次实验测试固定了LED的发射光功率为 $12.5\mathrm{mW}$ 。PN序列由多项式 $\mathbf{y} = x^{2} + x^{9}$ 生成，每一帧发送的PN序列长度为120字节，间隔发送2000帧。调制方式选用4-PPM，调制后信号速率为20Kbps。 

接收端选用干涉型的紫外滤光片，中心波长 $266\mathrm{nm}$ ，全波半宽 $12\mathrm{nm}$ ，峰值透过率 $20\%$ 。光电倍增管选用R7154型PMT（侧窗型，附带多极放大模块），光谱响应范围在 $160\mathrm{nm} \sim 320\mathrm{nm}$ 之间，其中放大增益可调，最大阴极放大增益达 $10^{5}$ 倍。A/D电路的有效输入在 $0 \sim 2.5\mathrm{V}$ 之间，8位输出。因此需调整功率放大的倍数，控制其输出信号的峰值为 $2.5\mathrm{V}$ 左右，使A/D变换的结果达到最佳。A/D采样频率为 $8\mathrm{MHz}$ ，有效满足了数字滤波器的需求[32]。硬判决门限设在最佳位置处（峰值的一半）。最后错误比特率通过串口（UART接口）在PC上显示。

# 6.4 本章小结

本章节将信道编码理论与无线光通信结合，设计并验证了RS码在紫外光通信中的性能。实验证明，信道编码可有效减小误码率，提升系统性能，RS码可以给紫外光系统带来平均1.37dB的通信距离增益。 

通过对信道编码以及无线光通信的研究，本论文提出将交织，卷积码和RS编码级联应用于无线光通信系统中，其中，交织将极大的分割突发错误，卷积码采用软译码方式，可以实现最大似然译码，RS码可以在以上两者的基础上进一步提高误码性能。因此如何将这三者有机的融入到无线光通信中，将有很高的研究价值。 

# 第七章 总结与展望

本论文在在目前已有的无线光通信的研究的基础上，对系统的整体架构进行自行设计和改进。针对无线光通信的特点，提出了几个解决其关键难题的技术方案。在此基础上本论文所在的项目组共同搭建了紫外光试验通信平台和白光视频通信演示平台。 

其中实验平台均采用了小型化，低功耗的LED作为光源。并使紫外光通信成功实现了距离120米，通信速率为10Kbps的准视距语音和数据通讯。白光视频通信演示系统成功完成了视频通信的演示。 

论文分四个章节分别介绍了无线光通信关键技术的改进，分析其机理并在所搭建成功的实验平台上进行了实际测试验证，所有关键技术均以应用到了目前的实验平台中。随着“绿色通信”的倡导，无线光通信作为一种无电磁污染，辐射小，功耗低的通信方式，必然获得广阔的发展前景，有重要的研究价值。 

但是，应该看到无线光通信距离真正的实用化还需付出更多的艰辛。目前，本论文所在的项目组在已有研究成果的基础之上，仍然继续着大量的科研工作，主要包括以下几个方面： 

（1）针对光源的进一步改进：受限于目前器件的工艺水平，虽然占空比变换尽最大可能的提升了LED的发光效率，节省了整体功耗，但发射机效率仍有更进一步提升的可能，对此可以从光学角度出发，利用光学聚焦技术提升光源的通信效率。 

（2）多接收技术的进一步拓展：无线光通信有别于射频通信，参照MIMO技术，继续研究多发送、多接收对于无线光通信的影响，从实验测试返回理论论证，分析其机理，完善无线光通信理论，为技术的继续改进提供理论依据。 

（3）信道调制编码：本论文将可正突发差错和随机差错的RS码应用于无线光通信，并实验证明了其对通信误比特率的影响，但其纠错能力是以更多的码字开销为代价换来的，且RS分组编码的延时过大不适合实时的语音通信。卷积码为连续的非分组编码，可以纠正随机差错，在相同实现复杂度的情况下，卷积码的软判决比RS有好的纠错性能，在实时语音通信中的应用更加适合。交织编码与卷积码相结合，可以改变信道特性，分割突发错误的位置，能将卷积码的最大似然译码性能发挥至最大，其应用于无线光通信的非实时数据传输，将能获得更高的通信性能。
