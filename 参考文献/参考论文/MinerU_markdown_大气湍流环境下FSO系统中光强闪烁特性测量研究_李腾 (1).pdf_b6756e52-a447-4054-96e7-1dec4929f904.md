# 1 绪论

# 1.1研究背景与意义

随着国家和民众对高速率信息传输需求的不断攀升，激光作为高速信息传输的载体，在 军事、通信、气象等领域的大量应用有力地促进了无线光通信技术的研究。无线光通信是以 激光为载波、大气空间为传输介质实现大容量信息传递的一种新型通信技术，它具有保密性 好、抗干扰能力强、信息容量大、传输速率高、系统尺寸小、重量轻、建造和维护费用低、 无须频率许可证等优点[1]。无线光通信在紧急情况下的应急保障通信、移动通信基站之间的 互连、军事设施或其他要害部门之间的保密通信等方面具有广泛的应用前景。 

由大气分子无规则运动产生的大气湍流效应会导致光学折射率随机起伏，使激光在传输 过程中发生光强闪烁、到达角起伏、相位起伏、光斑漂移、光束扩展等湍流效应[2]。湍流降 低了传输光束的相干性，影响了光束汇聚及准直效果，导致无法完美的进行激光传输，严重 影响了激光通信系统的功率。光强闪烁是影响通信性能的最主要因素，它会导致接收端接收 信号强度下降到判决门限以下，增大了通信系统的误码率，当光强闪烁效应强烈时，甚至会 中断当前通信[3-6]。 

研究表明，雾霾天气发生时，受雾霾顶层辐射冷却影响，大气温度变化加剧，导致湍流 强度增大[7]。同时研究人员发现雾霾的产生、消散及维持均与湍流的影响有关。当雾霾越严 重时，湍流强度也会越大[8,9]。西安地处关中地区，处在黄土高原和秦岭山脉的中间，积聚 后的污染物很难向外部扩散，同时冬季天气较为寒冷，每年的11月中旬至次年三月中旬会进 行大面积的供暖，造成大气污染，且城市规模大、建筑物众多，导致秋冬季节西安频繁出现 严重的雾霾天气[10,11]，使得该地区大气湍流较为复杂。因此，本文结合理论分析，通过搭建 实验测量系统分析了西安地区不同湍流环境下光强闪烁特性，为该地区无线光通信系统的应 用提供参考。 

# 1.2国内外研究现状

# 1.2.1国外研究现状

最早在二十世纪中期，人们就开始对光强闪烁特性进行研究。1948年，由VonKarman 提出了描述湍流规律的频谱来描述湍流的变化[12]。1958年，由Tatarskii提出了光强闪烁模型， 为此后光强闪烁的概率密度研究建立了基础[13]。1965年，Gracheva等人开始对光强概率密度 分布进行研究，并发现其分布模型与Rytov方差大小相关，其概率密度服从对数正态分布[14]。 1974年，T Wang等人发现对数正态分布并不能准确的描述光强闪烁，光强闪烁效应强烈时， 其概率密度与对数正态分布存在偏差[15]。 

二十世纪90年代，人们发现激光在湍流大气中传输时，其闪烁效应与湍流内外尺度有关。 1994年，Charles A.Davis等人基于Rytov理论研究了光强归一化起伏方差与大气湍流内尺度的 

关系，发现随着内尺度增大，归一化起伏方差会降低，最终达到饱和状态[16]。1999年， L.C.Andrews提出了Gamma-Gamma分布模型来描述高斯光束的闪烁模型，核心思想是用大尺 度涡旋来调制小尺度涡旋，并推导出了光强闪烁方差的表达式，解决了光束通过大气湍流传 输时受到不同尺度湍流的影响[17]。 

由于大气湍流的复杂性，研究人员发现分析湍流环境中激光的传输特性需要用实验研究 来弥补理论分析的局限性。二十一世纪初，各国研究人员开始通过开展实验，进一步分析激 光传输规律。 

2005年，美国的Philips等人开展了链路长度分别为1km、2km、5km的激光传输实验， 比较了不同模式的激光器在湍流环境中传输时的光斑漂移量，结果表明激光光束的光斑漂移 量与激光器发射孔径相关[18]。2007年，美国陆军研究实验室ArnoldTunick采用外场实验在 $2 . 3 3 \mathrm { k m }$ 链路上研究了光强闪烁规律，发现弱湍流下光强闪烁服从对数正态分布，并且温度是 影响大气折射率结构常数 $C _ { n } ^ { 2 }$ 的主要因素[19,20]。2007年，美国Central Florida大学F.S.Vetelino 等人通过开展实验研究了光强闪烁特性与接收孔径的关系，结果表明当发射孔径不变时，不 同接收孔径下的光强概率密度服从不同的概率分布模型，说明光强闪烁特性与接收孔径大小 相关[21]。2008年，伊朗A. Chaman Motlagh等人使用改进的模型分析了大气湍流对FSO系统在 弱湍流和强湍流条件下的信噪比和误码率的影响[22]。2013年，Halil T. Eyyuboglu使用Rytov 方法分析了艾里光束的闪烁特性，推演出单个接收器坐标位置闪烁指数表达式，并与高斯光 束相比较，发现在相同接收功率的基础上，艾里光束受到光强闪烁的影响更小[23]。2015年， Hamza Gerçekcioğlu等人通过实验比较了阵列光束和非线性光束在非Kolmogorov湍流中传输 时的光强闪烁特性[24]。2017年，Hamza Gerçekcioğlu等人通过仿真模拟研究了弱湍流海洋大 气介质中的高斯激光闪烁特性，使用Rytov方法导出平顶高斯光束的轴上闪烁指数的公式， 并发现闪烁概率密度分布服从对数正态分布，并比较了传播距离、波长、波束尺寸和信噪比 （SNR）的变化关系[25]。2018年，AksenovVP等人利用数值模拟比较了涡旋光束和高斯光 束在湍流环境中传输时的光强闪烁统计特性，发现涡旋光束光强闪烁概率密度函数更服从指 数分布，高斯光束光强闪烁概率密度函数服从对数正态分布[26]。 

# 1.2.2国内研究现状

虽然我国对光强闪烁特性的研究较晚，但通过中科院安徽光机所、西安理工大学、长春 理工大学、电子科技大学等单位对光强闪烁特性的充分研究，也取得了重要的成果。二十世 纪90年代，宋方正最早在《应用大气光学基础》一书中详细讨论了大气激光传输的基本理论 和大气激光传输的影响效应[27,28]。此后，研究人员陆续开展激光传输实验对光强闪烁特性进 行研究。1997年，中科院安徽光机所饶瑞中率先在合肥进行了链路长度为 $5 0 0 \mathrm { m }$ 的传输实验， 系统的观测全年光强闪烁与光斑统计特征。结果表明：大部分时间该地区光强闪烁处于弱起 伏状态，并且在弱起伏条件下，光强闪烁概率密度服从对数正态分布[29]。2007年，哈尔滨工 业大学潘峰等人开展了链路长度为3.5公里的激光传输实验，使用到达角起伏法测量了实验 期间大气折射率结构常数，发现城市中湍流强度较大，同时计算了保证无线光通信系研能够 

正常通信时的误码率和闪烁衰落量[30]。2008年，哈尔滨工业大学的可调谐实验室研究人员加 长了链路距离，进行了11.8公里的激光传输实验分析光强闪烁统计特性[31,32]。2010年，靖旭 等人通过实验比较了接收孔径与闪烁的关系。结果表明：受孔径平均的影响，相同的湍流状 态下，接收孔径越小，计算得到的大气折射率结构常数越大。当接收孔径固定时，不同实验 条件下的闪烁概率密度均服从对数正态分布，但其拟合效果受湍流强弱影响[33]。2011年，长 春理工大学陈纯毅等人通过使用闪烁指数法计算了实验期间的大气折射率结构常数，并比较 了接收孔径为50mm和200mm时的闪烁指数，发现可以通过增大接收孔径的方式抑制光强闪 烁效应[34]。2011年，哈尔滨工业大学赵雅琴等人研究了多波束传播技术在大气WOCDMA系 统中对光强闪烁现象的抑制效果，通过仿真实验，发现采用多波束传播技术可以减少光强闪 烁的影响，提高系统的性能，但不能抑制多址干扰[35]。2013年，西安电子科技大学易湘基于 非Kolmogorov幂律谱和扩展的Rytov理论对光强闪烁造成无线激光通信系统信号衰落的影响 进行了研究，并基于广义vonKármán谱，率先推导出从弱至强湍流环境下平面波和球面波的 光强闪烁系数数学表达式[36]。2015年，海军航空工程学院吴晓军等人利用大孔径闪烁仪分别 进行了链路长度 $1 . 4 \mathrm { k m }$ 和 $0 . 9 \mathrm { k m }$ 的激光传输实验来对比分析了公路环境和近海面环境的传输 差异。结果表明：公路环境下的湍流环境和光强闪烁程度均大于近海洋环境[37]。2015年，太 原理工大学Hongyan Wei等人研究了倾斜大气湍流中回波的闪烁指数，基于倾斜路径中的光 波传播理论和高度相关的ITU-R湍流结构常数模型，并考虑内部尺度和外部尺度，获得回波 的轴上闪烁指数的表达式。结果表明，轴上闪烁指数受内部尺度、外部尺度、波长和目标高 度的影响[38]。2016年，西安电子科技大学韩香娥等人基于广义惠更斯-菲涅尔原理，采用Rytov 方法推导出阵列光束在非Kolmogorov湍流大气中传输时的光强闪烁指数表达式[39]。2016年， 西安理工大学柯熙政等人利用对数正态分布和Gamma-Gamma分布分别对信道进行建模，研 究相位噪声对系统接收信噪比和误码率的影响，发现随着光强起伏方差的增加，星座图相位 角度和系统误码率也逐渐增大[40]。2017年，华侨大学陈子阳实验研究了涡旋光束在大气湍流 中的传播特性，以及具有不同拓扑电荷的涡旋光束的闪烁指数。结果表明，在弱湍流状态下， 涡旋光束的闪烁指数高于高斯光束，但在强湍流状态下传播 $4 0 0 \mathrm { m }$ 后，涡旋光束的闪烁指数 低于高斯光束[41]。2018年，兰州理工大学王慧琴等人进行了沙尘天气下光强闪烁及衰减特性 的实验测量，为兰州地区大气激光通信系统的发展提供了理论依据[42,43]。2018年，武汉大学 张娜娜等人基于液晶空间光调制器进行了Non-Kolmogorov湍流的室内模拟实验，通过分析光 强概率密度分布、光强闪烁指数及光强时间频谱，发现在弱湍流环境下，归一化对数光强服 从对数正态分布，并且当湍流强度减弱时，对数正态分布拟合效果更优；随着湍流功率谱幂 律的减小，光强时间频谱高频段的幂律也呈减小趋势[44]。2018年，哈尔滨工业大学付玉龙等 人进一步分析了强湍流到非Kolmogorov湍流激光束闪烁指数，利用广义有效大气光谱和扩展 的Rytov方法，推导出了通过强湍流到非Kolmogorov湍流传播的高斯光束的闪烁指数的理论 表达式[45]。

# 2.大气环境对激光传输的影响

当激光在大气信道中进行传输时，激光信号就不可避免地受到环境的影响，降低传输性 能。激光在传输过程中受大气中的气体分子和气溶胶粒子的影响，会发生大气吸收和散射效 应，降低了传输信号的功率。同时大气湍流也是影响激光信号传输特性的主要原因。本章将 针对无线激光通信系统介绍大气的组成结构及大气运动的特性，分析大气的组成成分对激光 传输的影响。 

# 2.1大气湍流描述

# 2.1.1 大气基本状态及结构组成

众所周知，大气是围绕地球的一层气体屏障，从地表垂直向上可延伸到数千千米的高空。 大气是由多种气体混合而成的，氮气大约占据全部大气的 $78 \%$ 、氧气大约占据全部大气的 $21 \%$ 、剩余部分由少量的氩、氢、氖、臭氧、二氧化碳等惰性气体组成。水汽在大气中含量 很少且都集中大气底层，超过 $50 \%$ 的水汽集中在 $2 \mathrm { k m }$ 以下，大约 $7 5 \%$ 的水汽集中在 $4 \mathrm { k m }$ 以下。 并且，水汽有多种存在形态，例如云、雾、霜、雪等，因此它是影响天气变化的主要因素。 同时大气中还存在很多的固体微粒，统称为气溶胶粒子[46]。 

大气环境是随着粒子运动随时变化的，为了便于研究，根据大气的温度、成分、电离状 态等在垂直方向的分布特征可将地球大气分为五个主要层次[47]： 

对流层：一般认为是紧贴地球表面的一层，距离地球表面10千米左右（不同纬度下对流 层的厚度不同，中纬度地区大约是9千米），受地面的影响最大。贴近地表的大气受热上升， 对流层顶层的空气下降，带动了大气分子做对流运动，因此这一层被称为对流层。对流层大 气约占大气总重量的 $7 5 \%$ ，海拔高度 $0 { \sim } 1 0$ 千米之间，这一层发生了所有的气象变化，无线激 光通信系统通常使用的信道也在对流层[48]。 

平流层：也称同温层，一般位于对流层顶一直延伸到海拔50千米的高空区域。相比对流 层气流运动，平流层大气运动相对平缓，并且基本在水平方向运动。这一层氧分子受紫外线 的作用变为臭氧分子并释放热能，是紫外辐射的主要吸收区，大气能见度高，并且基本不存 在尘埃粒子。 

中间层：位于平流层的顶部一直延伸到海拔 80千米的高空区域。中间层臭氧分子极少， 紫外线无法被吸收，因此气温随高度的增加迅速下降。这一层大气上下温差较大，层中空气 产生垂直对流运动，所以也称高空对流层或者上对流层。平流层与中间层合称为中层大气。 

热层：位于中间层向上500千米的高空区域，又称电离层。热层大气占大气总重量 $0 . 5 \%$ ， 空气密度很小，受太阳辐射的影响，大气中气体分子温度急剧上升，高度越高，热层大气温 度也越高。 

逸散层：热层以上的大气统称为散逸层。逸散层中仅有少量的气体分子，并且由于粒子 受到地球的引力很小，使得运动中的粒子挣脱地球引力逸散到太空，导致逸散层空气稀薄， 

温度更高。 

# 2.1.2大气湍流的形成及特点

大气层中空气密度的无规则起伏称为大气湍流，湍流对光束传输的影响称为湍流效应 [49]。本文主要研究湍流效应对大气激光传输造成的影响。1940年，有学者假定大气是一种粘 性流体，粘性流体运动过程中无规则的起伏就定义为大气湍流，这一理论被称为经典湍流理 论。流体运动主要表现为两种状态：层流和湍流。层流属于规则运动，湍流则属于不规则运 动。

图2-1 反映了层流和湍流的转化机理。由于大气温度和成分都是不均匀的，导致大气中 各个位置压强也不一致，并且大气中水汽和气溶胶粒子的运动强度很大，因此大气分子会被 带动进行无规则的运动，运动形式从层流向湍流变化，大气环境更加复杂，湍流就形成了。 通常可以用雷诺数（Reynolds）表示流体运动的状态特征。研究发现存在一个临界值，当流 体雷诺数小于此值时，流体处于规则的层流；反之，流体变为不规则的湍流。雷诺数可以被 定义为[50]： 

$$
R _ {e} = \frac {\rho V}{\mu} L = \frac {V}{v} L \tag {2-1}
$$

式中： 

$\rho$ 传输介质的密度； 

$V$ 流体的运动速度； 

$\nu$ 运动粘性系数； 

$\mu$ 流体的运动粘滞率； 

$L$ — 流体运动的尺度。 

在地表附近，表征流体的特征速度 $V$ 在 $1 \mathrm { m / s }$ 到 $5 \mathrm { m / s }$ 之间变化，典型的流体运动尺度 $L \approx 2 m$ ，运动粘性系数 $\nu = 0 . 1 5 \times 1 0 ^ { - 4 } m ^ { 2 } / s$ ，此时得到的雷诺数 $R _ { e } \approx 1 0 ^ { 5 }$ 。当 $R _ { e }$ 较大时，就可 以忽略该流体运动中内摩擦力的影响，运动产生的动能会持续转变为流体的势能，导致流体 变化为不规则的湍流状态；当 $R _ { e }$ 较小时，运动过程中流体内摩擦力的影响明显，迅速的将 

运动产生的动能转化为热能消耗，因此呈现很稳定的层流状态。当流体运动从规则的层流状 态向不规则的湍流状态开始变化时，会开始产生一些湍流涡旋，涡旋尺度与流体特征尺度相 近，这种情况的涡旋尺度就是湍流外尺度；当雷诺数持续增大时，大尺度涡旋将会因为自身 的不稳定性开始分裂，传递能量的同时也会产生一些尺度较小的涡旋。这个过程会一直持续， 随着雷诺数越来越小，最会只会存在特征尺度较小的涡旋，最终湍流涡旋停止分裂，这种情 况下的涡旋尺度就是湍流内尺度。 

经典湍流理论是以速度场起伏为核心的，但是在无线激光通信系统中，主要是由温度场 引起大气折射率随机起伏影响激光传播性能，造成这种折射率随机起伏的湍流被称为光学湍 流。与经典湍流相比，光学湍流最大的区别是光学湍流主要由温度起伏造成，但两者形成和 发展过程是类似的。本文主要针对光学湍流对激光传输的影响进行研究。 

# 2.2大气吸收与散射

大气激光通信过程中激光传输方向会发生微小的改变，造成能量衰减，其原因是受到由 大气分子和气溶胶粒子造成的吸收和散射效应影响，导致通信系统误码率增高。 

# 2.2.1 大气吸收

由于大气中不仅存在各类大气气体分子，而且还夹杂着大量的气溶胶粒子，当激光在大 气信道中传输时，大气分子与信号中的带电粒子发生极化作用，造成气溶胶粒子以激光的频 率开始运动。粒子内部能量增大，光辐射为了克服大气信道中的阻力，能量开始衰减，这就 是大气的吸收效应[51]。 

研究表明，影响大气吸收的主要因素之一是激光波长，其原因是大气中的水汽分子、臭 氧分子、二氧化碳等分子吸收波长不同。水汽分子是红外辐射的主要吸收体，水汽分子吸收 带间的空隙就形成了“大气窗口”[52]。当激光波长在“大气窗口”波段时，受到大气吸收效应影 响就非常小，因此目前用于无线激光通信的激光器波长都在这一波段。大气吸收效应主要发 生在对流层大气中，随着海拔高度的增加，其影响也明显减小。 

大气吸收效应主要由气体分子吸收和气溶胶粒子吸收两部分组成，其衰减系数表示为： 

$$
\alpha (\lambda) = \alpha_ {m} (\lambda) + \alpha_ {n} (\lambda) \tag {2-2}
$$

式中： 

$\alpha _ { m } ( \lambda )$ 气体分子吸收衰减系数； 

$\alpha _ { n } ( \lambda )$ 气溶胶粒子吸收衰减系数。 

# 2.2.2 大气散射

当激光通过大气时遇到气体分子及气溶胶粒子时，传输方向发生了改变，这种光学现象 就称为散射。散射并不会减少激光能量，只是改变了传输方向，导致接收端的光功率降低。 按照激光波长和散射粒子尺度相对大小，可以将散射分为瑞利散射和 Mie散射。 

通常，由于大气分子直径很小，所以在近红外波段，分子直径远小于激光波长，此时发 生的散射现象就可以定义为瑞利散射（分子散射）。瑞利散射系数可表示为[53]： 

$$
\sigma_ {m} = 0. 8 2 7 \times N \times A ^ {3} / \lambda^ {4} \tag {2-3}
$$

式中： 

N — 单位体积内分子数； 

A — 散射元横截面积； 

2 激光波长。 

气溶胶是由固体或液体小质点悬浮在气体介质中形成的胶体粒子。在实际激光传输中， 大气气溶胶粒子的直径大于激光波长，此时瑞利散射不在适用，通常使用Mie散射来进行分 析。Mie散射系数可表示为[54]： 

$$
\sigma_ {n} = N (r) \pi r ^ {2} Q _ {s} \left(X _ {r}, m\right) \tag {2-4}
$$

式中： 

r — 大气气溶胶粒子半径； 

$N { \left( r \right) }$ —— 大气信道中单位体积内的气溶胶粒子数； 

$Q _ { s }$ 散射效率。 

无论是吸收还是散射都会导致光辐射强度损耗，形成大气的衰减效应，引起光功率的衰 减，通常用朗伯特-比尔定律来进行描述[55]： 

$$
I (\lambda , L) = I (\lambda , 0) \exp [ - \alpha (\lambda) L ] \tag {2-5}
$$

式中： 

$L$ — 激光传输距离； 

$I ( \lambda , 0 )$ —— 波长为的激光在位置 $L = 0$ 处的光强； 

$\alpha ( \lambda )$ 衰减系数。 

# 2.3大气湍流的特征参数

大气折射率结构常数和大气相干长度等湍流特征参数能够很好的描述大气湍流状态，本 节主要对常用湍流特征参数进行阐述。 

# 2.3.1大气折射率结构常数

光学湍流主要是由大气温度变化引起折射率随机起伏，通常使用大气折射率结构常数 （ $C _ { n } ^ { 2 }$ ）来量度这种随机起伏强度。分析激光在大气湍流中传输的影响时，首先要考虑的是 小尺度涡旋造成的影响。根据Kolmogorov湍流理论，则任意空间两点的折射率结构函数表达 式为[56]： 

$$
D _ {n} \left(r, r ^ {\prime}\right) = \overline {{\left[ n (r) - n \left(r ^ {\prime}\right) \right] ^ {2}}} \tag {2-6}
$$

研究表明，介质均匀的空间任意两点的折射率结构常数只受这两点间距离的影响，并不 受其所在位置的干扰，因此，介质均匀的空间内两点的折射率结构函数也可表示为： 

$$
D _ {n} \left(r, r ^ {\prime}\right) = D _ {n} \left(r, - r ^ {\prime}\right) \tag {2-7}
$$

当激光在各向同性介质中传播时，其折射率结构函数为： 

$$
D _ {n} = C _ {n} ^ {2} r ^ {2 / 3} \tag {2-8}
$$

式中： 

r 惯性区域内任意两点之间的距离。 

Kolmogorov 使用量纲分析的方法发现湍流局部惯性区域内两点的折射率结构函数与它 们距离的 2/3 次方成正比，这一发现就是著名的“2/3 次方定律”[57]。因此，湍流大气的折射 率结构函数为： 

$$
n = 1 + 7 7. 6 \left(1 + 7. 2 5 \times 1 0 ^ {- 3} \lambda^ {- 2}\right) \left(\mathrm {P} / T\right) \times 1 0 ^ {- 6} \tag {2-9}
$$

式中： 

T 温度； 

$P$ — 大气压； 

-光波波长。 

对式（2-8）和式（2-9）两边进行微分，整理得到： 

$$
C _ {n} ^ {2} = \left[ 7 7. 6 \times 1 0 ^ {- 6} / \mathrm {T} ^ {2} \left(1 + 7. 2 5 \times 1 0 ^ {- 3} \lambda^ {- 2}\right) \bar {P} \right] ^ {2} C _ {T} ^ {2} \tag {2-10}
$$

式中： 

$C _ { T } ^ { 2 }$ 温度结构常数。 

根据 Davis 理论，湍流可按照折射率结构常数被划分为三类[58]，即当 $C _ { n } ^ { 2 } \geq 2 . 5 { \times } 1 0 ^ { - 1 3 } \mathrm { m } ^ { - 2 / 3 }$ 时 为强湍流；当 $6 . 4 { \times } 1 0 ^ { - 1 7 } \mathrm { m } ^ { - 2 / 3 } { \le } C _ { n } ^ { 2 } < 2 . 5 { \times } 1 0 ^ { - 1 3 } \mathrm { m } ^ { - 2 / 3 }$ 时为中等强度湍流；当 $C _ { n } ^ { 2 } < 6 . 4 \times 1 0 ^ { - 1 7 } \mathrm { m } ^ { - 2 / 3 }$ 时为 弱湍流。大气折射率结构常数随地理位置、高度、气象条件和昼夜条件的不同变化很大。大 气中气体分子和气溶胶粒子数量随着海拔的升高而降低，对流运动也随之变弱，导致大气湍 流强度降低。 

# 2.3.2大气相干长度

1965年，Fried在研究天文望远镜时发现了大气相干长度 $r _ { 0 }$ ，也称Fried参数[51]，是衡量 大气湍流强度常见的参数之一，描述了激光在湍流大气中传输时光束横截面上的空间相干特 性。大气相干长度 $r _ { 0 }$ 的物理定义为：在激光传输过程中，调整接收孔径的大小，使其相位起 伏方差为1rad，则此时的接收孔径半径就是当前状态的大气相干长度 $r _ { 0 }$ 。 

当激光在各向同性的湍流中传输时，相位空间结构函数 $D _ { \varphi } \left( r \right)$ 为[59]： 

$$
D _ {\varphi} (r) = 2. 9 1 k ^ {2} r ^ {5 / 3} \int_ {0} ^ {L} C _ {n} ^ {2} (h) d h \tag {2-11}
$$

式中： 

h 激光传输路径的积分变量； 

$k$ 激光波数； 

$r$ 距离标量； 

L 传输距离。 

Fried 研究发现光学成像质量与接收孔径和大气相干长度 $r _ { 0 }$ 相关，接收孔径越大，成像 效果越好，但当接收孔径大于大气相干长度 $r _ { 0 }$ 时，继续增大接收孔径，成像质量不再有所改 

善。此时图像分辨率达到最大值，因此，式（2-11）可简化为： 

$$
D _ {\varphi} (r) = 6. 8 8 \left(r / r _ {0}\right) ^ {5 / 3} \tag {2-12}
$$

相同的湍流环境下，平面波和球面波计算得到的大气相干长度 $r _ { 0 }$ 并不相同。其表达式分 别为： 

$$
r _ {0} = \left[ 0. 4 2 3 k ^ {2} \sec \varphi \int_ {0} ^ {L} C _ {n} ^ {2} (h) d h \right] ^ {- 3 / 5} (\text {平 面 波}) \tag {2-13}
$$

$$
r _ {0} = \left[ 0. 4 2 3 k ^ {2} \sec \varphi \int_ {0} ^ {L} C _ {n} ^ {2} (h) (h / L) ^ {5 / 3} d h \right] ^ {- 3 / 5} (\text {球 面 波}) \tag {2-14}
$$

式中： 

$\varphi$ 是天顶角； 

$C _ { n } ^ { 2 } ( h )$ — 大气折射率在高度h上的结构常数。 

# 2.3.3 等晕角

Fried 将等晕角定义为[60]： 

$$
\theta_ {0} = \left[ 2. 9 1 k ^ {2} (\sec \beta) ^ {8 / 3} \int_ {0} ^ {L} C _ {n} ^ {2} (\eta) \eta^ {5 / 3} d \eta \right] ^ {- 3 / 5} \tag {2-15}
$$

式中： 

$\beta$ —天顶角； 

$k$ — 激光波数， $k = 2 \pi / \lambda$ 。 

当激光在湍流环境中传输时，设传输路径与成像路径之间夹角为 $\theta$ ，则存在一个 $\theta < \theta _ { 0 }$ 的 圆锥区域，在这一区域内由大气湍流引起的激光相位畸变不发生变化，此时可用自适应光学 系统对其进行相位校正。当 $\theta > \theta _ { 0 }$ 时，光束不在这一圆锥区域，相位畸变不再相关，此时自 适应光学无法对其相位进行校正。 

# 2.4本章小结

本章主要对大气的组成结构及其产生的影响进行了阐述。太阳辐射以及各种气象因素会 改变大气温度，造成大气分子与气溶胶粒子随机运动，导致实际大气环境较为复杂。同时在 激光传输过程中，由大气分子和气溶胶粒子造成的大气吸收与散射效应会导致接收光功率衰 减，其衰减程度与激光波长相关，可以通过选择合适的工作波长来减小大气吸收和散射效应 的影响。 

# 3.湍流效应对激光传输的影响

大气湍流会导致光束扩展、光强闪烁和到达角起伏等现象，造成激光在传输过程中相干 性的降低，影响大气激光通信系统的性能。当激光在湍流环境中传输时，接收端的光斑直径 大于理想的光斑直径，这一现象被称为光束扩展；接收端光斑中心位置发生随机变化，这种 随机变化就是光斑漂移；接收端光强度随时间随机起伏，空间位置也发生改变，这种变化被 称为光强闪烁。本章将对湍流效应对大气激光通信系统的影响进行阐述。 

# 3.1 湍流效应的主要表现方式

湍流是大气运动的一种重要形式，它能够极大的加强大气中动量、热量、水汽及尘埃物 的随机交换，并且交换强度远大于大气分子运动。由温度场引起大气折射率随机起伏影响激 光传播性能，造成激光在传输过程中出现光束扩展、光斑漂移、到达角起伏、光强闪烁等现 象，其本质是湍流造成激光在传输过程中的相干性的降低。

$$

# 3.1.3 到达角起伏

到达角起伏效应的物理意义是指激光在湍流环境中传输时，其光束横截面受不同尺度的 湍流涡旋影响产生不同的折射率，导致激光束的相位面随机变化。 

当激光在湍流大气中传输时，任意距离处的波面倾角 $d \alpha _ { c }$ 可表示为[65]： 

$$
d \alpha_ {c} = \frac {\Delta n \left(z ^ {\prime}\right)}{w \left(z ^ {\prime}\right)} d z ^ {\prime} \left\langle I ^ {2} \right\rangle = \left\langle x ^ {2} \right\rangle \left\langle y ^ {2} \right\rangle = \left(1 + \sigma_ {x} ^ {2}\right) \left(1 + \sigma_ {y} ^ {2}\right) \tag {3-3}
$$

式中： 

$f$ — 激光曲率半径； 

$\alpha _ { 0 }$ a 激光光束半径; 

激光波长; 

z 传输距离； 

dz 垂直于传输方向的空气薄层。 

将所有位置的波面倾角 $d \alpha _ { c }$ 进行累加，就可以得到激光传输产生的到达角为： 

$$
\alpha_ {c} = \int_ {0} ^ {s} \frac {\Delta n \left(z ^ {\prime}\right)}{w \left(z ^ {\prime}\right)} \tag {3-4}
$$

因此，到达角起伏方差为： 

$$
\left\langle \alpha_ {c} ^ {2} \right\rangle = \int_ {0} ^ {s} \int_ {0} ^ {s} \frac {\left\langle \Delta n \left(z _ {1} ^ {\prime} z _ {2} ^ {\prime}\right) \right\rangle}{w \left(z _ {1} ^ {\prime}\right) w \left(z _ {2} ^ {\prime}\right)} d z _ {1} ^ {\prime} z _ {2} ^ {\prime} \tag {3-5}
$$

# 3.1.4 光强闪烁

光强闪烁效应是指激光在湍流大气中传输时，会通过大量不同尺度的湍流涡旋，受衍射 和散射效应的影响，导致接收端光强随机起伏。这种光强的随机起伏也称信道衰落，信道衰 落会导致接收信号的光强下降到判决门限之下，增大了通信系统误码率。光强闪烁严重时， 甚至造成通信中断。 

通常用归一化对数起伏方差 $\sigma _ { l } ^ { 2 }$ 来描述光强闪烁特性，其表达式为[66]： 

$$
\sigma_ {l} ^ {2} = \left(\left\langle I ^ {2} \right\rangle - \left\langle I \right\rangle^ {2}\right) / \left\langle I \right\rangle^ {2} \tag {3-6}
$$

式中： 

光强； 

 —— 系综平均即长时间的统计平均。 

弱湍流环境下，当满足Kolmogrov谱条件时，根据Rytov近似理论，平面波的对数强起伏 方差（也叫作Rytov方差） $\sigma _ { R y t o \nu } ^ { 2 }$ 为 

$$
\sigma_ {R y t o v} ^ {2} = \left(\left\langle \ln I \right\rangle - \left\langle \ln I \right\rangle^ {2}\right) = 1.23 C _ {n} ^ {2} k ^ {7 / 6} L ^ {1 / 6} \tag{3 - 7}
$$

式中： 

2Cn — $C _ { n } ^ { 2 }$ — 大气折射率结构常数； 

k — 激光波数； 

# $L$ — 传输距离。

由公式（3-7）可以看出：影响光强闪烁特性的因素主要有大气折射率结构常数、激光 波数和传输距离。当增大传输距离 $L$ 或湍流变化更为剧烈，即大气折射率结构常数 $C _ { n } ^ { 2 }$ 增大 时，闪烁指数 $\sigma _ { I } ^ { 2 }$ 也随之增加。 

激光在弱湍流环境下进行传输时，闪烁指数 $\sigma _ { I } ^ { 2 }$ 与 $\sigma _ { R y t o \nu } ^ { 2 }$ 的关系为： 

$$
\sigma_ {I} ^ {2} = \exp \left(\sigma_ {R y t o v} ^ {2}\right) - 1 \tag {3-8}
$$

在弱起伏状态下，存在Rytov近似关系，即 $\sigma _ { l } ^ { 2 } \approx \sigma _ { R y t o \nu } ^ { 2 }$ 。由式（3-8）可以看出当闪烁指 数 $\sigma _ { I } ^ { 2 }$ 逐渐增大时， $\sigma _ { R y t o \nu } ^ { 2 }$ 值会随之增大。当 $\sigma _ { I } ^ { 2 } > 1$ 时，光强闪烁达到饱和状态，饱和状态下 随着 $\sigma _ { R y t o \nu } ^ { 2 }$ 的增大，闪烁指数 $\sigma _ { I } ^ { 2 }$ 会开始减小。为了进一步分析光强闪烁特性，研究人员提出 了强起伏理论。 

为解决强湍流环境下光强闪烁饱和的问题，可以使用基于Rytov理论的Andrews-Prokhor ov和Churnside渐进分析模型对闪烁指数进行求解分析。当激光在由弱至强的湍流环境下进行 传输时，其闪烁指数为[67]： 

$$
\sigma_ {I} ^ {2} = \left(1 + \sigma_ {x} ^ {2}\right) \left(1 + \sigma_ {y} ^ {2}\right) - 1 \tag {3-9}
$$

式中： 

$\sigma _ { x } ^ { 2 }$ 正规化的大尺度起伏方差，主要源于大尺度涡旋的散射效应； 

$\boldsymbol { \sigma } _ { y } ^ { 2 }$ 正规化的小尺度起伏方差，主要源于小尺度涡旋的衍射效应。 

激光光斑的光强闪烁特性不仅可以用对数强度方差 $\sigma _ { I } ^ { 2 }$ 来表示，也可用对数振幅方差 $\sigma _ { \chi } ^ { 2 }$ 来表示。 

在弱湍流环境下，对数振幅方差 $\sigma _ { \chi } ^ { 2 }$ 可表示为[68]： 

$$
\sigma_ {\chi} ^ {2} = 0. 3 1 C _ {n} ^ {2} k _ {0} ^ {7 / 6} r ^ {1 1 / 6} (\text {平 面 波}) \tag {3-10}
$$

$$
\sigma_ {\chi} ^ {2} = 0. 1 2 5 C _ {n} ^ {2} k _ {0} ^ {7 / 6} r ^ {1 / 6} \text {(球 面 波)} \tag {3-11}
$$

此时， $\sigma _ { \chi } ^ { 2 }$ 是满足高斯分布形式的。 

在强湍流环境下，对数振幅方差 $\sigma _ { \chi } ^ { 2 }$ 可表示为： 

$$
\sigma_ {\chi} ^ {2} = 1 + 0. 8 6 \left(\sigma_ {I} ^ {2}\right) ^ {- 2 / 5} (\text {平 面 波}) \tag {3-12}
$$

$$
\sigma_ {\chi} ^ {2} = 1 + 2. 8 \left(\sigma_ {I} ^ {2}\right) ^ {- 2 / 5} (\text {球 面 波}) \tag {3-13}
$$

# 3.2光强闪烁概率分布模型

系统的研究激光在不同湍流环境中传输时的闪烁特性是设计无线激光通信系统中必不 可少的一环。对于实际大气湍流，概率密度分布是描述其统计特征的最基本方法。研究人员 已经提出不同的光强闪烁概率密度函数模型来描述不同湍流环境下的光强闪烁特性。 

# 3.2.1弱湍流环境光强闪烁概率分布

研究人员经过大量理论分析和实验结果已经证明对数强度起伏统计特性服从正态分布， 这也说明光强闪烁特性服从正态分布特征的。在弱湍流条件下，一般认为归一化光强I (关于 

$\langle I \rangle = 1$ 的归一化)服从对数正态分布，其概率密度函数为[69]： 

$$
p _ {l} (I) = \frac {1}{\sqrt {2 \pi \sigma_ {I} ^ {2}}} \frac {1}{I} \exp \left[ - \frac {\left(\ln I + \sigma_ {I} ^ {2} / 2\right) ^ {2}}{2 \sigma_ {I} ^ {2}} \right] \tag {3-14}
$$

# 3.2.2强湍流环境光强闪烁概率分布

当激光在强湍流条件下传输时，其光强闪烁概率分布可以用K分布（K-distribution）来 描述[70]： 

$$
p _ {k} (I) = \frac {2 \alpha^ {(\alpha + 1) / 2}}{\Gamma (\alpha)} I ^ {(\alpha - 1) / 2} K _ {\alpha - 1} (2 \sqrt {\alpha I}) \tag {3-15}
$$

式中： 

K  v 阶第二类修正贝塞尔函数。 

当光强闪烁处于饱和状态时，可以采用负指数分布（Negative exponential）来描述光强 闪烁概率密度[71]： 

$$
p _ {N E} (I) = \exp (- I) \tag {3-16}
$$

# 3.2.3由弱至强湍流环境光强闪烁概率分布

依据扩展的Rytov理论得知，归一化光强 $I = X Y$ 可以看做是大尺度光强起伏 $X$ 对小尺度 光强起伏Y 的乘性调制过程。Andrews基于这一理论，推导出了一个新的概率密度函数模型 [72]： 

$$
P _ {X} (X) = \frac {\alpha (\alpha X) ^ {\alpha - 1}}{\Gamma (\alpha)} \exp (- \alpha X) \tag {3-17}
$$

$$
P _ {Y} (Y) = \frac {\alpha (\alpha Y) ^ {\alpha - 1}}{\Gamma (\alpha)} \exp (- \alpha Y) \tag {3-18}
$$

式中： 

 —— Gamma 函数。 

假设 $X$ 一定，则 $Y = 1 / X$ ，得到 $I$ 基于 $X$ 的概率密度函数： 

$$
P _ {Y} (I \mid X) = \frac {\beta (\beta I / X) ^ {\beta - 1}}{X \Gamma (\beta)} \exp (- \beta I / X) \tag {3-19}
$$

将式（3-19）关于式（3-17）求平均得到Gamma-Gamma分布模型，其概率密度函数为： 

$$
p _ {G G} (I) = \frac {2 (\alpha \beta) (a + \beta) / 2}{\Gamma (\alpha) \Gamma (\beta)} I ^ {(\alpha + \beta) / 2 - 1} K _ {\alpha - \beta} (2 \sqrt {\alpha \beta I}) \tag {3-20}
$$

式中： 

I —— 归一化光强： 

$\alpha$ -大尺度散射元的有效数目； 

$\beta$ -小尺度散射元的有效数目。 

$\alpha$ $\beta$ 分别为： 

$$
\alpha = \frac {1}{\sigma_ {X} ^ {2}} = \frac {1}{\exp \left(\sigma_ {\ln X} ^ {2}\right) - 1}, \beta = \frac {1}{\sigma_ {Y} ^ {2}} = \frac {1}{\exp \left(\sigma_ {\ln Y} ^ {2}\right) - 1} \tag {3-21}
$$

因此，闪烁指数可以用 $\alpha$ 、 $\beta$ 表示为： 

$$
\sigma_ {I} ^ {2} = \frac {1}{\alpha} + \frac {1}{\beta} + \frac {1}{\alpha \beta} \tag {3-22}
$$

由式（3-22）可以看出分布参数 $\alpha$ 和 $\beta$ 与闪烁指数的关系，由于 $\alpha$ 和 $\beta$ 分别代表大尺度 涡旋与小尺度涡旋散射元的有效数目，同时湍流涡旋情况也反映了大气湍流状态相关，使得 Gamma-Gamma函数模型能够直接反映出大气信道的特性，因而被广泛使用。同时 Gamma-Gamma函数模型能够描述不同湍流状态下的闪烁特性。在强湍流环境下，即 $\beta = 1$ 时， Gamma-Gamma函数模型可化简为K分布模型；在闪烁饱和状态下，即 $\beta = 1$ 且 $\alpha \to \infty$ 时， Gamma-Gamma函数模型可化简为负指数分布模型；在弱湍流环境下， $\sigma _ { I } ^ { 2 } < 1$ ，Gamma-Gamma 函数模型等同于对数正态分布模型。 

随着湍流强度的增加，由于多次自干涉效应的影响，当接收孔径D大于产生光强起伏的 湍流尺度时，将产生孔径平均效应，对数正态分布模型和Gamma-Gamma信道衰落模型不再 适用。在中等到强湍流条件下，当 $D < < r _ { 0 }$ 时，不会发生孔径平均效应，此时光强闪烁特性 更符合Gamma-Gamma信道模型；当 $D > > r _ { 0 }$ 时，光强闪烁特性更符合对数正态分布。在考虑 孔径平均效应下，为了准确描述各种湍流强度下有限接收孔径上的光强闪烁情况，研究者提 出光强闪烁概率分布可以用威布尔指数模型来描述，其概率密度函数为[73]： 

$$
p _ {w} (I) = \frac {\alpha \beta}{\eta} \left(\frac {I}{\eta}\right) ^ {\beta - 1} \exp \left[ - \left(\frac {I}{\eta}\right) ^ {\beta} \right] \times \left\{1 - \exp \left[ - \left(\frac {I}{\eta}\right) ^ {\beta} \right] \right\} ^ {\alpha - 1} \tag {3-23}
$$

式中： 

$\alpha$ 和 $\beta$ 形状参数； 

n 一尺度参数，决定曲线的陡峭度。 

# 3.3本章小结

影响大气激光通信系统性能最主要的因素就是大气湍流效应。湍流效应会导致系统误码 率增大、通信质量下降。本章首先介绍了大气湍流效应的产生机理，阐述了光斑漂移、光束 扩展、光强闪烁的物理定义及其对通信系统具体的影响形式；其次针对不同的湍流环境，对 信道衰落的概率密度函数模型进行了描述，并进行分析比较。本章的理论分析将为后面实验 数据的处理提供理论依据。 

# 4.实验测量系统

当激光在湍流环境中传输时，受光强闪烁效应的影响，降低了大气参数测量精度、增加 了无线激光通信系统的误码率。为了对西安地区光强闪烁统计特性进行分析，本文进行了链 路长度 $2 5 0 0 \mathrm { m }$ 的激光传输实验，对晴天、阴天、多云、雪天等不同天气条件下的光强闪烁 特性进行了测量。本章详细介绍了实验系统的原理、结构及实验中各参数选取。 

# 4.1 几种大气折射率结构常数测量方法

研究人员根据不同的原理，提出了多种大气折射率结构常数的测量方法来分析不同环境 下的大气折射率结构常数变化规律。在无线光通信系统中，测量对象主要是激光传输路径中 的平均折射率结构常数。 

# 4.1.1闪烁指数测量法

激光在湍流环境中传输时，通过测量其闪烁强度，就可以通过计算得到折射率结构常数， 这种方法就是闪烁指数测量法，其表达式为[74]： 

$$
C _ {n} ^ {2} = 4. 4 8 C D _ {t} ^ {7 / 3} L ^ {- 3} \sigma_ {x} ^ {2} \tag {4-1}
$$

式中： 

$L$ 传输距离； 

$C = D _ { r } / D _ { t }$ — 发射端孔径尺寸/接收端孔径尺寸； 

$\sigma _ { x } ^ { 2 }$ 对数振幅方差，表达式为： 

$$
\sigma_ {x} ^ {2} = \frac {1}{4} \ln \left[ 1 + \left(\frac {s}{\langle I \rangle}\right) ^ {2} \right] \tag {4-2}
$$

式中： 

$\langle I \rangle$ — — 光强均值； 

s 光强标准差。 

# 4.1.2到达角起伏方差测量法

在不考虑湍流的内尺度和外尺度的情况下，平面波的到达角起伏方差为[75]： 

$$
\left\langle \beta^ {2} \right\rangle = 2. 9 1 C _ {n} ^ {2} L D ^ {- 1 / 3} \tag {4-3}
$$

式中： 

$C _ { n } ^ { 2 }$ — 大气折射率结构常数； 

$L$ — 传输距离； 

$D$ 接收孔径。 

因此，在测得 $\left. { \beta ^ { 2 } } \right.$ 的情况下，就可根据式（4-3）计算出 $C _ { n } ^ { 2 }$ 。经研究发现，与闪烁指数 测量法相比，两者具有一定的相关性，但存在一个固定的偏差，且到达角起伏法对光束的发 散角和功率不敏感。 

# 4.1.3热感探空仪测量技术

对于可见光和红外光，大气折射率结构常数可表示为[76]： 

$$
C _ {n} ^ {2} = \left(7 9 \times 1 0 ^ {- 6} P / T ^ {2}\right) ^ {2} C _ {T} ^ {2} \tag {4-4}
$$

式中： 

$P$ 压强； 

$T$ 温度； 

$C _ { T } ^ { 2 }$ 温度结构常数。 

热感探空仪工作原理是首先通过敏感度高的温度传感器对当前高度的温度结构函数进 行测量，再经过式（4-4）计算可得到大气折射率结构常数。将热感探空仪搭载于探空气球、 航天器或者不同高度的铁塔上，就可以测出不同高度的大气折射率结构常数。 

# 4.1.4激光雷达测量技术

激光传输时，通过对接收到的后向散射光某种特性进行测量，进而计算大气折射率结构 钢常数，这种方法就是激光雷达测量技术。这种技术将湍流大气分为若干层，每一层的折射 率结构常数可表示为[77]： 

$$
C _ {n} ^ {2} \left(z _ {l}\right) = \left(W _ {l}\right) - 1 \left[ f \left(z _ {l}\right) - \sum_ {s = 1} ^ {l - 1} W _ {s} C _ {n} ^ {2} \left(z _ {s}\right) \right] \tag {4-5}
$$

式中： 

$z _ { l }$ 第l层的高度； 

$z _ { s }$ z 第 $s$ 层的高度； 

$f$ —— 接收到的散射光某种特性； 

W — $f$ 对应的路径权重函数。 

通常W 是已知的，只需测出通过各层湍流大气所接收到的散射光特性 $f$ ，即可根据式 （4-5）得到各层湍流大气的折射率结构常数。同样，也能够通过测量散射光的其他特性来 计算各层的折射率结构常数，例如等晕角、到达角、光斑漂移量等。 

# 4.1.5无源测量技术

无源测量法的测量原理是首先测量太阳光源在湍流中传播的湍流效应，进而通过计算得 出大气折射率结构常数，主要使用SCIDAR（scintillation detection and ranging）、闪烁计阵 列和多孔径闪烁测量技术等。本节以SCIDAR技术为例来阐述无源测量的原理。 

各层湍流总的协方差函数 $C \left( \rho - \varphi h _ { g } \right)$ 需要各层大气折射率结构常数的全部参数，其计算 公式为[78]： 

$$
C \left(\rho - \varphi h _ {g}\right) = \int K \left(\rho , h _ {g}\right) C _ {n} ^ {2} \left(h _ {g}\right) d h _ {g} + n (\rho) \tag {4-6}
$$

式中： 

$\rho$ 径向坐标; 

两束自然光源的分离角； 

$C _ { p } ( \rho , \varphi )$ — 平行于双星间距的方差函数； 

$C _ { \perp } ( \rho , \varphi )$ — 垂直于双星间距的方差函数； 

$K { \left( { \rho , h _ { g } } \right) }$ 等效高度为 $h _ { g }$ 的单个湍流层产生的侧峰； 

$n ( \rho )$ — 噪声方差。 

首先测量 $C _ { p } ( \rho , \varphi )$ 和 $C _ { \perp } ( \rho , \varphi )$ ，计算出等效高度下的单个湍流层产生的侧峰，并估算出 噪声方差，按照最大统一性原则，能够通过式（4-6）反演得到各层湍流的大气折射率结构 常数。 

除了这几种测量大气折射率结构常数方法外，经常被使用的还有电波和声波雷达测量方 法。测量原理都是通过测量后向散射波的相关特征来反演大气折射率结构常数。 

相比其他测量方法，使用闪烁指数测量法和到达角起伏方差测量法得到的 $C _ { n } ^ { 2 }$ 是激光传 输路径上的平均折射率结构常数，没有反映接收端和发射端不确定情况下折射率结构常数的 变化规律，因此这两种测量方法不适合斜程路径或距离较远的情况，只有在发射端和接收端 位置确定时才能使用。使用热感探空仪来测量大气折射率结构常数的技术已经非常成熟，并 且具有各种技术中最高的垂直分辨率，可以达到 $5 0 \mathrm { m }$ ，但其主要针对不同高度的大气折射率 结构常数进行测量；无源测量方法会受到自然光源位置的影响，因此无法广泛使用；电波雷 达测量法虽然能够实现 $2 0 \mathrm { k m }$ 距离的测量，但成本高，占地空间大，并且数据处理也相对困 难。结合实验实际情况，本文采用闪烁指数法对实验期间大气折射率结构常数进行测量研究。

# 5.1 光强空间分布统计特性

# 5.1.1远场光强分布理论

光强闪烁效应是指激光在湍流大气中传输时，会通过大量不同尺度的湍流涡旋，受衍射 和散射效应的影响，导致接收端光强随机起伏。本次实验采取半导体激光器作为发射端光源， 当激光在大气信道中传输时，在任意位置处的光强分布为： 

$$
I \left(r _ {1}, d _ {1}\right) = I _ {0} \tau \frac {B}{G} \exp \left[ - \frac {2 B ^ {2}}{G ^ {2}} \frac {r _ {1} ^ {2}}{w _ {d} ^ {2}} \right] \tag {5-1}
$$

式中： 

$I _ { 0 }$ 光强值； 

$\tau$ 大气透过率； 

wd $w _ { d }$ 传输距离为 $d$ 时束腰半径； 

$r _ { 1 }$ 光斑半径； 

$B$ 和 $G$ 分别为 

$$
B = \frac {\pi}{\lambda \left(d _ {1} - d\right)} \tag {5-2}
$$

$$
G = \sqrt {\frac {1}{w _ {d} ^ {4}} + \frac {2}{w _ {d} ^ {2} \rho_ {0} ^ {2}} + \frac {\pi}{\lambda \left(d _ {1} - d\right) ^ {2}}} \tag {5-3}
$$

式中： 

$\rho _ { 0 } ^ { 2 } = \left( 0 . 5 4 5 C _ { n } ^ { 2 } k ^ { 2 } d _ { 1 } \right) ^ { - 3 / 5 }$ ， $k = 2 \pi / \lambda$ 空间波数； 

$C _ { n } ^ { 2 }$ 大气折射率结构常数； 

激光波长。 

可以看出，当激光在大气中传输时，影响光斑光强分布的主要因素有大气湍流强度、大 气透过率、传输距离、激光波长等。本次实验传输距离和波长是不变的，因此改变光斑空间 分布特性的主要因素是大气湍流环境。 

# 5.1.2光斑图像预处理

为了直观、准确反映光强的空间起伏规律，需要对采集得到的原始光斑图像进行中值滤 波、伪彩色变换、三维可视化等图像预处理来得到光强三维空间分布。 

# （1）中值滤波

在实验过程中，环境背景光会大大的干扰光斑采集的有效性。由于激光器接收功率较小， 当背景光较强时，将产生较大的噪声干扰，因此对实验数据进行处理的时候首先要消除背景 光的影响。 

本次实验得到的光斑灰度图像均受到噪声的干扰，图像质量受到了影响。中值滤波作为 一种经典的非线性的图像处理方法，在保持光斑图像边缘完整的同时可以完美过滤噪声影 响，更不会影响图像清晰度，并且弥补了其他滤波器的不足。 

中值滤波的基本思想是：将灰度图像划分为若干区域，并将灰度图像每一像素点的灰度 值设置为该点所在区域内所有像素点灰度值的中值.。 

使用中值滤波消除背景噪声具体操作为： 

1）首先定义一个长度为奇数的滤波窗口，将这一窗口按顺序在灰度图像中移动，使将 窗口中心依次与光斑像素点重合； 

2）统计滤波窗口中像素点周围所有灰度值； 

3）将灰度值按大小重新排列； 

4）找到灰度值的中值并定义为滤波窗口的输出值，即将灰度中值赋值给滤波窗口中心 像素点。 

5）继续移动滤波窗口，重复之前流程，完成整个图像的滤波处理。 

# （2）伪彩色变换

伪彩色变换是一种图像增强技术。其目的是将灰度值图像中每一个像素点的灰度值转化 成红、绿、蓝的强度值，更直观的展示其强度分布。灰度级-彩色变换法是一种基于空间域 将灰度值通过红、绿、蓝映射函数生成RGB 彩色图的方法，适用于256级以下的灰度图像， 并且得到连续的伪彩色增强图像。 

使用伪彩色变换处理实验得到灰度图像整体步骤为： 

首先将灰度值图像中的像素点通过函数映射为红、绿、蓝色彩空间的三个分量，其变换 过程为[79]： 

$$
I _ {R} (x, y) = f (x, y) R (l) \tag {5-4}
$$

$$
I _ {G} (x, y) = f (x, y) G (l) \tag {5-5}
$$

$$
I _ {B} (x, y) = f (x, y) B (l) \tag {5-6}
$$

式中： 

$I _ { \scriptscriptstyle R } ( x , y )$ ， $I _ { G } ( x , y )$ ， $I _ { \scriptscriptstyle B } ( x , y )$ 变换后的红、绿、蓝强度； 

Rl  ， $G ( l )$ ， $B ( l )$ —— 红、绿、蓝映射函数； 

$f ( x , y )$ —— 坐标 $\left( x , y \right)$ 处的像素点的灰度值。 

经过映射函数计算后得到的红、绿、蓝强度信号分别处理并编码，就能得到彩色图像。 处理后的图像中任意一个像素点的色调为： 

$$
g (x, y) = I _ {R} (x, y) + I _ {G} (x, y) + I _ {B} (x, y) \tag {5-7}
$$

灰度级-彩色变换方法作为一种非线性变换函数只要保证灰度值-彩色值映射函数是连续 即可得到光滑且连续的彩色编码。 

# （3）三维伪彩色可视化

三维图像可以更加直观的反映出光斑在不同区域的能量分布，并且能够进一步的通过变 换三维坐标观察光斑光强空间分布。首先将灰度图像像素点的位置作为 X 轴和 Y 轴，再使 用灰度差值技术将灰度值转化为伪彩色值作为 Z轴，就可以生成光斑三维伪彩色图像。由于 在灰度图像中，像素值仅在整数坐标处有定义，因此需要通过计算附近别的像素值的灰度值 来获取非整数坐标处的像素值，这种技术就叫做灰度差值技术。为了提高三维伪彩色图像的 清晰度和辨识度，本文使用双线性差值技术，实现了光斑图像的灰度差值转化。

# 5.2.2闪烁指数统计特征

大气折射率结构常数 $C _ { n } ^ { 2 }$ 反映了光学湍流的强弱。在不同的湍流强度下，光强闪烁程度也 不同。光斑的光强分布反映了光强在某一时刻的空间分布状态，为了反映光强随时间的变化 规律，采用光功率计测量了不同时间的光强变化来分析西安地区的光强统计特性。本文通过 Rytov 方差来描述光强起伏，其表达式为： 

$$
\sigma_ {I} ^ {2} = 1. 2 3 C _ {n} ^ {2} k ^ {7 / 6} L ^ {1 1 / 6} \tag {5-8}
$$

# 5.3本章小结

本章对西安地区光强闪烁特性进行了实验测量研究。首先将实验得到的光斑图像进行中 值滤波处理，去除掉背景噪声的影响，并通过三维伪彩色变换比较了不同天气下的光强空间 分布规律。之后研究了晴、阴、雾霾三种典型天气条件下夜间湍流变化趋势及光强闪烁统计 特性，通过对实验测量数据的分析处理，发现大气折射率结构常数 $C _ { n } ^ { 2 }$ 按照晴天、雾霾天、 阴天的顺序依次减小，起伏程度也依次变小。同时发现三种典型天气条件下的光强闪烁密度 均服从威布尔指数分布，且受孔径平均效应的影响，其拟合效果优于对数正态分布和 Gamma-Gamma分布。因此，以威布尔指数分布模型来描述该地区光强闪烁统计特征，能更 准确的反映激光信号的传输规律，具有一定的参考意义。

# 6.1工作总结

无线光通信技术的崛起，使得关于大气信道的研究成为一个热点问题，本文对由大气湍 流引起的激光光斑漂移、光束扩展和光强闪烁等湍流效应的相关理论进行了深入研究。同时 搭建了用于测量光强闪烁特性的实验系统，并在多种气象条件下进行了大量的实验测量。论 文主要工作如下： 

（1）研究了晴天、阴天、雨天、雪天和雾霾天气下光斑空间分布，发现在光斑几何中 心附近，晴天的光斑陡峭度最大，阴天次之，雨天和雪天较为平缓；光斑光强最大值按照晴 天、阴天、雾霾天、雨天、雪天的次序逐渐减小。 

（2）研究了西安地区秋冬季节晴、阴、雾霾三种典型天气条件下夜间湍流变化趋势及 光强闪烁统计特性，通过对实验测量数据的分析处理发现，在 19 时左右，受太阳辐射和地 表辐射平衡影响，热量传递减少，大气折射率结构常数 $C _ { n } ^ { 2 }$ 开始减小，湍流强度减弱，并在 19时30分达到夜间的极小值点，此后 $C _ { n } ^ { 2 }$ 的变化曲线波动剧烈，湍流变化较傍晚更为复杂； 并且 $C _ { n } ^ { 2 }$ 按照晴天、雾霾天、阴天的顺序依次减小，其起伏程度也依次变小。同时发现，温 度起伏越大， $C _ { n } ^ { 2 }$ 就越大，湍流变化程度也更加剧烈。计算其闪烁指数发现晴天闪烁指数起 伏程度最大，且晴天大部分时间光强起伏属于中等起伏区，雾霾天和阴天大部分时间属于弱 起伏区。三种典型天气条件下的光强闪烁密度均服从威布尔指数分布，且受孔径平均效应的 影响，其拟合效果优于对数正态分布和 Gamma-Gamma分布。因此，以威布尔指数分布模型 来描述该地区光强闪烁统计特征，能更准确的反映激光信号的传输规律，具有一定的参考意 义。 

# 6.2工作展望

研究大气湍流中远距离传输激光光束的光强闪烁特性，对于无线光激光通信系统的发展 具有一定的参考意义。论文只是在该问题上取得了一些初步成果，还有很多问题有待进一步 解决。未来工作将围绕以下几方面进行： 

（1）本文进行的测量实验仅在 $1 8 \colon 0 0 \sim$ 次日6:00开展，所得结论仍具有一定的局限性。 因此，后续将开展全天候的激光传输实验，针对大气湍流日变化、月变化趋势以及光强闪烁 特性进行实验研究； 

（2）本文只使用了波长为 $6 5 0 \mathrm { n m }$ 的激光器进行实验，下一步对比不同波长的发射光源 开展实验，研究如何能够有效抑制光强闪烁效应； 

（3）下一步将结合不同湍流环境对激光造成的传输衰减影响，同时针对不同下垫面及 天气条件进一步为西安地区无线光通信系统的应用提供参考依据。
